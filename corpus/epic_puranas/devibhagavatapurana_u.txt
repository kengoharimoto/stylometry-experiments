Devīgītā


atha prathamo 'dhyāyaḥ -
saptamaḥ skandhaḥ
atha ekatriṃśo 'dhyāyaḥ


janamejaya uvāca

dharādharādhīśa maulāvāvirāsītparaṃ mahaḥ /
yaduktaṃ bhavatā pūrvaṃ vistarāttadvadasva me // 1.1 = 7,31.1 //

ko virajyeta matimānpibañchaktikathāmṛtam /
sudhāṃ tu pibatāṃ mṛtyuḥ sa naitacchṛṇvato bhavet // 1.2 = 7,31.2 //


vyāsa uvāca

dhanyo 'si kṛtakṛtyo 'si śikṣito 'si mahātmabhiḥ /
bhāgyavānasi yaddevyāṃ nirvyājā bhaktirasti te // 1.3 = 7,31.3 //

śṛṇu rājanpurā vṛttaṃ satīdehe 'gnibharjite /
bhrāntaḥ śivastu babhrāma kvaciddeśe sthiro 'bhavat // 1.4 = 7,31.4 //

prapañcabhānarahitaḥ samādhigatamānasaḥ /
dhyāyandevīsvarūpaṃ tu kālaṃ ninye sa ātmavān // 1.5 = 7,31.5 //

saubhāgyarahitaṃ jātaṃ trailokyaṃ sacarācaram /
śaktihīnaṃ jagatsarvaṃ sābdhidvīpaṃ saparvatam // 1.6 = 7,31.6 //

ānandaḥ śuṣkatāṃ yātaḥ sarveṣāṃ hṛdayāntare /
udāsīnāḥ sarvalokāścintājarjaracetasaḥ // 1.7 = 7,31.7 //

sadā duḥkhodadhau magnā rogagrastāstadābhavan /
grahāṇāṃ devatānāṃ ca vaiparītyena vartanam // 1.8 = 7,31.8 //

adhibhūtādhidaivānāṃ satyabhāvānnṛpābhavan /
atha 'sminneva kāle tu tārakākhyo mahasuraḥ // 1.9 = 7,31.9 //

brahmadattavaro daityo 'bhavattrailokyanāyakaḥ /
śivaurasastu yaḥ putraḥ sa te hantā bhaviṣyati // 1.10 = 7,31.10 //

iti kalpitamṛtyuḥ sa devadevairmahāsuraḥ /
śivairasasutābhāvājjagarja ca nananda ca // 1.11 = 7,31.11 //

tena copadrutāḥ sarve svasthānātpracyutāḥ surāḥ /
śivairasasutābhāvāccintāmāpurduratyayām // 1.12 = 7,31.12 //

nāṅganā śaṅkarasyāsti kathaṃ tatsutasambhāvaḥ /
asmākaṃ bhāgyahīnānāṃ kathaṃ kāryaṃ bhaviṣyati // 1.13 = 7,31.13 //

iti cintāturāḥ sarve jagmurvaikuṇṭhamaṇḍale /
śaśaṃsurharimekānte sa copāyaṃ jagāda ha // 1.14 = 7,31.14 //

kutaścintāturāḥ sarve kāmakalpadrumā śivā /
jāgarti bhūvaneśānī maṇidvīpādhivāsinī // 1.15 = 7,31.15 //

asmākamanayādeva tadupekṣāsti nānyathā /
śikṣaiveyaṃ jaganmātrā kṛtāsmacchikṣaṇāya // 1.16 = 7,31.16 //

lālane tāḍane māturnā kāruṇyaṃ yathārbhake /
tadvadeva jaganmāturniyantrā guṇadoṣayoḥ // 1.17 = 7,31.17 //

aparādho bhavatyeva tanayasya pade pade /
ko 'paraḥ sahate loke kevalaṃ mātaraṃ vinā // 1.18 = 7,31.18 //

tasmadyūyaṃ parāmbāṃ tāṃ śaraṇaṃ yāta mā ciram /
nirvyājayā cittavṛttyā sā vaḥ kāryaṃ vidhāsyati // 1.19 = 7,31.19 //

ityādiśya surānsarvānmahāviṣṇuḥ svajāyayā /
saṃyuto nirjagāmāśu devaiḥ saha surādhipaḥ // 1.20 = 7,31.20 //

ājagāma mahāśailaṃ himavantaṃ nagādhipam /
abhavaṃśca surāḥ sarve puraścaraṇakarmiṇaḥ // 1.21 = 7,31.21 //

ambāyajñavidhānajñā ambāyajñaṃ ca cakrire /
tṛtīyādivratānyāśu cakruḥ sarve surā nṛpa // 1.22 = 7,31.22 //

kecitsamādhiniṣṇātāḥ kecinnāmaparāyaṇāḥ /
kecitsūktaparāḥ kecinnāmapārāyaṇotsukāḥ // 1.23 = 7,31.23 //

mantrapārāyaṇaparaḥ kecitkṛcchrādi kāriṇaḥ /
antaryāgaparāḥ kecitkecinnyāsaparāyaṇāḥ // 1.24 = 7,31.24 //

hṛllekhayā parāśakteḥ pūjāṃ cakruratandritāḥ /
ityevaṃ bahuvarṣāṇi kālo 'gājjanamejaya // 1.25 = 7,31.25 //

akasmāccaitramāsīyanavamyāṃ ca bhṛgordine /
prādurbabhūva puratastanmahaḥ śrutibodhitam // 1.26 = 7,31.26 //

caturdikṣu caturvedairmūrtimadbhirabhiṣṭutam /
koṭisūrypratīkāśaṃ candrakoṭisuśītalam // 1.27 = 7,31.27 //

vidyutkoṭisamānābhamaruṇaṃ tatparaṃ mahaḥ /
naiva cordhvaṃ na tiryakca na madhye parijagrabhat // 1.28 = 7,31.28 //

ādyantarahitaṃ tattu na hastādyaṅgasaṃyutam /
na ca strīrūpamathavā na puṃrūpamathobhayam // 1.29 = 7,31.29 //

dīptyā pidhānaṃ netrāṇāṃ teṣāmāsīnmahīpate /
punaśca dhairyamālambya yāvatte dadṛśuḥ surāḥ // 1.30 = 7,31.30 //

tāvattadeva strīrūpeṇābhāddivyaṃ manoharam /
atīva ramaṇīyāṅgīṃ kumārīṃ navayauvanām // 1.31 = 7,31.31 //

udyatpīnakucadvandvaninditāmbhojakuḍmalām /
raṇatkiṅkiṇikājālasiñjanmañjīramekhalām // 1.32 = 7,31.32 //

kanakāṅgadakeyūragraiveyakavibhūṣitām /
anarghyamaṇisambhinnagalabandhavirājitām // 1.33 = 7,31.33 //

tanuketakasaṃrājannīlabhramarakuntalām /
nitambabimbasubhagāṃ romarājivirājitām // 1.34 = 7,31.34 //

karpūraśakalonmiśratāmbūlapūritānanām /
kanatkanakatāṭaṅkaviṭaṅkavadanāmbujām // 1.35 = 7,31.35 //

aṣṭamīcandrabimbābhalalāṭāmāyatabhruvam /
raktāravindanayanāmunnasāṃ madhurādharām // 1.36 = 7,31.36 //

kundakuḍmaladantāgrāṃ muktāhāravirājitām /
ratnasambhinnamukuṭāṃ candrarekhāvatāmsinīm // 1.37 = 7,31.37 //

mallikāmālatīmālākeśapāśavirājitām /
kāśmīrabinduniṭilāṃ netratrayavilāsinīm // 1.38 = 7,31.38 //

pāśāṅkuśavarābhīticaturbāhuṃ trilocanām /
raktavastraparīdhānāṃ dāḍimīkusumaprabhām // 1.39 = 7,31.39 //

sarvaśṛṅgāraveṣāḍhyāṃ sarvadevanamaskṛtām /
sarvāśāpūrikāṃ sarvamātaraṃ sarvamohinīm // 1.40 = 7,31.40 //

prasādasumukhīmambāṃ mandasmitamukhāmbujām /
avyājakaruṇāmūrtiṃ dadṛśuḥ purataḥ surāḥ // 1.41 = 7,31.41 //

dṛṣṭvā tāṃ karuṇāmurtiṃ praṇemuḥ sakalāḥ surāḥ /
vaktuṃ nāśaknuvan kiñcidvāṣpasaṃruddhaniḥsvanāḥ // 1.42 = 7,31.42 //


kathañcitsthairyamālambya bhaktyā cānatakandharāḥ /
premāśrupūrṇanayanāstuṣṭuvurjagadambikām // 1.43 = 7,31.43 //


devā ūcuḥ

namo devyai mahādevyai śivāyai satataṃ namaḥ /
namaḥ prakṛtyai bhadrāyai niyatāḥ praṇatāḥ sma tām // 1.44 = 7,31.44 //

tām agnivarṇāṃ tapasā jvalantīṃ vairocanīṃ karmaphaleṣu juṣṭām /
durgāṃ devīṃ śaraṇamahaṃ prapadye sutarasi tarase namaḥ // 1.45 = 7,31.45 //

devīṃ vācamajanayanta devāstāṃ viśvarūpāḥ paśavo vadanti /
sā no mandreṣamūrjaṃ duhānā dhenur vāgasmānupasuṣṭutaitu // 1.46 = 7,31.46 //

kālarātrīṃ brahmastutāṃ vaiṣṇavīṃ skandamātaram /
sarasvatīmaditiṃ dakṣaduhitaraṃ namāmaḥ pāvanāṃ śivām // 1.47 = 7,31.47 //

mahālakṣmyai ca vidmahe sarvaśaktyai ca dhīmahi /
tanno devī pracodayāt // 1.48 = 7,31.48 //

namo virāṭsvarūpiṇyai namaḥ sūtrātmamūrtaye /
namo 'vyākṛtarūpiṇyai namaḥ śrībrahma mūrtaye // 1.49 = 7,31.49 //

yadajñānājjagaddhāti rajjusarpasragādivat /
yajjñānāllayamāpnoti numastāṃ bhuvaneśvarīm // 1.50 = 7,31.50 //

numastatpadalakṣyārthāṃ cidekarasarūpiṇīm /
akhaṇḍānandarūpāṃ tāṃ vedatātparyabhūmikām // 1.51 = 7,31.51 //

pañcakośātiriktāṃ tām avasthātrayasākṣiṇīm /
punastvampadalakṣyārthāṃ pratyagātmasvarūpiṇīm // 1.52 = 7,31.52 //

namaḥ praṇavarūpāyai namo hrīṅkāramūrtaye /
nānāmantrātmikāyai te karuṇāyai namo namaḥ // 1.53 = 7,31.53 //

iti stutā tadā devair maṇidvīpādhivāsinī /
prāha vācā madhurayā mattakokilaniḥsvanā // 1.54 = 7,31.54 //


śrīdevyuvāca

vedantu vibudhāḥ kāryaṃ yadarthamiha saṅgatāḥ /
varadāhaṃ sadā bhaktakāmakalpadrumāsmi ca // 1.55 = 7,31.55 //

tiṣṭhantyāṃ mayi kā cintā yuṣmākaṃ bhaktiśālinām /
samuddharāmi madbhaktānduḥkhasaṃsārasāgarāt // 1.56 = 7,31.56 //

iti pratijñāṃ me satyāṃ jānītha vibudhottamāḥ /
iti premākulāṃ vāṇīṃ śrutvā santuṣṭamānasāḥ // 1.57 = 7,31.57 //

nirbhayā nirjarā rājannūcurduḥkhaṃ svakīyakam /


devā ūcuḥ

nājñātaṃ kiñcidapyatra bhvatyāsti jagastraye // 1.58 = 7,31.58 //

sarvajñayā sarvasākṣirūpiṇyā parmeśvari /
tārakeṇāsurendreṇa pīḍitāḥ smo divāniśam // 1.59 = 7,31.59 //

śivāṅgajādvadhastasya nirmito brahmaṇāśive /
śivāṅganā tu naivāsti jānāsi tvaṃ maheśvari // 1.60 = 7,31.60 //

sarvajñapurataḥ kiṃ vā vaktavyaṃ pāmarairjanaiḥ /
etaduddeśataḥ proktamaparaṃ tarkayāmbike // 1.61 = 7,31.61 //

sarvadā caraṇāmbhoje bhaktiḥ syāttava niścalā /
prārthanīyamidaṃ mukhyamaparaṃ dehahetave // 1.62 = 7,31.62 //

iti teṣāṃ vacaḥ śrutvā provāca parameśvarī /
mama śaktistu yā gaurī bhaviṣyati himālaye // 1.63 = 7,31.63 //

śivāya sā pradeyā syāt sā vaḥ kāryaṃ vidhāsyati /
bhaktiryaccaraṇāmbhoje bhūyādyuṣmākamādarāt // 1.64 = 7,31.64 //

himālayo hi manasā māmupāste 'tibhaktitaḥ /
tatastasya gṛhe janma mama priyakaraṃ matam // 1.65 = 7,31.65 //


vyāsa uvāca

himālayo 'pi tacchrutvātyanugrahakaraṃ vacaḥ /
bāṣpaiḥ saṃruddhakaṇṭhākṣo mahārājñīṃ vaco 'bravīt // 1.66 = 7,31.66 //

mahattaraṃ taṃ kuruṣe yasyānugrahamicchasi /
nocetkvāhaṃ jaḍaḥ sthāṇuḥ kva tvaṃ saccitsvarūpiṇī // 1.67 = 7,31.67 //

asambhāvyaṃ janmaśataistvatpitṛtvaṃ mamānaghe /
aśvamedhādi puṇyairvā puṇyairvā tatsamādhijaiḥ // 1.68 = 7,31.68 //

adya prapañce kīrtiḥ syājjaganmātā sutābhavat /
aho himālayasyāsya dhanyo 'sau bhāgyavāniti // 1.69 = 7,31.69 //

yasyāstu jaṭhare santi brahmāṇḍānāṃ ca koṭayaḥ /
saiva yasya sutā jātā ko vā syāttatsamo bhuvi // 1.70 = 7,31.70 //

na jāne 'smatpitṝṇāṃ kiṃ sthānaṃ syānnirmitaṃ param /
etādṛśānāṃ vāsāya yeṣāṃ vaṃśe 'sti mādṛśaḥ // 1.71 = 7,31.71 //

idaṃ yathā ca dattaṃ me kṛpayā premapūrṇayā /
sarvavedāntasiddhaṃ ca tvadrūpaṃ brūhi me tathā // 1.72 = 7,31.72 //

yogaṃ ca bhaktisahitaṃ jñānaṃ ca śrutisammatam /
vadasva parameśāni tvamevāhaṃ yato bhaveḥ // 1.73 = 7,31.73 //


vyāsa uvāca

iti tasya vacaḥ prasannamukhapaṅkajā /
vaktumārabhatāmbā sā rahasyaṃ śrutigūhitam // 1.74 = 7,31.74 //

iti devīgītāyāṃ prathamo 'dhyāyaḥ -
iti śrīdevībhāgavate mahāpurāṇe saptamaskandhe devīgītāyāmekatriṃśo 'dhyāyaḥ

śrīdevyuvāca

śṛṇvantu nirjarāḥ sarve vyāharantyā vaco mama /
yasya śravaṇamātreṇa madrūpatvaṃ prapadyate // 2.1 = 7,32.1 //

aham evāsa pūrvaṃ tu nānyatkiñcinnagādhipa /
tadātmarūpaṃ citsaṃvit parabrahmaikanāmakam // 2.2 = 7,32.2 //

apratarkyam anirdeśyamanaupamyamanāmayam /
tasya kācitsvataḥ siddhā śaktirmāyeti viśrutā // 2.3 = 7,32.3 //

na satī sā nāsatī sā nobhayātmā virodhataḥ /
etadvilakṣaṇā kācidvastubhūtāsti sarvadā // 2.4 = 7,32.4 //

pāvakasyoṣṇateveyamuṣṇāṃśoriva dīdhitiḥ /
candrasya candrikeveyaṃ mameyaṃ sahajā dhruvā // 2.5 = 7,32.5 //

tasyāṃ karmaṇi jīvānāṃ jīvāḥ kālāśca sañcare /
abhedena vilīnāḥ syuḥ suṣuptau vyavahāravat // 2.6 = 7,32.6 //

svaśakteś ca samāyogādahaṃ bījātmatāṃ gatā /
svādhārāvaraṇāttasyā doṣatvaṃ ca samāgatam // 2.7 = 7,32.7 //

caitanyasya samāyogānnimittatvaṃ ca kathyate /
prapañcapariṇāmācca samavāyitvamucyate // 2.8 = 7,32.8 //

kecittāṃ tapa ityāhustamaḥ kecijjaḍaṃ pare /
jñānaṃ māyāṃ pradhānaṃ ca prakṛtiṃ śaktimapyajām // 2.9 = 7,32.9 //

vimarśa iti tāṃ prāhuḥ śaivaśāstraviśāradāḥ /
avidyāmitare prāhurvedatattvārthacintakāḥ // 2.10 = 7,32.10 //

evaṃ nānāvidhāni syurnāmāni nigamādiṣu /
tasyā jaḍatvaṃ dṛśyatvājjñānanāśāttato 'satī // 2.11 = 7,32.11 //

caitanyasya na dṛśyatvam dṛśyatve jaḍameva tat /
svaprakāśaṃ ca caitanyaṃ na pareṇa prakāśitam // 2.12 = 7,32.12 //

anavasthādoṣasattvānna svenāpi prakāśitam /
karmakartrīvirodhaḥ syāttasmāttaddīpavatsvayam // 2.13 = 7,32.13 //

prakāśamānamanyeṣāṃ bhāsakaṃ viddhi parvata /
ata eva ca nityatvaṃ siddhasaṃvittanormama // 2.14 = 7,32.14 //

jāgratsvapnasuṣuptyādau dṛśyasya vyabhicārataḥ /
saṃvido vyabhicāraśca nānubhūto 'sti karhicit // 2.15 = 7,32.15 //

yadi tasyāpyanubhavastarhyayaṃ yena sākṣiṇā /
anubhūtaḥ sa evātra śiṣṭaḥ saṃvidvapuḥ purā // 2.16 = 7,32.16 //

ata eva ca nityatvaṃ proktaṃ sacchāstrakovidaiḥ /
ānandarūpatā cāsyāḥ parapremāspadatvataḥ // 2.17 = 7,32.17 //

mā na bhūvaṃ hi bhūyāsamiti premātmani sthitam /
sarvasyānyasya mithyātvādasaṅgatvaṃ sphuṭaṃ mama // 2.18 = 7,32.18 //

aparicchinnatāpyevamata eva matā mama /
tacca jñānaṃ nātmadharmā dharmatve jaḍatātmanaḥ // 2.19 = 7,32.19 //

jñānasya jaḍaśeṣatvaṃ na dṛṣṭaṃ na ca saṃbhavi /
ciddharmatvaṃ tathā nāsti citaścinna hi bhidyate // 2.20 = 7,32.20 //

tasmādātmā jñānarūpaḥ sukharūpaśca sarvadā /
satyaḥ pūrṇo 'py asaṅgaśca dvaitajālavivarjitaḥ // 2.21 = 7,32.21 //

sa punaḥ kāmakarmādiyuktayā svīyamāyayā /
pūrvānubhūtasaṃskārātkālakarmavipākataḥ // 2.22 = 7,32.22 //

avivekācca tattvasya sisṛkṣāvānprajāyate /
abuddhipūrvaḥ sargo 'yaṃ kathitaste nagādhipa // 2.23 = 7,32.23 //

etaddhi yanmayā proktaṃ mama rūpamalaukikam /
avyākṛtaṃ tadavyaktaṃ māyāśabalamityapi // 2.24 = 7,32.24 //

procyate sarvaśāstreṣu sarvakāraṇakāraṇam /
tattvānāmādibhūtaṃ ca saccidānandavigraham // 2.25 = 7,32.25 //

sarvakarmaghanībhūtamicchājñānakriyāśrayam /
hrīṅkāramantravācyaṃ tadāditattvaṃ taducyate // 2.26 = 7,32.26 //

tasmādākāśa utpannaḥ śabdatanmātrarūpakaḥ /
bhavetsparśātmako vāyustejorūpātmakaṃ punaḥ // 2.27 = 7,32.27 //

jalaṃ rasātmakaṃ paścāttato gandhātmikā dharā /
śabdaikaguṇa ākāśo vāyuḥ sparśaravānvitaḥ // 2.28 = 7,32.28 //

śabdasparśarūpaguṇaṃ teja ityucyate budhaiḥ /
śabdasparśarūparasairāpo vedaguṇāḥ smṛtāḥ // 2.29 = 7,32.29 //

śabdasparśarūparasagandhaiḥ pañcaguṇā dharā /
tebhyo 'bhavanmahatsūtraṃ yalliṅgaṃ paricakṣate // 2.30 = 7,32.30 //

sarvātmakam tatsamproktaṃ sūkṣmadeho 'yamātmanaḥ /
avyaktaṃ kāraṇo dehaḥ sa coktaḥ pūrvam eva hi // 2.31 = 7,32.31 //

yasmiñjagadbījarūpaṃ sthitaṃ liṅgodbhavo yataḥ /
tataḥ sthūlāni bhūtāni pañcīkaraṇamārgataḥ // 2.32 = 7,32.32 //

pañcasaṃkhyāni jāyante tatprakārastvathocyate /
pūrvoktāni ca bhūtāni pratyekaṃ vibhajed dvidhā // 2.33 = 7,32.33 //

ekaikaṃ bhāgamekasya caturdhā vibhajedgire /
svasvetaradvitīyāṃśe yojanātpañca pañca te // 2.34 = 7,32.34 //

tatkāryam ca virāḍdehaḥ sthūladeho 'yamātmanaḥ /
pañcabhūtasthasattvāṃśaiḥ śrotrādīnāṃ samudbhavaḥ // 2.35 = 7,32.35 //

jñānendriyāṇāṃ rājendra pratyekaṃ militaistu taiḥ /
antaḥkaraṇamekaṃ syād vṛttibhedāccaturvidham // 2.36 = 7,32.36 //

yadā tu saṅkalpavikalpakṛtyaṃ tadā bhavettanmana ityabhikhyam /
syād buddhisaṃjñaṃ ca yadā pravetti suniścitaṃ saṃśayahīnarūpam // 2.37 = 7,32.37 //

anusandhānarūpaṃ taccittam ca parikīrtitam /
ahaṅkṛtyātmavṛttyā tu tadahaṅkāratāṃ gatam // 2.38 = 7,32.38 //

teṣāṃ rajoṃśairjñātāni kramātkarmendriyāṇi ca /
pratyekaṃ militairtairtu prāṇo bhavati pañcadhā // 2.39 = 7,32.39 //

hṛdi prāṇo gude 'pāno nābhisthastu samānakaḥ /
kaṇṭhadeśe 'pyudānaḥ syādvyānaḥ sarvaśarīragaḥ // 2.40 = 7,32.40 //

jānendriyāṇi pañcaiva pañca karmendriyāṇi ca /
prāṇādipañcakaṃ caiva dhiyāca sahitaṃ manaḥ // 2.41 = 7,32.41 //

etatsūkṣmaśarīraṃ syānmama liṇgaṃ yaducyate /
tatra yā prakṛtiḥ proktā sā rājandvididhā smṛtā // 2.42 = 7,32.42 //

sattvātmikā tu māyā syādavidyā guṇamiśritā /
svāśrayaṃ yā tu saṃrakṣetsā māyeti nigadyate // 2.43 = 7,32.43 //

tasyāṃ yatpratibimbaṃ syādbimbabhūtasya ceśituḥ /

karoti vividhaṃ viśvaṃ nānābhogāśrayaṃ punaḥ /

etādṛśaṃ mahārūpaṃ dadṛśuḥ surapuṅgavāḥ /

daṃṣṭrākaṭakaṭārāvaṃ vamantaṃ vahnimakṣibhiḥ /

sahasraśīrṣanayanaṃ sahasracaraṇaṃ tathā /

upasaṃhara deveśi rūpametadalaukikam /

śrī devyuvāca

kva yūyaṃ mandabhāgyā vai kvedaṃ rūpaṃ mahādbhutam /

na vedādhyayanairyogair na dānaistapasejyayā /

punastatsaṃskṛtivaśānnānā karmarataḥ sadā /

ajñānanāśane śaktā vidyaiva tu paṭīyasī // 4.8 = 7,34.8 //

na karma tajjannopāstirvirodhābhāvato gire /
pratyutāśājñānanāśe karmaṇā naiva bhāvyatām // 4.9 = 7,34.9 //

anarthadāni karmāṇi punaḥ punaruśanti hi /
tato rāgastato doṣastato 'nartho mahānbhavet // 4.10 = 7,34.10 //

tasmāt sarva prayatnena jñānaṃ sampādayennaraḥ /
kurvanneveha karmāṇītyataḥ karmāpyavaśyakam // 4.11 = 7,34.11 //

jñānādeva hi kaivalyam ataḥ syāt tatsamuccayaḥ /
sahāyatāṃ vrajet karma jñānasya hitakāri ca // 4.12 = 7,34.12 //

iti kecidvadantyatra tadvirodhānna sambhavet /
jñānādhṛdgranthibhedaḥ syādhṛdgranthau karmasambhavaḥ // 4.13 = 7,34.13 //

yaugapadyaṃ na sambhāvyaṃ virodhāt tu tatastayoḥ /
tamaḥ prakāśayoryadvadyaugapadyaṃ na sambhavi // 4.14 = 7,34.14 //

tasmāt sarvāṇi karmāṇi vaidikāni mahāmate /
cittaśuddhyantam eva syustāni kuryāt prayatnataḥ // 4.15 = 7,34.15 //

śamo damastitikṣā ca vairāgyaṃ sattvasambhavaḥ /
tāvat paryantam eva syuḥ karmāṇi na tataḥ param // 4.16 = 7,34.16 //

tadante caiva saṃnyasya saṃśrayedgurum ātmavān /
śrotriyaṃ brahmaniṣṭhaṃ ca bhaktyā nirvyājayā punaḥ // 4.17 = 7,34.17 //

vedāntaśravaṇaṃ kuryānnityam evam atandritaḥ /
tat tvam asyādi vākyasya nityam arthaṃ vicārayet // 4.18 = 7,34.18 //

tat tvam asyādi vākyaṃ tu jīvabrahmaikyabodhakam /
aikye jñāte nirbhayastu madrūpo hi prajāyate // 4.19 = 7,34.19 //

padārthāvagatiḥ pūrvaṃ vākyārthāvagatistataḥ /

vācyārthayorviruddhatvāt aikyaṃ naiva ghaṭeta ha /

indriyāṇi hayān āhur viṣayāṃs teṣu gocarān /

vijñānasārathiryastu manaḥ pragrahavān naraḥ /
so 'dhvanaḥ pāram āpnoti madīyaṃ yatparaṃ padam // 4.39 = 7,34.39 //

itthaṃ śrutyā ca matyā ca niścityātmānam ātmanā /

yogavṛtteḥ purā svasmin bhāvayedakṣaratrayam /

hakāraḥ sthūladehaḥ syād rakāraḥ sūkṣmadehakaḥ /
īkāraḥ kāraṇātmāsau hrīṅkāro 'haṃ turīyakam // 4.42 = 7,34.42 //

evaṃ samaṣṭidehe 'pi jñātvā bījatrayam kramāt /

samādhikālātpūrvaṃ tu bhāvayitvaivam ādṛtaḥ /

prāṇāpānau samau kṛtvā nāsābhyantaracāriṇau /
nivṛttaviṣayākāṅkṣo vītadoṣo vimatsaraḥ // 4.45 = 7,34.45 //

bhaktyā nirvyājayā yukto guhāyāṃ niḥsvane sthale /

vācyavācakatāhīnaṃ dvaitabhāvavivarjitam /

yogayuktyānayā dṛṣṭvā mām ātmānaṃ parātparam /

atha pañcamo 'dhyāyaḥ -
saptamaḥ skandhaḥ
atha pañcatriṃśo 'dhyāyaḥ


himālaya uvāca

yogaṃ veda maheśāni sāṅgaṃ saṃvitpradāyakam /
kṛtena yena yogyo 'haṃ bhaveyaṃ tattvadarśane // 5.1 = 7,35.1 //


śrīdevyuvāca

na yogo nabhasaḥ pṛṣṭhe na bhūmau na rasātale /
aikyaṃ jīvātmanor āhur yogaṃ yogaviśāradāḥ // 5.2 = 7,35.2 //

tatpratyūhāḥ ṣaḍākhyātā yogavighnakarānagha /
kāmakrodhau lobhamohau madamātsaryasañjñakau // 5.3 = 7,35.3 //

yogāṅgaireva bhittvā tānyogino yogamāpnuyuḥ /
yamaṃ niyamamāsanaprāṇāyāmau tataḥ param // 5.4 = 7,35.4 //

pratyāhāraṃ dhāraṇākhyaṃ dhyānaṃ sārdhaṃ samādhinā /
aṣṭāṅgānyāhuretāni yogināṃ yogasādhane // 5.5 = 7,35.5 //

ahiṃsā satyamasteyaṃ brahmacaryaṃ dayārjavam /
kṣamā dhṛtirmitāhāraḥ śaucaṃ ceti yamā daśa // 5.6 = 7,35.6 //

tapaḥ santoṣa āstikyaṃ dānaṃ devasya pūjanam /
siddhāntaśravaṇaṃ caiva hrīrmatiśca japo hutam // 5.7 = 7,35.7 //

daśaite niyamāḥ proktā mayā parvatanāyaka /
padmāsanaṃ svastikaṃ ca bhadraṃ vajrāsanaṃ tathā // 5.8 = 7,35.8 //

vīrāsanamiti proktaṃ kramādāsanapañcakam /
ūrvorupari vinyasya samyakpādatale śubhe // 5.9 = 7,35.9 //

aṅguṣṭhau ca nibadhnīyāddhastābhyāṃ vyutkramāt tataḥ /
padmāsanamiti proktaṃ yogināṃ hṛdayaṅgamam // 5.10 = 7,35.10 //

jānūrvorantare samyak kṛtvā pādatale śubhe /
kṛjukāyo viśedyogī svastikaṃ tatpracakṣate // 5.11 = 7,35.11 //

sīvanyāḥ pārśvayornyasya gulphayugmaṃ suniścitam /
vṛṣaṇādhaḥ pādapārṣṇī pārṣṇibhyāṃ paribandhayet // 5.12 = 7,35.12 //

bhadrāsanamiti proktaṃ yogibhiḥ paripūjitam /
urvoḥ pādau kramānnyasya jānvoḥ pratyaṅmukhāṅgulī // 5.13 = 7,35.13 //

karau vidadhyādākhyātaṃ vajrāsanamanuttamam /
ekaṃ pādamadhaḥ kṛtvā vinyasyoruṃ tathottare // 5.14 = 7,35.14 //

kṛjukāyo viśedyogī vīrāsanamitīritam /
iḍayākarṣayet vāyuṃ bāhyaṃ ṣoḍaśamātrayā // 5.15 = 7,35.15 //

dhārayetpūritaṃ yogī catuḥṣaṣṭyā tu mātrayā /
suṣumnāmadhyagaṃ samyagdvātriṃśanmātrayā śanaiḥ // 5.16 = 7,35.16 //

nāḍyā piṅgalayā caiva recayedyogavittamaḥ /
prāṇāyāmamimaṃ prāhūryogaśāstraviśāradāḥ // 5.17 = 7,35.17 //

bhūyobhūyaḥ kramāt tasya bāhyamevaṃ samācaret /
mātrāvṛddhiḥ krameṇaiva samyagdvādaśa ṣoḍaśa // 5.18 = 7,35.18 //

japadhyānādibhiḥ sārdhaṃ sagarbhaṃ taṃ vidurbudhāḥ /
tadapetaṃ vigarbhaṃ ca prāṇāyāmaṃ pare viduḥ // 5.19 = 7,35.19 //

kramādabhyasyataḥ puṃso dehe svedodgamo 'dhamaḥ /
madhyamaḥ kampasaṃyukto bhūmityāgaḥ paro mataḥ // 5.20 = 7,35.20 //

uttamasya guṇāvāptiryāvacchīlanam iṣyate /
indriyāṇāṃ vicaratāṃ viṣayeṣu nirargalam // 5.21 = 7,35.21 //

balādāharaṇaṃ tebhyaḥ pratyāhāro 'bhidhīyate /
aṅguṣṭhagulphajānūrumūlādhāraliṅganābhiṣu // 5.22 = 7,35.22 //

hṛdgrīvākaṇṭhadeśeṣu lambikāyāṃ tato nasi /
bhrūmadhye mastake mūrdhni dvādaśānte yathāvidhi // 5.23 = 7,35.23 //

dhāraṇaṃ prāṇamaruto dhāraṇeti nigadyate /
samāhitena manasā caitanyāntaravartinā // 5.24 = 7,35.24 //

ātmanyabhīṣṭadevānāṃ dhyānaṃ dhyānamihocyate /
samatvabhāvanā nityaṃ jīvātmaparamātmanoḥ // 5.25 = 7,35.25 //

samādhimāhurmunayaḥ proktam aṣṭāṅgalakṣaṇam /
idānīṃ kathaye te 'haṃ mantrayogamanuttamam // 5.26 = 7,35.26 //

viśvaṃ śarīramityuktaṃ pañcabhūtātmakaṃ naga /
candrasūryāgnitejobhirjīvabrahmaikyarūpakam // 5.27 = 7,35.27 //

tisraḥ koṭyastadardhena śarīre nāḍayo matāḥ /
tāsu mukhyā daśa proktāstābhyastisro vyavasthitāḥ // 5.28 = 7,35.28 //

pradhānā merudaṇḍe 'tra candrasūryāgrarūpiṇī
iḍā vāme sthitā nāḍī śubhrā tu candrarūpiṇī // 5.29 = 7,35.29 //

śaktirūpā tu sā nāḍī sākṣādamṛtavigrahā /
dakṣiṇe yā piṅgalākhyā puṃrūpā sūryavigrahā // 5.30 = 7,35.30 //

sarvatejomayī sā tu suṣumnā vahnirūpiṇī /
tasyā madhye vicitrākhye icchājñānakriyātmakam // 5.31 = 7,35.31 //

madhye svayaṃbhūliṅgaṃ tu koṭisūryasamaprabham /
tadūrdhvaṃ māyābījaṃ tu harātmā bindunādakam // 5.32 = 7,35.32 //

tadūrdhvaṃ tu śikhākārā kuṇḍalī raktavigrahā /
devyātmikā tu sā proktā madabhinnā nagādhipa // 5.33 = 7,35.33 //

tadbāhye hemarūpābhaṃ vādisāntacaturdalam /
drutahemasamaprakhyaṃ padmaṃ tatra vicintayet // 5.34 = 7,35.34 //

tadūrdhvaṃ tvanalaprakhyaṃ ṣaḍdalaṃ hīrakaprabham /
bādilāntaṣaḍvarṇena svādhiṣṭhānamanuttamam // 5.35 = 7,35.35 //

mūlamādhāraṣaṭkoṇaṃ mūlādhāraṃ tato viduḥ /
svaśabdena paraṃ liṅgaṃ svādhiṣṭhānaṃ tato viduḥ // 5.36 = 7,35.36 //

tadūrdhvaṃ nābhideśe tu maṇipūraṃ mahāprabham /
meghābhaṃ vidyudābhaṃ ca bahutejomayaṃ tataḥ // 5.37 = 7,35.37 //

maṇivadbhinnaṃ tatpadmaṃ maṇipadmaṃ tathocyate /
daśabhiśca dalairyuktaṃ ḍādiphāntākṣarānvitam // 5.38 = 7,35.38 //

viṣṇunādhiṣṭhitaṃ padmaṃ viṣṇvālokanakāraṇam /
tadūrdhve 'nāhataṃ padmamudyadādityasannibham // 5.39 = 7,35.39 //

kādiṭhāntadalairekaṃ patraiśca samadhiṣṭhitam /
tanmadhye bāṇaliṅgaṃ tu sūryāyutasamaprabham // 5.40 = 7,35.40 //

śabdabrahmamayaṃ śabdānāhataṃ tatra dṛśyate /
anāhatākhyaṃ tatpadmaṃ munibhiḥ parikīrtitam // 5.41 = 7,35.41 //

ānandasadanaṃ tattu puruṣādhiṣṭhitaṃ param /
tadūrdhvaṃ tu viśuddhākhyaṃ dalaṃ ṣoḍaśapaṅkajam // 5.42 = 7,35.42 //

svaraiḥ ṣoḍaśabhiryuktaṃ dhūmravarṇam mahāprabham /
viśuddhaṃ tanute yasmājjīvasya haṃsalokanāt // 5.43 = 7,35.43 //

viśuddhaṃ padmamākhyātamākāśākhyaṃ mahādbhutam /
ājñācakraṃ tadūrdhve tu ātmanādhiṣṭhitaṃ param // 5.44 = 7,35.44 //

ājñāsaṃkramaṇaṃ tatra tenājñeti prakīrtitam /
dvidalaṃ hakṣasaṃyuktaṃ padmaṃ tatsumanoharam // 5.45 = 7,35.45 //

kailāsākhyaṃ tadūrdhvaṃ tu rodhinī tu tadūrdhvataḥ /
evaṃ tvādhāracakrāṇi proktāni tava suvrata // 5.46 = 7,35.46 //

sahasrārayutaṃ bindusthānaṃ tadūrdhvamīritam /
ityetatkathitaṃ sarvam yogamārgamanuttamam // 5.47 = 7,35.47 //

ādau pūrakayogenāpyādhāre yojayenmanaḥ /
gudameḍhrāntare śaktistāmākuñcya prabodhayet // 5.48 = 7,35.48 //

liṅgabhedakrameṇaiva binducakraṃ ca prāpayet /
śambhunā tāṃ parāṃ śaktimekībhūtāṃ vicintayet // 5.49 = 7,35.49 //

tatrotthitāmṛtam yattu drutalākṣārasopamam /
pāyayitvā tu tāṃ śaktiṃ māyākhyāṃ yogasiddhidam // 5.50 = 7,35.50 //

ṣaḍcakradevatāstatra santarpyāmṛtadhārayā /
ānayettena mārgeṇa mūlādhāraṃ tataḥ sudhī // 5.51 = 7,35.51 //

evamabhyasyamānasyāpyahanyahani niścitam /
pūrvoktadūṣitā mantrāḥ sarve sidhyanti nānyathā // 5.52 = 7,35.52 //

jarāmaraṇaduḥkhādyairmucyate bhavabandhanāt /
ye guṇāḥ santi devyā me jaganmāturyathā tathā // 5.53 = 7,35.53 //

te guṇāḥ sādhakavare bhavantyeva na cānyathā /
ityevaṃ kathitaṃ tāta vāyudhāraṇamuttamam // 5.54 = 7,35.54 //

idānīṃ dhāraṇākhyaṃ tu śṛṇuṣvāvahito mama /
dikkālādyanavacchinnadevyāṃ ceto vidhāya ca // 5.55 = 7,35.55 //

tanmayo bhavati kṣipraṃ jīvabrahmaikyayojanāt /
athavā samalaṃ ceto yadi kṣipraṃ na siddhyati // 5.56 = 7,35.56 //

tadāvayavayogena yogī yogānsamabhyaset /
madīyahastapādādāvaṅge tu madhure naga // 5.57 = 7,35.57 //

cittaṃ saṃsthāpayenmantrī sthānasthānajayātpunaḥ /
viśuddhacittaḥ sarvasminrūpe samsthāpayenmanaḥ // 5.58 = 7,35.58 //

yāvanmano layaṃ yāti devyāṃ saṃvidi parvata /
tāvadiṣṭamanuṃ mantrī japahomaiḥ samabhyaset // 5.59 = 7,35.59 //

mantrābhyāsena yogena jñeyajñānāya kalpate /
na yogena vinā mantro na mantreṇa vinā hi saḥ // 5.60 = 7,35.60 //

dvayorabhyāsayogo hi brahmasaṃsiddhikāraṇam /
tamaḥ parivṛte gehe ghaṭo dīpena dṛśyate // 5.61 = 7,35.61 //

evaṃ māyāvṛto hyātmā manunā gocarīkṛtaḥ /
iti yogavidhiḥ kṛtsnaḥ sāṅgaḥ prokto mayā 'dhunā // 5.62 = 7,35.62 //

gurūpadeśato jñeyo nānyathā śāstrakoṭibhiḥ // 5.63 = 7,35.63 //

iti devīgītāyāṃ pañcamo 'dhyāyaḥ -
iti śrīdevībhāgavate mahāpurāṇe saptamaskandhe devīgītāyāṃ pañcatriṃśo 'dhyāyaḥ

atha ṣaṣṭho 'dhyāyaḥ -
saptamaḥ skandhaḥ
atha ṣaṭtriṃśo 'dhyāyaḥ


devyuvāca

ityādiyogayuktātmā dhyāyenmāṃ brahmarūpiṇīm /
bhaktyā nirvyājayā rājannāsane samupasthitaḥ // 6.1 = 7,36.1 //

āviḥ sannihitaṃ guhācaraṃ nāma mahatpadam /
atraitat sarvam arpitam ejat prāṇannimiṣacca yat // 6.2 = 7,36.2 //


etajjānatha sadasadvareṇyaṃ paraṃ vijñānādyadvariṣṭhaṃ prajānām /
yadarcimadyadaṇubhyo 'ṇu ca yasmiṃllokā nihitā lokinaśca // 6.3 = 7,36.3 //

tadetadakṣaraṃ brahma sa prāṇastadu vāṅmanaḥ /
tadetat satyam amṛtaṃ tadveddhavyaṃ saumya viddhi // 6.4 = 7,36.4 //

dhanurgṛhītvaupaniṣadaṃ mahāstraṃ śaraṃ hyupāsāniśitaṃ sandhayīta /
āyamya tadbhāvagatena cetasā lakṣyaṃ tadevākṣaraṃ saumya viddhi // 6.5 = 7,36.5 //

praṇavo dhanuḥ śaro hyātmā brahma tallakṣyam ucyate /
apramattena veddhavyaṃ śaravattanmayo bhavet // 6.6 = 7,36.6 //

yasmin dyauśca pṛthivī cāntarikṣam otaṃ manaḥ saha prāṇaiśca sarvaiḥ /
tam evaikaṃ jānathātmānam anyā vāco vimuñcathāmṛtasyaiṣa setuḥ // 6.7 = 7,36.7 //

arā iva rathanābhau saṃhatā yatra nāḍyaḥ /
sa eṣo 'ntaścarate bahudhā jāyamānaḥ // 6.8 = 7,36.8 //

Omityevaṃ dhyāyathātmānaṃ svasti vaḥ pārāya tamasaḥ parastāt /
divye brahmapure vyomni ātmā sampratiṣṭhitaḥ // 6.9 = 7,36.9 //

manomayaḥ prāṇaśarīranetā pratiṣṭhito 'nne hṛdayaṃ sannidhāya /
tadvijñānena paripaśyanti dhīrā ānandarūpam amṛtaṃ yadvibhāti // 6.10 = 7,36.10 //

bhidyate hṛdayagranthiśchidyante sarvasaṃśayāḥ /
kṣīyante cāsya karmāṇi tasmin dṛṣṭe parāvare // 6.11 = 7,36.11 //

hiraṇmaye pare kośe virājaṃ brahma niṣkalam /
tacchubhraṃ jyotiṣāṃ jyotis tadyadātmavido viduḥ // 6.12 = 7,36.12 //

na tatra sūryo bhāti na candratārakaṃ nemā vidyuto bhānti kuto 'yam agniḥ /
tameva bhāntam anubhāti sarvaṃ tasya bhāsā sarvam idaṃ vibhāti // 6.13 = 7,36.13 //

brahmaivedam amṛtaṃ purastādbrahma paścādbrahma dakṣiṇataścottareṇa /
adhaścordhvaṃ ca prasṛtaṃ brahmaivedaṃ viśvaṃ variṣṭham // 6.14 = 7,36.14 //

etādṛganubhavo yasya sa kṛtārtho narottamaḥ /
brahmabhūtaḥ prasannātmā na śocati na kāṅkṣati // 6.15 = 7,36.15 //

dvitīyādvai bhayaṃ rājaṃstadabhāvādbibheti na /
na tadviyogo me 'pyasti madviyogo 'pi tasya na // 6.16 = 7,36.16 //

aham eva sa so 'haṃ vai niścitaṃ viddhi parvata /
maddarśanaṃ tu tatra syādyatra jñānī sthito mama // 6.17 = 7,36.17 //

nāhaṃ tīrthe na kailāse vaikuṇṭhe vā na karhicit /
vasāmi kintu majjñānihṛdayāmbhojamadhyame // 6.18 = 7,36.18 //

matpūjākoṭiphaladaṃ sakṛnmajjñānino 'rcanam /
kulaṃ pavitraṃ tasyāsti jananī kṛtakṛtyakā // 6.19 = 7,36.19 //

viśvambharā puṇyavatī cillayo yasya cetasaḥ /
brahmajñānaṃ tu yatpṛṣṭaṃ tvayā parvata sattama // 6.20 = 7,36.20 //

kathitaṃ tanmayā sarvaṃ nāto vaktavyam asti hi /
idaṃ jyeṣṭhāya putrāya bhaktiyuktāya śīline // 6.21 = 7,36.21 //

śiṣyāya ca yathoktāya vaktavyaṃ nānyathā kvacit /
yasya deve parā bhaktiryathā deve tathā gurau // 6.22 = 7,36.22 //

tasyaite kathitā hyarthāḥ prakāśante mahātmanaḥ /
yenopadiṣṭāvidyeyaṃ sa eva parameśvaraḥ // 6.23 = 7,36.23 //

yasyāyaṃ sukṛtaṃ kartum asamarthastato ṛṇī /
pitrorapyadhikaḥ prokto brahmajanmapradāyakaḥ // 6.24 = 7,36.24 //

pitṛjātaṃ janma naṣṭaṃ netthaṃ jātaṃ kadācana /
tasmai na druhyedityādi nigamo 'pyavadannaga // 6.25 = 7,36.25 //

tasmācchāstrasya siddhānto brahmadātā guruḥ paraḥ /
śive ruṣṭe gurustrātā gurau ruṣṭe na śaṅkaraḥ // 6.26 = 7,36.26 //

tasmāt sarvaprayatnena śrīguruṃ toṣayennaga /
kāyena manasā vācā sarvadā tatparo bhavet // 6.27 = 7,36.27 //

anyathā tu kṛtaghnaḥ syāt kṛtaghne nāsti niṣkṛtiḥ /
indreṇātharvaṇāyoktā śiraścheda pratijñayā // 6.28 = 7,36.28 //

aśvibhyāṃ kathane tasya śiraśchinnaṃ ca vajriṇā /
aśvīyaṃ tacchiro naṣṭaṃ dṛṣṭvā vaidyau surottamau // 6.29 = 7,36.29 //

punaḥ saṃyojitaṃ svīyaṃ tābhyāṃ muniśirastadā /
iti saṅkaṭasampādyā brahmavidyā nagādhipa // 6.30 = 7,36.30 //

lābdhā yena sa dhanyaḥ syāt kṛtakṛtyaśca bhūdhara

iti devīgītāyāṃ ṣaṣṭo 'dhyāyaḥ
iti śrīdevībhāgavate mahāpurāṇe saptamaskandhe devīgītāyāṃ ṣaṭtriṃśo 'dhyāyaḥ

atha saptamo 'dhyāyaḥ -
saptamaḥ skandhaḥ
atha saptatriṃśo 'dhyāyaḥ


himālaya uvāca

svīyāṃ bhaktiṃ vadasvāmba yena jñānaṃ sukhena hi /
jāyeta manujasyāsya madhyamasyāvirāgiṇaḥ // 7.1 = 7,37.1 //


devyuvāca

mārgāstrayo me vikhyātā mokṣaprāptau nagādhipa /
karmayogo jñānayogo bhaktiyogaśca sattama // 7.2 = 7,37.2 //

trayāṇāmapyayaṃ yogyaḥ kartuṃ śakyo 'sti sarvathā /
sulabhatvānmānasatvāt kāyacittādyapīḍanāt // 7.3 = 7,37.3 //

guṇabhedānmanuṣyāṇāṃ sā bhaktistrividhā matā /
parapīḍāṃ samuddiśya dambhaṃ kṛtvā puraḥsaram // 7.4 = 7,37.4 //

mātsaryakrodhayukto yastasya bhaktis tu tāmasī /
parapīḍādirahitaḥ svakalyāṇārtham eva ca // 7.5 = 7,37.5 //

nityaṃ sakāmo hṛdayaṃ yaśorthī bhogalolupaḥ /
tattatphalasamāvāptyai māmupāste 'tibhaktitaḥ // 7.6 = 7,37.6 //

bhedabuddhyā tu māṃ svasmādanyāṃ jānāti pāmaraḥ /
tasya bhaktiḥ samākhyātā nagādhipa tu rājasī // 7.7 = 7,37.7 //

parameśārpaṇaṃ karma pāpasaṅkṣālanāya ca /
vedoktatvādavaśyaṃ tatkartavyaṃ tu mayāniśam // 7.8 = 7,37.8 //

iti niścitabuddhistu bhedabuddhim upāśritaḥ /
karoti prītaye karma bhaktiḥ sā naga sattvikī // 7.9 = 7,37.9 //

parabhakteḥ prāpikeyaṃ bhedabuddhyavalambanāt /
pūrvaprokte hyubhe bhaktī na paraprāpike mate // 7.10 = 7,37.10 //

adhunā parabhaktiṃ tu procyamānāṃ nibodha me /
madguṇaśravaṇaṃ nityaṃ mama nāmānukīrtanam // 7.11 = 7,37.11 //

kalyāṇaguṇaratnānāmākarāyāṃ mayi sthiram /
cetaso vartanaṃ caiva tailadhārāsamaṃ sadā // 7.12 = 7,37.12 //

hetustu tatra ko vāpi na kadācidbhavedapi /
sāmīpyasārṣṭisāyujyasālokyānāṃ na caiṣaṇā // 7.13 = 7,37.13 //

matsevāto 'dhikaṃ kiñcinnaiva jānāti karhicit /
sevyasevakatābhāvāt tatra mokṣaṃ na vāñchati // 7.14 = 7,37.14 //

parānuraktyā mām eva cintayedyo hyatandritaḥ /
svābhedenaiva māṃ nityaṃ jānāti na vibhedataḥ // 7.15 = 7,37.15 //

madrūpatvena jīvānāṃ cintanaṃ kurute tu yaḥ /
yathā svasyātmani prītistathaiva ca parātmani // 7.16 = 7,37.16 //

caitanyasya samānatvānna bhedaṃ kurute tu yaḥ /
sarvatra vartamānānāṃ sarvarūpāṃ ca sarvadā // 7.17 = 7,37.17 //

namate yajate caivāpyācāṇḍālāntam īśvara /
na kutrāpi drohabuddhiṃ kurute bhedavarjanāt // 7.18 = 7,37.18 //

matsthānadarśane śraddhā madbhaktadarśane tathā /
macchāstraśravaṇe śraddhā mantratantrādiṣu prabho // 7.19 = 7,37.19 //

māyi premākulamatī romāñcitatanuḥ sadā /

saṃvideva paraṃ rūpamupādhirahitaṃ mama // 9.44 = 7,39.44 //

ataḥ saṃvidi madrūpe cetaḥ sthāpyaṃ nirāśrayam /

ataḥ paraṃ bāhyapūjāvistāraḥ kathyate mayā /
sāvadhānena manasā śṛṇu parvatasattama // 9.47 = 7,39.47 //

iti devīgītāyāṃ navamo 'dhyāyaḥ -
