pracaṇḍatāṇḍavāṭope prakṣiptā yena diggajāḥ  /
bhavantu vighnabhaṅgāya bhavasya caraṇāmbujāḥ  // Mang.1 //
pātālādutpatiṣṇor makaravasatayo yasya pucchābhighātād ūrdhvaṃ brahmāṇḍakhaṇḍavyatikaravihitavyatyayenāpatanti  /
viṣṇormatsyāvatāre sakalavasumatīmaṇḍalaṃ vyaśnuvānās tasyāsyodīritānāṃ dhvanir apaharatād aśriyaṃ vaḥ śrutīnām  // 1.1 //
nārāyaṇaṃ namaskṛtya naraṃ caiva narottamam  /
devīṃ sarasvatīṃ caiva tato jayam udīrayet  // 1.2 //
ajo 'pi yaḥ kriyāyogān nārāyaṇa iti smṛtaḥ  /
triguṇāya trivedāya namas tasmai svayambhuve  // 1.3 //
sūtamekāgramāsīnaṃ naimiṣāraṇyavāsinaḥ  /
munayo dīrghasattrānte papracchur dīrghasaṃhitām  // 1.4 //
pravṛttāsu purāṇīṣu dharmyāsu lalitāsu ca  /
kathāsu śaunakādyās tu abhinandya muhurmuhuḥ  // 1.5 //
kathitāni purāṇāni yānyasmākaṃ tvayānagha  /
tānyevāmṛtakalpāni śrotum icchāmahe punaḥ  // 1.6 //
kathaṃ sasarja bhagavaṃl lokanāthaścarācaram  /
kasmācca bhagavān viṣṇur matsyarūpatvam āśritaḥ  // 1.7 //
bhairavatvaṃ bhavasyāpi purāritvaṃ ca kena hi  /
kasya hetoḥ kapālitvaṃ jagāma vṛṣabhadhvajaḥ  // 1.8 //
sarvam etat samācakṣva sūta vistaraśaḥ kramāt  /
tvadvākyenāmṛtasyeva na tṛptiriha jāyate  // 1.9 //
sūta uvāca
puṇyaṃ pavitram āyuṣyam idānīṃ śṛṇuta dvijāḥ  /
mātsyaṃ purāṇamakhilaṃ yajjagāda gadādharaḥ  // 1.10 //
purā rājā manur nāma cīrṇavān vipulaṃ tapaḥ  /
putre rājyaṃ samāropya kṣamāvān ravinandanaḥ  // 1.11 //
malayasyaikadeśe tu sarvātmaguṇasaṃyutaḥ  /
samaduḥkhasukho vīraḥ prāptavān yogam uttamam  // 1.12 //
babhūva varadaś cāsya varṣāyutaśate gate  /
varaṃ vṛṇīṣva provāca prītaḥ sa kamalāsanaḥ  // 1.13 //
evamukto 'bravīd rājā praṇamya sa pitāmaham  /
ekam evāham icchāmi tvatto varamanuttamam  // 1.14 //
bhūtagrāmasya sarvasya sthāvarasya carasya ca  /
bhaveyaṃ rakṣaṇāyālaṃ pralaye samupasthite  // 1.15 //
evamastviti viśvātmā tatraivāntaradhīyata  /
puṣpavṛṣṭiḥ sumahatī khātpapāta surārpitā  // 1.16 //
kadācidāśrame tasya kurvataḥ pitṛtarpaṇam  /
papāta pāṇyor upari śapharī jalasaṃyutā  // 1.17 //
dṛṣṭvā tacchapharīrūpaṃ sa dayālurmahīpatiḥ  /
rakṣaṇāyākarodyatnaṃ sa tasminkarakodare  // 1.18 //
ahorātreṇa caikena ṣoḍaśāṅgulavistṛtaḥ  /
so 'bhavanmatsyarūpeṇa pāhi pāhīti cābravīt  // 1.19 //
sa tamādāya maṇike prākṣipajjalacāriṇam  /
tatrāpi caikarātreṇa hastatrayam avardhata  // 1.20 //
punaḥ prāhārtanādena sahasrakiraṇātmajam  /
sa matsyaḥ pāhi pāhīti tvāmahaṃ śaraṇaṃ gataḥ  // 1.21 //
tataḥ sa kūpe taṃ matsyaṃ prāhiṇod ravinandanaḥ  /
yadā na māti tatrāpi kūpe matsyaḥ sarovare  // 1.22 //
kṣipto 'sau pṛthutāmāgāt punar yojanasaṃmitām  /
tatrāpyāha punar dīnaḥ pāhi pāhi nṛpottama  // 1.23 //
tataḥ sa manunā kṣipto gaṅgāyāmapyavardhata  /
yadā tadā samudre taṃ prākṣipanmedinīpatiḥ  // 1.24 //
yadā samudramakhilaṃ vyāpyāsau samupasthitaḥ  /
tadā prāha manurbhītaḥ ko 'pi tvamasureśvaraḥ  // 1.25 //
athavā vāsudevastvam anya īdṛkkathaṃ bhavet  /
yojanāyutaviṃśatyā kasya tulyaṃ bhavedvapuḥ  // 1.26 //
jñātastvaṃ matsyarūpeṇa māṃ khedayasi keśava  /
hṛṣīkeśa jagannātha jagaddhāma namo 'stu te  // 1.27 //
evamuktaḥ sa bhagavān matsyarūpī janārdanaḥ  /
sādhu sādhviti covāca samyagjñātas tvayānagha  // 1.28 //
acireṇaiva kālena medinī medinīpate  /
bhaviṣyati jale magnā saśailavanakānanā  // 1.29 //
naur iyaṃ sarvadevānāṃ nikāyena vinirmitā  /
mahājīvanikāyasya rakṣaṇārthaṃ mahīpate  // 1.30 //
svedāṇḍajodbhido ye vai ye ca jīvā jarāyujāḥ  /
asyāṃ nidhāya sarvāṃs tān anāthān pāhi suvrata  // 1.31 //
yugāntavātābhihatā yadā bhavati naur nṛpa  /
śṛṅge 'sminmama rājendra tademāṃ saṃyamiṣyasi  // 1.32 //
tato layānte sarvasya sthāvarasya carasya ca  /
prajāpatistvaṃ bhavitā jagataḥ pṛthivīpate  // 1.33 //
evaṃ kṛtayugasyādau sarvajño dhṛtimānnṛpaḥ  /
manvantarādhipaścāpi devapūjyo bhaviṣyasi  // 1.34 //

sūta uvāca
evamukto manustena papraccha madhusūdanam  /
bhagavankiyadbhirvarṣair bhaviṣyatyantarakṣayaḥ  // 2.1 //
sattvāni ca kathaṃ nātha rakṣiṣye madhusūdana  /
tvayā saha punaryogaḥ kathaṃ vā bhavitā mama  // 2.2 //
matsya uvāca
adyaprabhṛtyanāvṛṣṭir bhaviṣyati mahītale  /
yāvadvarṣaśataṃ sāgraṃ durbhikṣam aśubhāvaham  // 2.3 //
tato 'lpasattvakṣayadā raśmayaḥ sapta dāruṇāḥ  /
saptasapterbhaviṣyanti prataptāṅgāravarṣiṇaḥ  // 2.4 //
aurvānalo 'pi vikṛtiṃ gamiṣyati yugakṣaye  /
viṣāgniścāpi pātālāt saṃkarṣaṇamukhacyutaḥ  /
bhavasyāpi lalāṭotthas tṛtīyanayanānalaḥ  // 2.5 //
trijagannirdahan kṣobhaṃ sameṣyati mahāmune  /
evaṃ dagdhā mahī sarvā yadā syādbhasmasaṃnibhā  // 2.6 //
ākāśamūṣmaṇā taptaṃ bhaviṣyati paraṃtapa  /
tataḥ sadevanakṣatraṃ jagadyāsyati saṃkṣayam  // 2.7 //
saṃvarto bhīmanādaśca droṇaścaṇḍo balāhakaḥ  /
vidyutpatākaḥ śoṇastu saptaite layavāridāḥ  // 2.8 //
agniprasvedasambhūtāḥ plāvayiṣyanti medinīm  /
samudrāḥ kṣobhamāgatya caikatvena vyavasthitāḥ  // 2.9 //
etadekārṇavaṃ sarvaṃ kariṣyanti jagattrayam  /
vedanāvamimāṃ gṛhya sattvabījāni sarvaśaḥ  // 2.10 //
āropya rajjuyogena matpradattena suvrata  /
saṃyamya nāvaṃ macchṛṅge matsyabhāvābhirakṣitaḥ  // 2.11 //
ekaḥ sthāsyasi deveṣu dagdheṣvapi paraṃtapa  /
somasūryāvahaṃ brahmā caturlokasamanvitaḥ  // 2.12 //
narmadā ca nadī puṇyā mārkaṇḍeyo mahānṛṣiḥ  /
bhavo vedāḥ purāṇāni vidyābhiḥ sarvatovṛtam  // 2.13 //
tvayā sārdhamidaṃ viśvaṃ sthāsyatyantarasaṃkṣaye  /
evamekārṇave jāte cākṣuṣāntarasaṃkṣaye  // 2.14 //
vedānpravartayiṣyāmi tvatsargādau mahīpate  /
evamuktvā sa bhagavāṃs tatraivāntaradhīyata  // 2.15 //
manur apyāsthito yogaṃ vāsudevaprasādajam  /
abhyasan yāvad ābhūta- samplavaṃ pūrvasūcitam  // 2.16 //
kāle yathokte saṃjāte vāsudevamukhodgate  /
śṛṅgī prādurbabhūvātha matsyarūpī janārdanaḥ  // 2.17 //
bhujaṃgo rajjurūpeṇa manoḥ pārśvamupāgamat  /
bhūtānsarvānsamākṛṣya yogenāropya dharmavit  // 2.18 //
bhujaṃgarajjvā matsyasya śṛṅge nāvamayojayat  /
uparyupasthitastasyāḥ praṇipatya janārdanam  // 2.19 //
ābhūtasamplave tasminn atīte yogaśāyinā  /
pṛṣṭena manunā proktaṃ purāṇaṃ matsyarūpiṇā  /
tadidānīṃ pravakṣyāmi śṛṇudhvamṛṣisattamāḥ  // 2.20 //
yadbhavadbhiḥ purā pṛṣṭaḥ sṛṣṭyādikamahaṃ dvijāḥ  /
tad evaikārṇave tasmin manuḥ papraccha keśavam  // 2.21 //
manuruvāca
utpattiṃ pralayaṃ caiva vaṃśānmanvantarāṇi ca  /
vaṃśyānucaritaṃ caiva bhuvanasya ca vistaram  // 2.22 //
dānadharmavidhiṃ caiva śrāddhakalpaṃ ca śāśvatam  /
varṇāśramavibhāgaṃ ca tatheṣṭāpūrtasaṃjñitam  // 2.23 //
devatānāṃ pratiṣṭhādi yaccānyadvidyate bhuvi  /
tatsarvaṃ vistareṇa tvaṃ dharmaṃ vyākhyātumarhasi  // 2.24 //
matsya uvāca
mahāpralayakālānta etadāsīttamomayam  /
prasuptamiva cātarkyam aprajñātamalakṣaṇam  // 2.25 //
avijñeyamavijñātaṃ jagat sthāsnu cariṣṇu ca  /
tataḥ svayambhūr avyaktaḥ prabhavaḥ puṇyakarmaṇām  // 2.26 //
vyañjayannetadakhilaṃ prādurāsīt tamonudaḥ  /
yo 'tīndriyaḥ paro vyaktād aṇur jyāyān sanātanaḥ  /
nārāyaṇa iti khyātaḥ sa ekaḥ svayam udbabhau  // 2.27 //
yaḥ śarīrād abhidhyāya sisṛkṣurvividhaṃ jagat  /
apa eva sasarjādau tāsu vīryam avāsṛjat  // 2.28 //
tadevāṇḍaṃ samabhavad dhemarūpyamayaṃ mahat  /
saṃvatsarasahasreṇa sūryāyutasamaprabham  // 2.29 //
praviśyāntarmahātejāḥ svayam evātmasambhavaḥ  /
prabhāvād api tadvyāptyā viṣṇutvam agamat punaḥ  // 2.30 //
tadantarbhagavāneṣa sūryaḥ samabhavatpurā  /
ādityaścādibhūtatvād brahmā brahma paṭhann abhūt  // 2.31 //
divaṃ bhūmiṃ samakarot tadaṇḍaśakaladvayam  /
sa cākaroddiśaḥ sarvā madhye vyoma ca śāśvatam  // 2.32 //


4
manur uvāca
aho kaṣṭataraṃ caitad aṅgajāgamanaṃ vibho  /
kathaṃ na doṣamagamat karmaṇānena padmabhūḥ  // 4.1 //
parasparaṃ ca sambandhaḥ sagotrāṇām abhūt katham  /
vaivāhikastatsutānāṃ chinddhi me saṃśayaṃ vibho  // 4.2 //
matsya uvāca
divyeyamādisṛṣṭistu rajoguṇasamudbhavā  /
atīndriyendriyā tadvad atīndriyaśarīrikā  // 4.3 //
divyatejomayī bhūpa divyajñānasamudbhavā  /
na martyairabhitaḥ śakyā vaktuṃ vai māṃsacakṣubhiḥ  // 4.4 //
yathā bhujaṃgāḥ sarpāṇām ākāśaṃ viśvapakṣiṇām  /
vidanti mārgaṃ divyānāṃ divyā eva na mānavāḥ  // 4.5 //
kāryākārye na devānāṃ śubhāśubhaphalaprade  /
yasmāttasmānna rājendra tadvicāro nṛṇāṃ śubhaḥ  // 4.6 //
anyacca sarvavedānām adhiṣṭhātā caturmukhaḥ  /
gāyatrī brahmaṇastadvad aṅgabhūtā nigadyate  // 4.7 //
amūrtaṃ mūrtimad vāpi mithunaṃ tatpracakṣate  /
viriñcir yatra bhagavāṃs tatra devī sarasvatī  /
bhāratī yatra yatraiva tatra tatra prajāpatiḥ  // 4.8 //
yathātapo na rahitaś chāyayā dṛśyate kvacit  /
gāyatrī brahmaṇaḥpārśvaṃ tathaiva na vimuñcati  // 4.9 //
vedarāśiḥ smṛto brahmā sāvitrī tadadhiṣṭhitā  /
tasmānna kaściddoṣaḥ syāt sāvitrīgamane vibho  // 4.10 //
tathāpi lajjāvanataḥ prajāpatir abhūtpurā  /
svasutopagamād brahmā śaśāpa kusumāyudham  // 4.11 //
yasmānmamābhibhavatā manaḥ saṃkṣobhitaṃ śaraiḥ  /
tasmāttvaddehamacirād rudro bhasmīkariṣyati  // 4.12 //
tataḥ prasādayāmāsa kāmadevaścaturmukham  /
na māmakāraṇe śaptuṃ tvamihārhasi mānada  // 4.13 //
ahamevaṃvidhaḥ sṛṣṭas tvayaiva caturānana  /
indriyakṣobhajanakaḥ sarveṣāmeva dehinām  // 4.14 //
strīpuṃsoravicāreṇa mayā sarvatra sarvadā  /
kṣobhyaṃ manaḥ prayatnena tvayaivoktaṃ purā vibho  // 4.15 //
tasmādanaparādho 'haṃ tvayā śaptastathā vibho  /
kuru prasādaṃ bhagavan svaśarīrāptaye punaḥ  // 4.16 //
brahmovāca
vaivasvate 'ntare prāpte yādavānvayasambhavaḥ  /
rāmo nāma yadā martyo matsattvabalamāśritaḥ  // 4.17 //
avatīryāsuradhvaṃsī dvārakām adhivatsyati  /
tadbhrātustatsamasya tvaṃ tadā putratvameṣyasi  // 4.18 //
evaṃ śarīramāsādya bhuktvā bhogānaśeṣataḥ  /
tato bharatavaṃśānte bhūtvā vatsanṛpātmajaḥ  // 4.19 //
vidyādharādhipatyaṃ ca yāvad ābhūtasaṃplavam  /
sukhāni dharmataḥ prāpya matsamīpaṃ gamiṣyasi  // 4.20 //
evaṃ śāpaprasādābhyām upetaḥ kusumāyudhaḥ  /
śokapramodābhiyuto jagāma sa yathāgatam  // 4.21 //
manuruvāca
ko 'sau yaduriti prokto yadvaṃśe kāmasambhavaḥ  /
kathaṃ ca dagdho rudreṇa kimatha kusumāyudhaḥ  // 4.22 //
bharatasyānvaye kasya kā ca sṛṣṭiḥ purābhavat  /
etatsarvaṃ samācakṣva mūlataḥ saṃśayo hi me  // 4.23 //
matsya uvāca
yā sā dehārdhasambhūtā gāyatrī brahmavādinī  /
jananī yā manordevī śatarūpā śatendriyā  // 4.24 //
ratirmanastapo buddhir mahān diksambhramas tathā  /
tataḥ sa śatarūpāyāṃ saptāpatyānyajījanat  // 4.25 //
ye marīcyādayaḥ putrā mānasāstasya dhīmataḥ  /
teṣāmayamabhūllokaḥ sarvajñānātmakaḥ purā  // 4.26 //
tato 'sṛjadvāmadevaṃ triśūlavaradhāriṇam  /
sanatkumāraṃ ca vibhuṃ pūrveṣāmapi pūrvajam  // 4.27 //
vāmadevastu bhagavān asṛjanmukhato dvijān  /
rājanyān asṛjadbāhvor viṭchūdrān ūrupādayoḥ  // 4.28 //
vidyuto 'śanimeghāṃśca rohitendradhanūṃṣi ca  /
chandāṃsi ca sasarjādau parjanyaṃ ca tataḥ param  // 4.29 //
tataḥ sādhyagaṇānīśas trinetrānasṛjatpunaḥ  /
koṭīśca caturaśītiṃ jarāmaraṇavarjitāḥ  // 4.30 //
vāmo 'sṛjannamartyāṃs tān brahmaṇā vinivāritaḥ  /
naivaṃvidhā bhavetsṛṣṭir jarāmaraṇavarjitā  // 4.31 //
śubhāśubhātmikā yā tu saiva sṛṣṭiḥ praśasyate  /
evaṃ sthitaḥ sa tenādau sṛṣṭeḥ sthāṇurato 'bhavat  // 4.32 //
svāyambhuvo manurdhīmāṃs tapastaptvā suduścaram  /
patnīmevāpa rūpāḍhyām anantā nāma nāmataḥ  // 4.33 //
priyavratottānapādau manustasyām ajījanat  /
dharmasya kanyā caturā sūnṛtā nāma bhāminī  // 4.34 //
uttānapādāttanayān prāpa mantharagāminī  /
apasyatim apasyantaṃ kīrtimantaṃ dhruvaṃ tathā  // 4.35 //
uttānapādo 'janayat sūnṛtāyāṃ prajāpatiḥ  /
dhruvo varṣasahasrāṇi trīṇi kṛtvā tapaḥ purā  // 4.36 //
divyamāpa tataḥ sthānam acalaṃ brahmaṇo varāt  /
tameva purataḥ kṛtvā dhruvaṃ saptarṣayaḥ sthitāḥ  // 4.37 //
dhanyā nāma manoḥ kanyā dhruvācchiṣṭam ajījanat  /
agnikanyā tu succhāyā śiṣṭātsā suṣuve sutān  // 4.38 //
kṛpaṃ ripuṃ jayaṃ vṛttaṃ vṛkaṃ ca vṛkatejasam  /
cakṣuṣaṃ brahmadauhitryāṃ vīriṇyāṃ sa ripuṃjayaḥ  // 4.39 //
vīraṇasyātmajāyāṃ tu cakṣurmanumajījanat  /
manur vai rājakanyāyāṃ naḍvalāyāṃ sa cākṣuṣaḥ  // 4.40 //
janayāmāsa tanayān daśa śūrānakalmaṣān  /
ūruḥ pūruḥ śatadyumnas tapasvī satyavāgghaviḥ  // 4.41 //
agniṣṭudatirātraśca sudyumnaścāparājitaḥ  /
abhimanyustu daśamo naḍvalāyām ajāyata  // 4.42 //
ūror ajanayat putrān ṣaḍ āgneyī tu suprabhān  /
agniṃ sumanasaṃ khyātiṃ kratumaṅgirasaṃ gayam  // 4.43 //
pitṛkanyā sunīthā tu venamaṅgādajījanat  /
venamanyāyinaṃ viprā mamanthus tatkarād abhūt  /
pṛthurnāma mahātejāḥ sa putrau dvāv ajījanat  // 4.44 //
antardhānas tu mārīcaṃ śikhaṇḍinyāmajījanat  /
havirdhānāt ṣaḍ āgneyī dhiṣaṇājanayat sutān  /
prācīnabarhiṣaṃ sāṅgaṃ yamaṃ śukraṃ balaṃ śubham  // 4.45 //
prācīnabarhir bhagavān mahān āsīt prajāpatiḥ  /
havirdhānāḥ prajās tena bahavaḥ sampravartitāḥ  // 4.46 //
savarṇāyāṃ tu sāmudryāṃ daśādhatta sutānprabhuḥ  /
sarve pracetaso nāma dhanurvedasya pāragāḥ  // 4.47 //
tattaporakṣitā vṛkṣā babhur loke samantataḥ  /
devādeśācca tānagnir adahadravinandana  // 4.48 //
somakanyābhavat patnī mārīṣā nāma viśrutā  /
tebhyastu dakṣamekaṃ sā putram agryam ajījanat  // 4.49 //
dakṣādanantaraṃ vṛkṣān auṣadhāni ca sarvaśaḥ  /
ajījanatsomakanyā nadīṃ candravatīṃ tathā  // 4.50 //
somāṃśasya ca tasyāpi dakṣasyāśītikoṭayaḥ  /
vakṣye tāsāṃ tu vistāraṃ loke yaḥ supratiṣṭhitaḥ  // 4.51 //
dvipadaścābhavan kecit kecid bahupadā narāḥ  /
valīmukhāḥ śaṅkukarṇāḥ karṇaprāvaraṇās tathā  // 4.52 //
aśvaṛkṣamukhāḥ kecit kecit siṃhānanāstathā  /
śvasūkaramukhāḥ kecit keciduṣṭramukhās tathā  // 4.53 //
janayāmāsa dharmātmā mlecchānsarvānanekaśaḥ  /
sa sṛṣṭvā manasā dakṣaḥ striyaḥ paścādajījanat  // 4.54 //
dadau sa daśa dharmāya kaśyapāya trayodaśa  /
saptaviṃśatiṃ somāya dadau nakṣatrasaṃjñitāḥ  /
devāsuramanuṣyādi tābhyaḥ sarvamabhūjjagat  // 4.55 //

devānāṃ dānavānāṃ ca gandharvoragarakṣasām  /
utpattiṃ vistareṇaiva sūta brūhi yathātatham  // 5.1 //
sūta uvāca
saṃkalpād darśanātsparśāt pūrveṣāṃ sṛṣṭirucyate  /
dakṣātprācetasād ūrdhvaṃ sṛṣṭir maithunasambhavā  // 5.2 //
prajāḥ sṛjeti vyādiṣṭaḥ pūrvaṃ dakṣaḥ svayambhuvā  /
yathā sasarja caivādau tathaiva śṛṇuta dvijāḥ  // 5.3 //
yadā tu sṛjatastasya devarṣigaṇapannagān  /
na vṛddhim agamallokas tadā maithunayogataḥ  /
dakṣaḥ putrasahasrāṇi pāñcajanyāmajījanat  // 5.4 //
tāṃs tu dṛṣṭvā mahābhāgaḥ sisṛkṣurvividhāḥ prajāḥ  /
nāradaḥ prāha haryaśvān dakṣaputrānsamāgatān  // 5.5 //
bhuvaḥ pramāṇaṃ sarvatra jñātvordhvam adha eva ca  /
tataḥ sṛṣṭiṃ viśeṣeṇa kurudhvamṛṣisattamāḥ  // 5.6 //
te tu tadvacanaṃ śrutvā prayātāḥ sarvato diśam  /
adyāpi na nivartante samudrādiva sindhavaḥ  // 5.7 //
haryaśveṣu pranaṣṭeṣu punardakṣaḥ prajāpatiḥ  /
vairiṇyāmeva putrāṇāṃ sahasramasṛjatprabhuḥ  // 5.8 //
śabalā nāma te viprāḥ sametāḥ sṛṣṭihetavaḥ  /
nārado 'nugatānprāha punastānpūrvavatsa tān  // 5.9 //
bhuvaḥ pramāṇaṃ sarvatra jñātvā bhrātṝn atho punaḥ  /
āgatya cātha sṛṣṭiṃ ca kariṣyatha viśeṣataḥ  // 5.10 //
te 'pi tenaiva mārgeṇa jagmurbhrātṛpathā tadā  /
tataḥ prabhṛti na bhrātuḥ kanīyānmārgam icchati  /
anviṣyanduḥkhamāpnoti tena tatparivarjayet  // 5.11 //
tatasteṣu vinaṣṭeṣu ṣaṣṭiṃ kanyāḥ prajāpatiḥ  /
vairiṇyāṃ janayāmāsa dakṣaḥ prācetasastathā  // 5.12 //
prādātsa daśa dharmāya kaśyapāya trayodaśa  /
saptaviṃśatiṃ somāya catasro 'riṣṭanemaye  // 5.13 //
dve caiva bhṛguputrāya dve kṛśāśvāya dhīmate  /
dve caivāṅgirase tadvat tāsāṃ nāmāni vistarāt  // 5.14 //
śṛṇudhvaṃ devamātṝṇāṃ prajāvistāram āditaḥ  /
marutvatī vasur yāmī lambā bhānur arundhatī  // 5.15 //
saṃkalpā ca muhūrtā ca sādhyā viśvā ca bhāminī  /
dharmapatnyaḥ samākhyātās tāsāṃ putrān nibodhata  // 5.16 //
viśve devāstu viśvāyāḥ sādhyā sādhyānajījanat  /
marutvatyāṃ marutvanto vasos tu vasavastathā  // 5.17 //
bhānostu bhānavastadvan muhūrtāyāṃ muhūrtakāḥ  /
lambāyāṃ ghoṣanāmāno nāgavīthī tu yāmijā  // 5.18 //
pṛthivītalasambhūtam arundhatyām ajāyata  /
saṃkalpāyāstu saṃkalpo vasusṛṣṭiṃ nibodhata  // 5.19 //
jyotiṣmantastu ye devā vyāpakāḥ sarvato diśam  /
vasavaste samākhyātās teṣāṃ sargaṃ nibodhata  // 5.20 //
āpo dhruvaśca somaśca dharaścaivānilo 'nalaḥ  /
pratyūṣaśca prabhāsaśca vasavo 'ṣṭau prakīrtitāḥ  // 5.21 //
āpasya putrāś catvāraḥ śānto vai daṇḍa eva ca  /
śāmbo 'tha maṇivaktraśca yajñarakṣādhikāriṇaḥ  // 5.22 //
dhruvasya kālaḥ putrastu varcāḥ somādajāyata  /
draviṇo havyavāhaśca dharaputrāv ubhau smṛtau  // 5.23 //
kalyāṇinyāṃ tataḥ prāṇo ramaṇaḥ śiśiro 'pi ca  /
manoharā dharātputrān avāpātha hareḥ sutā  // 5.24 //
śivā manojavaṃ putram avijñātagatiṃ tathā  /
avāpa cānalāt putrāv agniprāyaguṇau punaḥ  // 5.25 //
agniputraḥ kumārastu śarastambe vyajāyata  /
tasya śākho viśākhaśca naigameyaś ca pṛṣṭhajāḥ  // 5.26 //
apatyaṃ kṛttikānāṃ tu kārttikeyas tataḥ smṛtaḥ  /
pratyūṣasa ṛṣiḥ putro vibhur nāmnātha devalaḥ  /
viśvakarmā prabhāsasya putraḥ śilpī prajāpatiḥ  // 5.27 //
prāsādabhavanodyāna- pratimābhūṣaṇādiṣu  /
taḍāgārāmakūpeṣu smṛtaḥ so 'maravardhakiḥ  // 5.28 //
ajaikapādahirbudhnyo virūpākṣo 'tha raivataḥ  /
haraśca bahurūpaśca tryambakaśca sureśvaraḥ  // 5.29 //
sāvitraśca jayantaśca pinākī cāparājitaḥ  /
ete rudrāḥ samākhyātā ekādaśa gaṇeśvarāḥ  // 5.30 //
eteṣāṃ mānasānāṃ tu triśūlavaradhāriṇām  /
koṭayaścaturaśītis tatputrāś cākṣayā matāḥ  // 5.31 //
dikṣu sarvāsu ye rakṣāṃ prakurvanti gaṇeśvarāḥ  /
putrapautrasutāścaite surabhīgarbhasambhavāḥ  // 5.32 //


6
kaśyapasya pravakṣyāmi patnībhyaḥ putrapautrakān  /
aditirditirdanuścaiva ariṣṭā surasā tathā  // 6.1 //
surabhir vinatā tadvat tāmrā krodhavaśā irā  /
kadrūrviśvā munis tadvat tāsāṃ putrān nibodhata  // 6.2 //
tuṣitā nāma ye devāś cākṣuṣasyāntare manoḥ  /
vaivasvate 'ntare caite ādityā dvādaśa smṛtāḥ  // 6.3 //
indro dhātā bhagas tvaṣṭā mitro 'tha varuṇo yamaḥ  /
vivasvānsavitā pūṣā aṃśumān viṣṇur eva ca  // 6.4 //
ete sahasrakiraṇā ādityā dvādaśa smṛtāḥ  /
mārīcātkaśyapādāpa putrānaditiruttamān  // 6.5 //
bhṛśāśvasya ṛṣeḥ putrā devapraharaṇāḥ smṛtāḥ  /
ete devagaṇā viprāḥ pratimanvantareṣu ca  // 6.6 //
utpadyante pralīyante kalpe kalpe tathaiva ca  /
ditiḥ putradvayaṃ lebhe kaśyapād iti naḥ śrutam  // 6.7 //
hiraṇyakaśipuṃ caiva hiraṇyākṣaṃ tathaiva ca  /
hiraṇyakaśipostadvaj jātaṃ putracatuṣṭayam  // 6.8 //
prahlādaś cānuhlādaśca saṃhlādo hlāda eva ca  /
prahlādaputra āyuṣmān śibir bāṣkala eva ca  // 6.9 //
virocanaś caturthaśca sa baliṃ putramāptavān  /
baleḥ putraśataṃ tv āsīd bāṇajyeṣṭhaṃ tato dvijāḥ  // 6.10 //
dhṛtarāṣṭras tathā sūryaś candraścandrāṃśutāpanaḥ  /
nikumbhanābho gurvakṣaḥ kukṣibhīmo vibhīṣaṇaḥ  // 6.11 //
evamādyās tu bahavo bāṇajyeṣṭhā guṇādhikāḥ  /
bāṇaḥ sahasrabāhuś ca sarvāstragaṇasaṃyutaḥ  // 6.12 //
tapasā toṣito yasya pure vasati śūlabhṛt  /
mahākālatvam agamat sāmyaṃ yaśca pinākinaḥ  // 6.13 //
hiraṇyākṣasya putro 'bhūd ulūkaḥ śakunistathā  /
bhūtasaṃtāpanaś caiva mahānābhastathaiva ca  // 6.14 //
etebhyaḥ putrapautrāṇāṃ koṭayaḥ saptasaptatiḥ  /
mahābalā mahākāyā nānārūpā mahaujasaḥ  // 6.15 //
danuḥ putraśataṃ lebhe kaśyapād baladarpitam  /
vipracittiḥ pradhāno 'bhūd yeṣāṃ madhye mahābalaḥ  // 6.16 //
dvimūrdhā śakuniścaiva tathā śaṅkuśirodharaḥ  /
ayomukhaḥ śambaraś ca kapiśo nāmatastathā  // 6.17 //
mārīcir meghavāṃścaiva irā garbhaśirās tathā  /
vidrāvaṇaśca ketuśca ketuvīryaḥ śatahradaḥ  // 6.18 //
indrajit saptajiccaiva vajranābhastathaiva ca  /
ekacakro mahābāhur vajrākṣas tārakas tathā  // 6.19 //
asilomā pulomā ca bindurbāṇo mahāsuraḥ  /
svarbhānurvṛṣaparvā ca evamādyā danoḥ sutāḥ  // 6.20 //
svarbhānostu prabhā kanyā śacī caiva pulomajā  /
upadānavī mayasyāsīt tathā mandodarī kuhūḥ  // 6.21 //
śarmiṣṭhā sundarī caiva candrā ca vṛṣaparvaṇaḥ  /
pulomā kālakā caiva vaiśvānarasute hi te  // 6.22 //
bahvapatye mahāsattve mārīcasya parigrahe  /
tayoḥ ṣaṣṭisahasrāṇi dānavānāmabhūtpurā  // 6.23 //
paulomānkālakeyāṃśca mārīco 'janayatpurā  /
avadhyā ye 'marāṇāṃ vai hiraṇyapuravāsinaḥ  // 6.24 //
caturmukhāllabdhavarās te hatā vijayena tu  /
vipracittiḥ saiṃhikeyān siṃhikāyām ajījanat  // 6.25 //
hiraṇyakaśiporye vai bhāgineyās trayodaśa  /
vyaṃsaḥ kalpaśca rājendra nalo vātāpireva ca  // 6.26 //
ilvalo namuciścaiva śvasṛpaś cājanas tathā  /
narakaḥ kālanābhaśca saramāṇas tathaiva ca  // 6.27 //
kālavīyeś ca vikhyāto danuvaṃśavivardhanāḥ  /
saṃhrādasya tu daityasya nivātakavacāḥ smṛtāḥ  // 6.28 //
avadhyāḥ sarvadevānāṃ gandharvoragarakṣasām  /
ye hatā bhargam āśritya tv arjunena raṇājire  // 6.29 //
ṣaṭkanyā janayāmāsa tāmrā mārīcabījataḥ  /
śukī śyenī ca bhāsī ca sugrīvī gṛdhrikā śuciḥ  // 6.30 //
śukī śukānulūkāṃśca janayāmāsa dharmataḥ  /
śyenī śyenāṃstathā bhāsī kurarānapyajījanat  // 6.31 //
gṛdhrī gṛdhrānkapotāṃśca pārāvatavihaṃgamān  /
haṃsasārasakrauñcāṃś ca plavāñchucirajījanat  // 6.32 //
ajāśvameṣoṣṭrakharān sugrīvī cāpyajījanat  /
eṣa tāmrānvayaḥ prokto vinatāyāṃ nibodhata  // 6.33 //
garuḍaḥ patatāṃ nāthaḥ aruṇaśca patatriṇām  /
saudāmanī tathā kanyā yeyaṃ nabhasi viśrutā  // 6.34 ///
sampātiś ca jaṭāyuśca aruṇasya sutāv ubhau  /
sampātiputro babhruśca śīghragaścāpi viśrutaḥ  // 6.35 //
jaṭāyuṣaḥ karṇikāraḥ śatagāmī ca viśrutau  /
sāraso rajjuvālaśca bheruṇḍaścāpi tatsutāḥ  // 6.36 //
teṣāmanantamabhavat pakṣiṇāṃ putrapautrakam  /
surasāyāḥ sahasraṃ tu sarpāṇām abhavatpurā  // 6.37 //
sahasraśirasāṃ kadrūḥ sahasraṃ cāpi suvrata  /
pradhānāsteṣu vikhyātāḥ ṣaḍviṃśatir ariṃdama  // 6.38 //
śeṣavāsukikarkoṭa- śaṅkhairāvatakambalāḥ  /
dhanaṃjayamahānīla- padmāśvataratakṣakāḥ  // 6.39 //
elāpattramahāpadma- dhṛtarāṣṭrabalāhakāḥ  /
śaṅkhapālamahāśaṅkha- puṣpadaṃṣṭraśubhānanāḥ  // 6.40 //
śaṅkuromā ca bahulo vāmanaḥ pāṇinastathā  /
kapilo durmukhaścāpi patañjaliriti smṛtāḥ  // 6.41 //
eṣāmanantamabhavat sarveṣāṃ putrapautrakam  /
prāyaśo yatpurā dagdhaṃ janamejayamandire  // 6.42 //
rakṣogaṇaṃ krodhavaśā svanāmānam ajījanat  /
daṃṣṭriṇāṃ niyutaṃ teṣāṃ bhīmasenādagātkṣayam  // 6.43 //
rudrāṇāṃ ca gaṇaṃ tadvad gomahiṣyo varāṅganāḥ  /
surabhirjanayāmāsa kaśyapātsaṃyatavratā  // 6.44 //
munirmunīnāṃ ca gaṇaṃ gaṇamapsarasāṃ tathā  /
tathā kiṃnaragandharvān ariṣṭājanayad bahūn  // 6.45 //
tṛṇavṛkṣalatāgulmam irā sarvam ajījanat  /
viśvā tu yakṣarakṣāṃsi janayāmāsa koṭiśaḥ  // 6.46 //
tata ekonapañcāśan marutaḥ kaśyapādditiḥ  /
janayāmāsa dharmajñān sarvānamaravallabhān  // 6.47 //

7
ṛṣaya ūcuḥ
diteḥ putrāḥ kathaṃ jātā maruto devavallabhāḥ  /
devairjagmuśca sāpatnaiḥ kasmātte sakhyamuttamam  // 7.1 //
sūta uvāca
purā devāsure yuddhe hṛteṣu hariṇā suraiḥ  /
putrapautreṣu śokārtā gatvā bhūlokamuttamam  // 7.2 //
syamantapañcake kṣetre sarasvatyāstaṭe śubhe  /
bhartur ārādhanaparā tapa ugraṃ cacāra ha  // 7.3 //
tadā ditir daityamātā ṛṣirūpeṇa suvratā  /
phalāhārā tapas tepe kṛcchraṃ cāndrāyaṇādikam  // 7.4 //
yāvadvarṣaśataṃ sāgraṃ jarāśokasamākulā  /
tataḥ sā tapasā taptā vasiṣṭhādīnapṛcchata  // 7.5 //
kathayantu bhavanto me putraśokavināśanam  /
vrataṃ saubhāgyaphaladam ihaloke paratra ca  // 7.6 //
ūcur vasiṣṭhapramukhā madanadvādaśīvratam  /
yasyāḥ prabhāvādabhavat sutaśokavivarjitā  // 7.7 //
ṛṣaya ūcuḥ
śrotumicchāmahe sūta madanadvādaśīvratam  /
sutānekonapañcāśad yena lebhe ditiḥ punaḥ  // 7.8 //
sūta uvāca
yadvasiṣṭhādibhiḥ pūrvaṃ diteḥ kathitamuttamam  /
vistareṇa tadevedaṃ matsakāśānnibodhata  // 7.9 //
caitre māsi site pakṣe dvādaśyāṃ niyatavrataḥ  /
sthāpayedavraṇaṃ kumbhaṃ sitataṇḍulapūritam  // 7.10 //
nānāphalayutaṃ tadvad ikṣudaṇḍasamanvitam  /
sitavastrayugacchannaṃ sitacandanacarcitam  // 7.11 //
nānābhakṣyasamopetaṃ sahiraṇyaṃ tu śaktitaḥ  /
tāmrapātraṃ guḍopetaṃ tasyopari niveśayet  // 7.12 //
tasmādupari kāmaṃ tu kadalīdalasaṃsthitam  /
kuryāccharkarayopetāṃ ratiṃ tasya ca vāmataḥ  // 7.13 //
gandhaṃ dhūpaṃ tato dadyād gītaṃ vādyaṃ ca kārayet  /
tadabhāve kathāṃ kuryāt kāmakeśavayornaraḥ  // 7.14 //
kāmanāmnā harerarcāṃ snāpayedgandhavāriṇā  /
śuklapuṣpākṣatatilair arcayenmadhusūdanam  // 7.15 //
kāmāya pādau sampūjya jaṅghe saubhāgyadāya ca  /
ūrū smarāyeti punar manmathāyeti vai kaṭim  // 7.16 //
svacchodarāyetyudaram anaṅgāyetyuro hareḥ  /
mukhaṃ padmamukhāyeti bāhū pañcaśarāya vai  // 7.17 //
namaḥ sarvātmane maulim arcayediti keśavam  /
tataḥ prabhāte taṃ kumbhaṃ brāhmaṇāya nivedayet  // 7.18 //
brāhmaṇānbhojayedbhaktyā svayaṃ ca lavaṇādṛte  /
bhuktyā tu dakṣiṇāṃ dadyād imaṃ mantramudīrayet  // 7.19 //
prīyatām atra bhagavān kāmarūpī janārdanaḥ  /
hṛdaye sarvabhūtānāṃ ya ānando 'bhidhīyate  // 7.20 //
anena vidhinā sarvaṃ māsi māsi vrataṃ caret  /
upavāsī trayodaśyām arcayedviṣṇumavyayam  // 7.21 //
phalamekaṃ ca saṃprāśya dvādaśyāṃ bhūtale svapet  /
tatastrayodaśe māsi dhṛtadhenusamanvitām  // 7.22 //
śayyāṃ dadyādanaṅgāya sarvopaskarasaṃyutām  /
kāñcanaṃ kāmadevaṃ ca śuklāṃ gāṃ ca payasvinīm  // 7.23 //
vāsobhir dvijadāmpatyaṃ pūjyaṃ śaktyā vibhūṣaṇaiḥ  /
śayyāgandhādikaṃ dadyāt prīyatām ityudīrayet  // 7.24 //
homaḥ śuklatilaiḥ kāryaḥ kāmanāmāni kīrtayet  /
gavyena haviṣā tadvat pāyasena ca dharmavit  // 7.25 //
viprebhyo bhojanaṃ dadyād vittaśāṭhyaṃ vivarjayet  /
ikṣudaṇḍānatho dadyāt puṣpamālāś ca śaktitaḥ  // 7.26 //
yaḥ kuryād vidhinānena madanadvādaśīmimām  /
sa sarvapāpanirmuktaḥ prāpnoti harisāmyatām  // 7.27 //
ihaloke varān putrān saubhāgyaphalam aśnute  /
yaḥ smaraḥ saṃsmṛto viṣṇur ānandātmā maheśvaraḥ  // 7.28 //
sukhārthī kāmarūpeṇa smaredaṅgajamīśvaram  /
etacchrutvā cakārāsau ditiḥ sarvamaśeṣataḥ  // 7.29 //
kaśyapo vratamāhātmyād āgatya parayā mudā  /
cakāra karkaśāṃ bhūyo rūpayauvanaśālinīm  // 7.30 //
varair āchandayāmāsa sā tu vavre tato varam  /
putraṃ śakravadhārthāya samarthamamitaujasam  // 7.31 //
varayāmi mahātmānaṃ sarvāmaraniṣūdanam  /
uvāca kaśyapo vākyam indrahantāram ūrjitam  // 7.32 //
pradāsyāmyahameveha kiṃ tv etat kriyatāṃ śubhe  /
āpastambaḥ karotviṣṭiṃ putrīyāmadya suvrate  // 7.33 //
vidhāsyāmi tato garbham indraśatruniṣūdanam  /
āpastambastataścakre putreṣṭiṃ draviṇādhikām  // 7.34 //
indraśatrur bhavasveti juhāva ca savistaram  /
devā mumudire daityā vimukhāḥ syuśca dānavāḥ  // 7.35 //
dityāṃ garbham athādhatta kaśyapaḥ prāha tāṃ punaḥ  /
tvayā yatno vidhātavyo hy asmingarbhe varānane  // 7.36 //
saṃvatsaraśataṃ tv ekam asminn eva tapovane  /
saṃkhyāyāṃ naiva bhoktavyaṃ garbhiṇyā varavarṇini  // 7.37 //
na sthātavyaṃ na gantavyaṃ vṛkṣamūleṣu sarvadā  /
nopaskareṣūpaviśen musalolūkhalādiṣu  // 7.38 //
jale ca nāvagāheta śūnyāgāraṃ ca varjayet  /
valmīkāyāṃ na tiṣṭheta na codvignamanā bhavet  // 7.39 //
vilikhenna nakhairbhūmiṃ nāṅgāreṇa na bhasmanā  /
na śayāluḥ sadā tiṣṭhed vyāyāmaṃ ca vivarjayet  // 7.40 //
na tuṣāṅgārabhasmāsthi- kapāleṣu samāviśet  /
varjayetkalahaṃ lokair gātrabhaṅgaṃ tathaiva ca  // 7.41 //
na muktakeśā tiṣṭheta nāśuciḥ syātkadācana  /
na śayītottaraśirā na cāparaśirāḥ kvacit  // 7.42 //
na vastrahīnā nodvignā na cārdracaraṇā satī  /
nāmaṅgalyāṃ vadedvācaṃ na ca hāsyādhikā bhavet  // 7.43 //
kuryāttu guruśuśrūṣāṃ nityaṃ māṅgalyatatparā  /
sarvauṣadhībhiḥ koṣṇena vāriṇā snānamācaret  // 7.44 //
kṛtarakṣā subhūṣā ca vāstupūjanatatparā  /
tiṣṭhetprasannavadanā bhartuḥ priyahite ratā  // 7.45 //
dānaśīlā tṛtīyāyāṃ pārvaṇyaṃ naktamācaret  /
itivṛttā bhavennārī viśeṣeṇa tu garbhiṇī  // 7.46 //
yastu tasyā bhavetputraḥ śīlāyurvṛddhisaṃyutaḥ  /
anyathā garbhapatanam avāpnoti na saṃśayaḥ  // 7.47 //
tasmāttvamanayā vṛttyā garbhe 'sminyatnamācara  /
svastyastu te gamiṣyāmi tathetyuktastayā punaḥ  // 7.48 //
paśyatāṃ sarvabhūtānāṃ tatraivāntaradhīyata  /
tataḥ sā kaśyapoktena vidhinā samatiṣṭhata  // 7.49 //
atha bhītastathendro 'pi diteḥ pārśvamupāgamat  /
vihāya devasadanaṃ tacchuśrūṣuravasthitaḥ  // 7.50 //
diteśchidrāntaraprepsur abhavatpākaśāsanaḥ  /
vinīto 'bhavad avyagraḥ praśāntavadano bahiḥ  // 7.51 //
ajānankila tatkāryam ātmanaḥ śubhamācaran  /
tato varṣaśatānte sā nyūne tu divasais tribhiḥ  // 7.52 //
mene kṛtārthamātmānaṃ prītyā vismitamānasā  /
akṛtvā pādayoḥ śaucaṃ prasuptā muktamūrdhajā  // 7.53 //
nidrābharasamākrāntā divāparaśirāḥ kvacit  /
tatastadantaraṃ labdhvā praviṣṭastu śacīpatiḥ  // 7.54 //
vajreṇa saptadhā cakre taṃ garbhaṃ tridaśādhipaḥ  /
tataḥ saptaiva te jātāḥ kumārāḥ sūryavarcasaḥ  // 7.55 //
rudantaḥ sapta te bālā niṣiddhā giridāriṇā  /
bhūyo 'pi rudataścaitān ekaikaṃ saptadhā hariḥ  // 7.56 //
cicheda vṛtrahantā vai punastadudare sthitaḥ  /
evamekonapañcāśad bhūtvā te rurudurbhṛśam  // 7.57 //
indro nivārayāmāsa māṃ rodiṣṭa punaḥ punaḥ  /
tataḥ sa cintayāmāsa kimetaditi vṛtrahā  // 7.58 //
dharmasya kasya māhātmyāt punaḥ saṃjīvitāstvamī  /
viditvā dhyānayogena madanadvādaśīphalam  // 7.59 //
nūnam etat pariṇatam adhunā kṛṣṇapūjanāt  /
vajreṇāpi hatāḥ santo na vināśam avāpnuyuḥ  // 7.60 //
eko 'pyanekatāmāpa yasmādudarago 'pyalam  /
avadhyā nūnamete vai tasmāddevā bhavantviti  // 7.61 //
yasmānmā rudatetyuktā rudanto garbhasaṃsthitāḥ  /
maruto nāma te nāmnā bhavantu makhabhāginaḥ  // 7.62 //
tataḥ prasādya deveśaḥ kṣamasveti ditiṃ punaḥ  /
arthaśāstraṃ samāsthāya mayaitad duṣkṛtaṃ kṛtam  // 7.63 //
kṛtvā marudgaṇaṃ devaiḥ samānamamarādhipaḥ  /
ditiṃ vimānamāropya sasutāmanayaddivam  // 7.64 //
yajñabhāgabhujo jātā marutaste tato dvijāḥ  /
na jagmuraikyamasurair ataste suravallabhāḥ  // 7.65 //


8
ṛṣaya ūcuḥ
ādisargaśca yaḥ sūta kathito vistareṇa tu  /
pratisargaśca ye yeṣām adhipās tānvadasva naḥ  // 8.1 //
sūta uvāca
yadābhiṣiktaḥ sakalādhirājye pṛthurdharitryāmadhipo babhūva  /
tadauṣadhīnāmadhipaṃ cakāra yajñavratānāṃ tapasāṃ ca candram  // 8.2 //
nakṣatratārādvijavṛkṣagulma- latāvitānasya ca rukmagarbhaḥ  /
apāmadhīśaṃ varuṇaṃ dhanānāṃ rājñāṃ prabhuṃ vaiśravaṇaṃ ca tadvat  // 8.3 //
viṣṇuṃ ravīṇāmadhipaṃ vasūnām agniṃ ca lokādhipatiścakāra  /
prajāpatīnāmadhipaṃ ca dakṣaṃ cakāra śakraṃ marutāmadhīśam  // 8.4 //
daityādhipānāmatha dānavānāṃ prahlādamīśaṃ ca yamaṃ pitṝṇām  /
piśācarakṣaḥpaśubhūtayakṣa- vetālarājaṃ tv atha śūlapāṇim  // 8.5 //
prāleyaśailaṃ ca patiṃ girīṇām īśaṃ samudraṃ sasarinnadānām  /
gandharvavidyādharakiṃnarāṇām īśaṃ punaścitrarathaṃ cakāra  // 8.6 //
nāgādhipaṃ vāsukimugravīryaṃ sarpādhipaṃ takṣakamādideśa  /
diśāṃ gajānāmadhipaṃ cakāra gajendramairāvatanāmadheyam  // 8.7 //
suparṇamīśaṃ patatāmathāśva- rājānamuccaiḥśravasaṃ cakāra  /
siṃhaṃ mṛgāṇāṃ vṛṣabhaṃ gavāṃ ca plakṣaṃ punaḥ sarvavanaspatīnām  // 8.8 //
pitāmahaḥ pūrvamathābhyaṣiñcac caitānpunaḥ sarvadiśādhināthān  /
pūrveṇa dikpālam athābhyaṣiñcan nāmnā sudharmāṇam arātiketum  // 8.9 //
tato 'dhipaṃ dakṣiṇataścakāra sarveśvaraṃ śaṅkhapadābhidhānam  /
sa ketumantaṃ ca digīśamīśaś cakāra paścād bhuvanāṇḍagarbhaḥ  // 8.10 //
hiraṇyaromāṇam udagdigīśaṃ prajāpatir devasutaṃ cakāra  /
adyāpi kurvanti diśām adhīśāḥ śatrūn dahantastu bhuvo 'bhirakṣām  // 8.11 //
caturbhir ebhiḥ pṛthunāmadheyo nṛpo 'bhiṣiktaḥ prathamaṃ pṛthivyām  /
gate 'ntare cākṣuṣanāmadheye vaivasvatākhye ca punaḥ pravṛtte  /
prajāpatiḥ so 'sya carācarasya babhūva sūryānvayavaṃśacihnaḥ  // 8.12 //


9
sūta uvāca
evaṃ śrutvā manuḥ prāha punareva janārdanam  /
pūrveṣāṃ caritaṃ brūhi manūnāṃ madhusūdana  // 9.1 //
matsya uvāca
manvantarāṇi rājendra manūnāṃ caritaṃ ca yat  /
pramāṇaṃ caiva kālasya tāṃ sṛṣṭiṃ ca samāsataḥ  // 9.2 //
ekacittaḥ praśāntātmā śṛṇu mārtaṇḍanandana  /
yāmā nāma purā devā āsan svāyambhuvāntare  // 9.3 //
saptaiva ṛṣayaḥ pūrve ye marīcyādayaḥ smṛtāḥ  /
āgnīdhraścāgnibāhuśca sahaḥ savana eva ca  // 9.4 //
jyotiṣmān dyutimān havyo medhā medhātithir vasuḥ  /
svāyambhuvasyāsya manor daśaite vaṃśavardhanāḥ  // 9.5 //
pratisargamime kṛtvā jagmuryatparamaṃpadam  /
etat svāyambhuvaṃ proktaṃ svārociṣamataḥ param  // 9.6 //
svārociṣasya tanayāś catvāro devavarcasaḥ  /
nabhonabhasyaprasṛti- bhānavaḥ kīrtivardhanāḥ  // 9.7 //
datto niścyavanastambaḥ prāṇaḥ kaśyapa eva ca  /
aurvo bṛhaspatiścaiva saptaite ṛṣayaḥ smṛtāḥ  // 9.8 //
devāśca tuṣitā nāma smṛtāḥ svārociṣe 'ntare  /
hastīndraḥ sukṛto mūrtir āpo jyotirayaḥ smayaḥ  // 9.9 //
vasiṣṭhasya sutāḥ sapta ye prajāpatayaḥ smṛtāḥ  /
dvitīyametatkathitaṃ manvantaramataḥ param  // 9.10 //
auttamīyaṃ pravakṣyāmi tathā manvantaraṃ śubham  /
manurnāmauttamiryatra daśa putrānajījanat  // 9.11 //
īṣa ūrjaśca tarjaś ca śuciḥ śukrastathaiva ca  /
madhuś ca mādhavaścaiva nabhasyo 'tha nabhāstathā  // 9.12 //
sahaḥ kanīyāneteṣām udāraḥ kīrtivardhanaḥ  /
bhāvanās tatra devāḥ syur ūrjāḥ saptarṣayaḥ smṛtāḥ  // 9.13 //
kaukuruṇḍiśca dālbhyaśca śaṅgaḥ pravahaṇaḥ śivaḥ  /
sitaśca sasmitaścaiva saptaite yogavardhanāḥ  // 9.14 //
manvantaraṃ caturthaṃ tu tāmasaṃ nāma viśrutam  /
kaviḥ pṛthus tathaivāgnir akapiḥ kapireva ca  // 9.15 //
tathaiva jalpadhīmānau munayaḥ sapta tāmase  /
sādhyā devagaṇā yatra kathitāstāmase 'ntare  // 9.16 //
akalmaṣas tathā dhanvī tapomūlastapodhanaḥ  /
taporatistapasyaśca tapodyutiparaṃtapau  // 9.17 //
tapobhāgī tapoyogī dharmācāraratāḥ sadā  /
tāmasasya sutāḥ sarve daśa vaṃśavivardhanāḥ  // 9.18 //
pañcamasya manos tadvad raivatasyāntaraṃ śṛṇu  /
devabāhuḥ subāhuśca parjanyaḥ somapo muniḥ  // 9.19 //
hiraṇyaromā saptāśvaḥ saptaite ṛṣayaḥ smṛtāḥ  /
devāścābhūtarajasas tathā prakṛtayaḥ śubhāḥ  // 9.20 //
aruṇas tattvadarśī ca vittavānhavyapaḥ kapiḥ  /
yukto nirutsukaḥ sattvo nirmoho 'tha prakāśakaḥ  // 9.21 //
dharmavīryabalopetā daśaite raivatātmajāḥ  /
bhṛguḥ sudhāmā virajāḥ sahiṣṇurnāda eva ca  // 9.22 //
vivasvānatināmā ca ṣaṣṭhe saptarṣayo 'pare  /
cākṣuṣasyāntare devā lekhā nāma pariśrutāḥ  // 9.23 //
ṛbhavo 'tha ṛbhādyāś ca vārimūlā divaukasaḥ  /
cākṣuṣasyāntare proktā devānāṃ pañca yonayaḥ  // 9.24 //
ruruprabhṛtayastadvac cākṣuṣasya sutā daśa  /
proktāḥ svāyambhuve vaṃśe ye mayā pūrvameva tu  // 9.25 //
antaraṃ cākṣuṣaṃ caitan mayā te parikīrtitam  /
saptamaṃ tatpravakṣyāmi yadvaivasvatamucyate  // 9.26 //
atriś caiva vasiṣṭhaśca kaśyapo gautamastathā  /
bharadvājastathā yogī viśvāmitraḥ pratāpavān  // 9.27 //
jamadagniśca saptaite sāmprataṃ ye maharṣayaḥ  /
kṛtvā dharmavyavasthānaṃ prayānti paramaṃ padam  // 9.28 //
sādhyā viśve ca rudrāśca maruto vasavo 'śvinau  /
ādityāśca surāstadvat sapta devagaṇāḥ smṛtāḥ  // 9.29 //
ikṣvākupramukhāścāsya daśa putrāḥ smṛtā bhuvi  /
manvantareṣu sarveṣu sapta sapta maharṣayaḥ  // 9.30 //
kṛtvā dharmavyavasthānaṃ prayānti paramaṃ padam  /
sāvarṇyasya pravakṣyāmi manorbhāvi tathāntaram  // 9.31 //
aśvatthāmā śaradvāṃśca kauśiko gālavastathā  /
śatānandaḥ kāśyapaśca rāmaśca ṛṣayaḥ smṛtāḥ  // 9.32 //
dhṛtir varīyān yavasaḥ suvarṇo vaṣṭireva ca  /
cariṣṇur īḍyaḥ sumatir vasuḥ śukraśca vīryavān  // 9.33 //
bhaviṣyā daśa sāvarṇer manoḥ putrāḥ prakīrtitāḥ  /
raucyādayastathānye 'pi manavaḥ saṃprakīrtitāḥ  // 9.34 //
ruceḥ prajāpateḥ putro raucyo nāma bhaviṣyati  /
manur bhūtisutas tadvad bhautyo nāma bhaviṣyati  // 9.35 //
tatastu merusāvarṇir brahmasūnur manuḥ smṛtaḥ  /
ṛtaśca ṛtadhāmā ca viṣvakseno manus tathā  // 9.36 //
atītānāgatāścaite manavaḥ parikīrtitāḥ  /
ṣaḍūnaṃ yugasāhasram ebhir vyāptaṃ narādhipa  // 9.37 //
sve sve 'ntare sarvamidam utpādya sacarācaram  /
kalpakṣaye vinirvṛtte mucyante brahmaṇā sahā  // 9.38 //
ete yugasahasrānte vinaśyanti punaḥ punaḥ  /
brahmādyā viṣṇusāyujyaṃ yātā yāsyanti vai dvijāḥ  // 9.39 //

10
ṛṣaya ūcuḥ
bahubhir dhāriṇī bhuktā bhūpālaiḥ śrūyate purā  /
pārthivāḥ pṛthivīyogāt pṛthivī kasya yogataḥ  // 10.1 //
kim arthaṃ ca kṛtā saṃjñā bhūmeḥ kiṃ pāribhāṣikī  /
gaur itīyaṃ ca vikhyātā sūta kasmādbravīhi naḥ  // 10.2 //
sūta uvāca
vaṃśe svāyambhuvasyāsīd aṅgo nāma prajāpatiḥ  /
mṛtyostu duhitā tena pariṇītā sudurmukhā  // 10.3 //
sunīthā nāma tasyāstu veno nāma sutaḥ purā  /
adharmanirataścāsīd balavān vasudhādhipaḥ  // 10.4 //
loke 'pyadharmakṛjjātaḥ parabhāryāpahārakaḥ  /
dharmācārasya siddhyarthaṃ jagato 'tha maharṣibhiḥ  // 10.5 //
anunīto 'pi na dadāv anujñāṃ sa yadā tataḥ  /
śāpena mārayitvainam arājakamayārditāḥ  // 10.6 //
mamanthur brāhmaṇāstasya balāddehamakalmaṣāḥ  /
tatkāyānmathyamānāttu nipeturmlecchajātayaḥ  // 10.7 //
śarīre māturaṃśena kṛṣṇāñjanasamaprabhāḥ  /
pituraṃśasya cāṃśena dhārmiko dharmacāriṇaḥ  // 10.8 //
utpanno dakṣiṇāddhastāt sadhanuḥ saśaro gadī  /
divyatejomayavapuḥ saratnakavacāṅgadaḥ  // 10.9 //
pṛthorevābhavadyatnāt tataḥ pṛthurajāyata  /
sa viprairabhiṣikto 'pi tapaḥ kṛtvā sudāruṇam  // 10.10 //
viṣṇorvareṇa sarvasya prabhutvam agamat punaḥ  /
niḥsvādhyāyavaṣaṭkāraṃ nirdharmaṃ vīkṣya bhūtalam  // 10.11 //
dagdhumevodyataḥ kopāc chareṇāmitavikramaḥ  /
tato gorūpamāsthāya bhūḥ palāyitum udyatā  // 10.12 //
pṛṣṭhato 'nugatastasyāḥ pṛthurdīptaśarāsanaḥ  /
tataḥ sthitvaikadeśe tu kiṃ karomīti cābravīt  // 10.13 //
pṛthur apyavadad vākyam īpsitaṃ dehi suvrate  /
sarvasya jagataḥ śīghraṃ sthāvarasya carasya ca  // 10.14 //
tathaiva sābravīd bhūmir dudoha sa narādhipaḥ  /
svake pāṇau pṛthurvatsaṃ kṛtvā svāyambhuvaṃ manum  // 10.15 //
tadannamabhavacchuddhaṃ prajā jīvanti yena vai  /
tatastu ṛṣibhirdugdhā vatsaḥ somastadābhavat  // 10.16 //
dogdhā bṛhaspatirabhūt pātraṃ vedastapo rasaḥ  /
devaiśca vasudhā dugdhā dogdhā mitrastadābhavat  // 10.17 //
indro vatsaḥ samabhavat kṣīramūrjaskaraṃ balam  /
devānāṃ kāñcanaṃ pātraṃ pitṝṇāṃ rājataṃ tathā  // 10.18 //
antakaścābhavaddogdhā yamo vatsaḥ svadhā rasaḥ  /
alābupātraṃ nāgānāṃ takṣako vatsako 'bhavat  // 10.19 //
viṣaṃ kṣīraṃ tato dogdhā dhṛtarāṣṭro 'bhavatpunaḥ  /
asurairapi dugdheyam āyase śakrapīḍinīm  // 10.20 //
pātre māyām abhūd vatsaḥ prāhlādis tu virocanaḥ  /
dogdhā dvimūrdhā tatrāsīn māyā yena pravartitā  // 10.21 //
yakṣaiśca vasudhā dugdhā purāntardhānam īpsubhiḥ  /
kṛtvā vaiśravaṇaṃ vatsam āmapātre mahīpate  // 10.22 //
pretarakṣogaṇairdugdhā dharā rudhiramulbaṇam  /
raupyanābho 'bhavaddogdhā sumālī vatsa eva tu  // 10.23 //
gandharvaiśca purā dugdhā vasudhā sāpsarogaṇaiḥ  /
vatsaṃ caitrarathaṃ kṛtvā gandhānpadmadale tathā  // 10.24 //
dogdhā vararucirnāma nāṭyavedasya pāragaḥ  /
giribhirvasudhā dugdhā ratnāni vividhāni ca  // 10.25 //
auṣadhāni ca divyāni dogdhā merurmahācalaḥ  /
vatso 'bhūddhimavāṃs tatra pātraṃ śailamayaṃ punaḥ  // 10.26 //
vṛkṣaiśca vasudhā dugdhā kṣīraṃ chinnaprarohaṇam  /
pālāśapātre dogdhā tu śālaḥ puṣpalatākulaḥ  // 10.27 //
plakṣo 'bhavattato vatsaḥ sarvavṛkṣo dhanādhipaḥ  /
evamanyaiśca vasudhā tadā dugdhā yathepsitam  // 10.28 //
āyur dhanāni saukhyaṃ ca pṛthau rājyaṃ praśāsati  /
na daridrastadā kaścin na rogī na ca pāpakṛt  // 10.29 //
nopasargabhayaṃ kiṃcit pṛthau rājani śāsati  /
nityaṃ pramuditā lokā duḥkhaśokavivarjitāḥ  // 10.30 //
dhanuṣkoṭyā ca śailendrān utsārya sa mahābalaḥ  /
bhuvastalaṃ samaṃ cakre lokānāṃ hitakāmyayā  // 10.31 //
na puragrāmadurgāṇi na cāyudhadharā narāḥ  /
kṣayātiśayaduḥkhaṃ ca nārthaśāstrasya cādaraḥ  // 10.32 //
dharmaikavāsanā lokāḥ pṛthau rājyaṃ praśāsati  /
kathitāni ca pātrāṇi yatkṣīraṃ ca mayā tava  // 10.33 //
yeṣāṃ yatra rucistattad deyaṃ tebhyo vijānatā  /
yajñaśrāddheṣu sarveṣu mayā tubhyaṃ niveditam  // 10.34 //
duhitṛtvaṃ gatā yasmāt pṛthordharmavato mahī  /
tadānurāgayogācca pṛthivī viśrutā budhaiḥ  // 10.35 //


11
ṛṣaya ūcuḥ
ādityavaṃśamakhilaṃ vada sūta yathākramam  /
somavaṃśaṃ ca tattvajña yathāvadvaktumarhasi  // 11.1 //
sūta uvāca
vivasvānkaśyapāt pūrvam adityāmabhavat sutaḥ  /
tasya patnītrayaṃ tadvat saṃjñā rājñī prabhā tathā  // 11.2 //
raivatasya sutā rājñī revataṃ suṣuve sutam  /
prabhā prabhātaṃ suṣuve tvāṣṭrī saṃjñā tathā manum  // 11.3 //
yamaśca yamunā caiva yamalau tu babhūvatuḥ  /
tatastejomayaṃ rūpam asahantī vivasvataḥ  // 11.4 //
nārīmutpādayāmāsa svaśarīrādaninditām  /
tvāṣṭrī svarūparūpeṇa nāmnā chāyeti bhāminī  // 11.5 //
purataḥ saṃsthitāṃ dṛṣṭvā saṃjñā tāṃ pratyabhāṣata  /
chāye taṃ bhaja bhartāram asmadīyaṃ varānane  // 11.6 //
apatyāni madīyāni mātṛsnehena pālaya  /
tathetyuktvā tu sā devam agamat kvāpi suvratā  // 11.7 //
kāmayāmāsa devo 'pi saṃjñeyamiti cādarāt  /
janayāmāsa tasyāṃ tu putraṃ ca manurūpiṇam  // 11.8 //
savarṇatvācca sāvarṇir manorvaivasvatasya ca  /
tataḥ śaniṃ ca tapatīṃ viṣṭiṃ caiva krameṇa tu  // 11.9 //
chāyāyāṃ janayāmāsa saṃjñeyamiti bhāskaraḥ  /
chāyā svaputre 'bhyadhikaṃ snehaṃ cakre manau tathā  // 11.10 //
pūrvo manustu cakṣāma na yamaḥ krodhamūrchitaḥ  /
saṃtarjayāmāsa tadā pādamudyamya dakṣiṇam  // 11.11 //
śaśāpa ca yamaṃ chāyā sakṣataḥ kṛmisaṃyutaḥ  /
pādo 'yameko bhavitā pūyaśoṇitavisravaḥ  // 11.12 //
nivedayāmāsa pitur yamaḥ śāpādamarṣitaḥ  /
niṣkāraṇamahaṃ śapto mātrā deva sakopayā  // 11.13 //
bālabhāvānmayā kiṃcid udyataś caraṇaḥ sakṛt  /
manunā vāryamāṇāpi mama śāpam adād vibho  // 11.14 //
prāyo na mātā sāsmākaṃ śāpenāhaṃ yato hataḥ  /
devo 'pyāha yamaṃ bhūyaḥ kiṃ karomi mahāmate  // 11.15 //
maurkhyātkasya na duḥkhaṃ syād athavā karmasaṃtatiḥ  /
anivāryā bhavasyāpi kā kathānyeṣu jantuṣu  // 11.16 //
kṛkavākurmayā datto yaḥ kṛmīnbhakṣayiṣyati  /
kledaṃ ca rudhiraṃ caiva vatsāyam apaneṣyati  // 11.17 //
evamuktastapastepe yamastīvraṃ mahāyaśāḥ  /
gokarṇatīrthe vairāgyāt phalapattrānilāśanaḥ  // 11.18 //
ārādhayanmahādevaṃ yāvadvarṣāyutāyutam  /
varaṃ prādānmahādevaḥ saṃtuṣṭaḥ śūlabhṛttadā  // 11.19 //
vavre sa lokapālatvaṃ pitṛloke nṛpālayam  /
dharmādharmātmakasyāpi jagatastu parīkṣaṇam  // 11.20 //
evaṃ sa lokapālatvam agamacchūlapāṇinaḥ  /
pitṝṇāṃ cādhipatyaṃ ca dharmādharmasya cānagha  // 11.21 //
vivasvānatha tajjñātvā saṃjñāyāḥ karmaceṣṭitam  /
tvaṣṭuḥ samīpamagamad ācacakṣe ca roṣavān  // 11.22 //
tamuvāca tatastvaṣṭā sāntvapūrvaṃ dvijottamāḥ  /
tavāsahantī bhagavan mahastīvraṃ tamonudam  // 11.23 //
vaḍabārūpam āsthāya matsakāśam ihāgatā  /
nivāritā mayā sā nu tvayā caiva divākara  // 11.24 //
yasmād avijñātatayā matsakāśam ihāgatā  /
tasmānmadīyaṃ bhavanaṃ praveṣṭuṃ na tvamarhasi  // 11.25 //
evamuktā jagāmātha marudeśamaninditā  /
vaḍabārūpam āsthāya bhūtale sampratiṣṭhitā  // 11.26 //
tasmātprasādaṃ kuru me yadyanugrahabhāgaham  /
apaneṣyāmi te tejo yantre kṛtvā divākara  // 11.27 //
rūpaṃ tava kariṣyāmi lokānandakaraṃ prabho  /
tathetyuktaḥ sa raviṇā bhramau kṛtvā divākaram  // 11.28 //
pṛthakcakāra tattejaś cakraṃ viṣṇor akalpayat  /
triśūlaṃ cāpi rudrasya vajramindrasya cādhikam  // 11.29 //
daityadānavasaṃhartuḥ sahasrakiraṇātmakam  /
rūpaṃ cāpratimaṃ cakre tvaṣṭā padbhyāmṛte mahat  // 11.30 //
na śaśākātha taddraṣṭuṃ pādarūpaṃ raveḥ punaḥ  /
arcāsvapi tataḥ pādau na kaścitkārayet kvacit  // 11.31 //
yaḥ karoti sa pāpiṣṭhāṃ gatimāpnoti ninditām  /
kuṣṭharogamavāpnoti loke 'sminduḥkhasaṃyutaḥ  // 11.32 //
tasmācca dharmakāmārthī citreṣvāyataneṣu ca  /
na kvacitkārayetpādau devadevasya dhīmataḥ  // 11.33 //
tataḥ sa bhagavāngatvā bhūrlokam amarādhipaḥ  /
kāmayāmāsa kāmārto mukha eva divākaraḥ  // 11.34 //
aśvarūpeṇa mahatā tejasā ca samāvṛtaḥ  /
saṃjñā ca manasā kṣobham agamadbhayavihvalā  // 11.35 //
nāsāpuṭābhyāmutsṛṣṭaṃ paro 'yamiti śaṅkayā  /
tadretasas tato jātāv aśvināv iti niścitam  // 11.36 //
dasrau sutatvāt saṃjātau nāsatyau nāsikāgrataḥ  /
jñātvā cirācca taṃ devaṃ saṃtoṣam agamat param  // 11.37 //
vimānenāgamatsvargaṃ patyā saha mudānvitā  /
sāvarṇo 'pi manur merāv adyāpyāste tapodhanaḥ  /
śanistapobalādāpa grahasāmyaṃ tataḥ punaḥ  // 11.38 //
yamunā tapatī caiva punarnadyau babhūvatuḥ  /
viṣṭirghorātmikā tadvat kālatvena vyavasthitā  // 11.39 //
manor vaivasvatasyāsan daśa putrā mahābalāḥ  /
ilas tu prathamasteṣāṃ putreṣṭyāṃ samajāyata  // 11.40 //
ikṣvākuḥ kuśanābhaśca ariṣṭo dhṛṣṭa eva ca  /
nariṣyantaḥ karūṣaśca śaryātiśca mahābalāḥ  /
pṛṣadhraścātha nābhāgaḥ sarve te divyamānuṣāḥ  // 11.41 //
abhiṣicya manuḥ putram ilaṃ jyeṣṭhaṃ sa dhārmikaḥ  /
jagāma tapase bhūyaḥ sa mahendravanālayam  // 11.42 //
atha digjayasiddhyartham ilaḥ prāyānmahīm imām  /
bhramandvīpāni sarvāṇi kṣmābhṛtaḥ sampradharṣayan  // 11.43 //
jagāmopavanaṃ śambhor aśvākṛṣṭaḥ pratāpavān  /
kalpadrumalatākīrṇaṃ nāmnā śaravaṇaṃ mahat  // 11.44 //
ramate yatra deveśaḥ śambhuḥ somārdhaśekharaḥ  /
umayā samayastatra purā śaravaṇe kṛtaḥ  // 11.45 //
puṃnāma sattvaṃ yatkiṃcid āgamiṣyati te vane  /
strītvameṣyati tatsarvaṃ daśayojanamaṇḍale  // 11.46 //
ajñātasamayo rājā ilaḥ śaravaṇe purā  /
strītvamāpa viśann eva vaḍabātvaṃ hayastadā  // 11.47 //
puruṣatvaṃ hṛtaṃ sarvaṃ strīrūpe vismito nṛpaḥ  /
ileti sābhavannārī pīnonnataghanastanī  // 11.48 //
unnataśroṇijaghanā padmapattrāyatekṣaṇā  /
pūrṇenduvadanā tanvī vilāsollāsitekṣaṇā  // 11.49 //
mūlonnatāyatabhujā nīlakuñcitamūrdhajā  /
tanulomā sudaśanā mṛdugambhīrabhāṣiṇī  // 11.50 //
śyāmagaureṇa varṇena haṃsavāraṇagāminī  /
kārmukabhrūyugopetā tanutāmranakhāṅkurā  // 11.51 //
bhramantī ca vane tasmiṃś cintayāmāsa bhāminī  /
ko me pitāthavā bhrātā kā me mātā bhavediha  // 11.52 //
kasya bharturahaṃ dattā kiyadvatsyāmi bhūtale  /
cintayantīti dadṛśe somaputreṇa sāṅganā  // 11.53 //
ilā rūpasamākṣipta- manasā varavarṇinī  /
budhastadāptaye yatnam akarotkāmapīḍitaḥ  // 11.54 //
viśiṣṭākāravāndaṇḍī sakamaṇḍalupustakaḥ  /
veṇudaṇḍakṛtāneka- pavitrakagaṇatrikaḥ  // 11.55 //
dvijarūpaḥ śikhī brahmā nigadankarṇakuṇḍalaḥ  /
baṭubhiścānvito yuktaiḥ samitpuṣpakuśodakaiḥ  // 11.56 //
kilānviṣan vane tasmin nājuhāva sa tāmilām  /
bahir vanasyāntaritaḥ kila pādapamaṇḍale  // 11.57 //
sasaṃbhramamakasmāttāṃ sopālambhamivāvadat  /
tyaktvāgnihotraśuśrūṣāṃ kva gatā mandirānmama  // 11.58 //
iyaṃ vihāravelā te hy atikrāmati sāmpratam  /
ehyehi pṛthusuśroṇi saṃbhrāntā kena hetunā  // 11.59 //
iyaṃ sāyantanī velā vihārasyeha vartate  /
kṛtvopalepanaṃ puṣpair alaṃkuru gṛhaṃ mama  // 11.60 //
sā tv abravīd viramṛtāhaṃ sarvametattapodhana  /
ātmānaṃ tvāṃ ca bhartāraṃ kulaṃ ca vada me 'nagha  // 11.61 //
budhaḥ provāca tāṃ tanvīm ilā tvaṃ varavarṇinī  /
ahaṃ ca kāmuko nāma bahuvidyo budhaḥ smṛtaḥ  // 11.62 //
tejasvinaḥ kule jātaḥ pitā me brāhmaṇādhipaḥ  /
iti sā tasya vacanāt praviṣṭā budhamandiram  // 11.63 //
ratnastambhasamāyuktaṃ divyamāyāvinirmitam  /
ilā kṛtārthamātmānaṃ mene tadbhavanasthitā  // 11.64 //
aho vṛttamaho rūpam aho dhanamaho kulam  /
mama cāsya ca me bhartur aho lāvaṇyam uttamam  // 11.65 //
reme ca sā tena samam atikālamilā tataḥ  /
sarvabhogamaye gehe yathendrabhavane tathā  // 11.66 //

sūta uvāca
athānviṣanto rājānaṃ bhrātaras tasya mānavāḥ  /
ikṣvākupramukhā jagmus tadā śaravaṇāntikam  // 12.1 //
tataste dadṛśuḥ sarve vaḍabām agrataḥ sthitām  /
ratnaparyāṇakiraṇa- dīptakāyām anuttamām  // 12.2 //
paryāṇapratyabhijñānāt sarve vismayam āgatāḥ  /
ayaṃ candraprabho nāma vājī tasya mahātmanaḥ  // 12.3 //
agamad vaḍabārūpam uttamaṃ kena hetunā  /
tatas tu maitrāvaruṇiṃ papracchuste purodhasam  // 12.4 //
kimityetadabhūccitraṃ vada yogavidāṃ vara  /
vasiṣṭhaścābravīt sarvaṃ dṛṣṭvā taddhyānacakṣuṣā  // 12.5 //
samayaḥ śambhudayitā- kṛtaḥ śaravaṇe purā  /
yaḥ pumānpraviśed atra sa nārītvamavāpsyati  // 12.6 //
ayamaśvo 'pi nārītvam agādrājñā sahaiva tu  /
punaḥ puruṣatāmeti yathāsau dhanadopamaḥ  // 12.7 //
tathaiva yatnaḥ kartavyaś cārādhyaiva pinākinam  /
tataste mānavā jagmur yatra devo maheśvaraḥ  // 12.8 //
tuṣṭuvur vividhaiḥ stotraiḥ pārvatīparameśvarau  /
tāv ūcatur alaṅghyo 'yaṃ samayaḥ kiṃtu sāmpratam  // 12.9 //
ikṣvākoraśvamedhena yatphalaṃ syāttadāvayoḥ  /
dattvā kimpuruṣo vīraḥ sa bhaviṣyatyasaṃśayam  // 12.10 //
tathetyuktāstataste tu jagmur vaivasvatātmajāḥ  /
ikṣvākoścāśvamedhena celaḥ kimpuruṣo 'bhavat  // 12.11 //
māsamekaṃ pumānvīraḥ strī ca māsamabhūtpunaḥ  /
budhasya bhavane tiṣṭhan nilo garbhadharo 'bhavat  // 12.12 //
ajījanatputramekam anekaguṇasaṃyutam  /
budhaścotpādya taṃ putraṃ svarlokam agamattataḥ  // 12.13 //
ilasya nāmnā tadvarṣam ilāvṛtam abhūttadā  /
somārkavaṃśayor ādāv ilo 'bhūnmanunandanaḥ  // 12.14 //
evaṃ purūravāḥ puṃsor abhavad vaṃśavardhanaḥ  /
ikṣvākur arkavaṃśasya tathaivoktastapodhanāḥ  // 12.15 //
ilaḥ kimpuruṣatve ca sudyumna iti cocyate  /
punaḥ putratrayamabhūt sudyumnasyāparājitam  // 12.16 //
utkalo vai gahas tadvad dharitāśvaś ca vīryavān  /
utkalasyotkalā nāma gayasya tu gayā matā  // 12.17 //
haritāśvasya dikpūrvā viśrutā kurubhiḥ saha  /
pratiṣṭhāne 'bhiṣicyātha sa purūravasaṃ sutam  // 12.18 //
jagāmelāvṛtaṃ bhoktuṃ varṣaṃ divyaphalāśanam  /
ikṣvākurjyeṣṭhadāyādo madhyadeśamavāptavān  // 12.19 //
nariṣyantasya putro 'bhūc chuco nāma mahābalaḥ  /
nābhāgasyāmbarīṣastu dhṛṣṭasya ca sutatrayam  // 12.20 //
dhṛtaketuścitranātho raṇadhṛṣṭaśca vīryavān  /
ānarto nāma śaryāteḥ sukanyā caiva dārikā  // 12.21 //
ānartasyābhavatputro rocamānaḥ pratāpavān  /
ānarto nābha deśo 'bhūn nagarī ca kuśasthalī  // 12.22 //
rocamānasya putro 'bhūd revo raivata eva ca  /
kakudmī cāparaṃ nāma jyeṣṭhaḥ putraśatasya ca  // 12.23 //
revatī tasya sā kanyā bhāryā rāmasya viśrutā  /
karūṣasya tu kārūṣā bahavaḥ prathitā bhuvi  // 12.24 //
pṛṣadhro govadhācchūdro guruśāpādajāyata  /
ikṣvākuvaṃśaṃ vakṣyāmi śṛṇudhvamṛṣisattamāḥ  // 12.25 //
ikṣvākoḥ putratām āpa vikukṣir nāma devarāṭ  /
jyeṣṭhaḥ putraśatasyāsīd daśa pañca ca tatsutāḥ  // 12.26 //
meroruttarataste tu jātāḥ pārthivasattamāḥ  /
caturdaśottaraṃ cānyac chatamasya tathābhavat  // 12.27 //
merordakṣiṇato ye ye rājānaḥ saṃprakīrtitāḥ  /
jyeṣṭhaḥ kakutstho nāmnābhūt tatsutastu suyodhanaḥ  // 12.28 //
tasya putraḥ pṛthurnāma viśvagaścapṛthoḥ sutaḥ  /
industasya ca putro 'bhūd yuvanāśvastato 'bhavat  // 12.29 //
śrāvastaśca mahātejā vatsakastatsuto 'bhavat  /
nirmitā yena śrāvastī gauḍadeśe dvijottamāḥ  // 12.30 //
śrāvastādbṛhadaśvo 'bhūt kuvalāśvas tato 'bhavat  /
dhundhumāratvamagamad dhundhunāmnā hataḥ purā  // 12.31 //
tasya putrās trayo jātā dṛḍhāśvo daṇḍa eva ca  /
kapilāśvaśca vikhyāto dhaundhumāriḥ pratāpavān  // 12.32 //
dṛḍhāśvasya pramodaśca haryaśvastasya cātmajaḥ  /
haryaśvasya nikumbho 'bhūt saṃhatāśvas tato 'bhavat  // 12.33 //
akṛtāśvo raṇāśvaśca saṃhatāśvasutāv ubhau  /
yuvanāśvo raṇāśvasya māndhātā ca tato 'bhavat  // 12.34 //
māndhātuḥ purukutso 'bhūd dharmasenaśca pārthivaḥ  /
mucukundaśca vikhyātaḥ śatrujicca pratāpavān  // 12.35 //
purukutsasya putro 'bhūd vasudo narmadāpatiḥ  /
sambhūtistasya putro 'bhūt tridhanvā ca tato 'bhavat  // 12.36 //
tridhanvanaḥ suto jātas trayyāruṇa iti smṛtaḥ  /
tasmātsatyavrato nāma tasmātsatyarathaḥ smṛtaḥ  // 12.37 //
tasya putro hariścandro hariścandrācca rohitaḥ  /
rohitācca vṛko jāto vṛkādbāhurajāyata  // 12.38 //
sagarastasya putro 'bhūd rājā paramadhārmikaḥ  /
dve bhārye sagarasyāpi prabhā bhānumatī tathā  // 12.39 //
tābhyāmārādhitaḥ pūrvam aurvo 'gniḥ putrakāmyayā  /
aurvastuṣṭastayoḥ prādād yatheṣṭaṃ varamuttamam  // 12.40 //
ekā ṣaṣṭisahasrāṇi sutamekaṃ tathāparā  /
gṛhṇātu vaṃśakartāraṃ prabhāgṛhṇād bahūṃstadā  // 12.41 //
ekaṃ bhānumatī putram agṛhṇādasamañjasam  /
tataḥ ṣaṣṭisahasrāṇi suṣuve yādavī prabhā  // 12.42 //
khanantaḥ pṛthivīṃ dagdhā viṣṇunā ye 'śvamārgaṇe  /
asamañjasastu tanayo yo 'ṃśumānnāma viśrutaḥ  // 12.43 //
tasya putro dilīpas tu dilīpāttu bhagīrathaḥ  /
yena bhāgīrathī gaṅgā tapaḥ kṛtvāvatāritā  // 12.44 //
bhagīrathasya tanayo nābhāga iti viśrutaḥ  /
nābhāgasyāmbarīṣo 'bhūt sindhudvīpas tato 'bhavat  // 12.45 //
tasyāyutāyuḥ putro 'bhūd ṛtuparṇastato 'bhavat  /
tasya kalmāṣapādastu sarvakarmā tataḥ smṛtaḥ  // 12.46 //
tasyānaraṇyaḥ putro 'bhūn nighnastasya suto 'bhavat  /
nighnaputrāv ubhau jātāv anamitraraghū nṛpau  // 12.47 //
anāmitro vanamagād bhavitā sa kṛte nṛpaḥ  /
raghor abhūddilīpastu dilīpādajakastathā  // 12.48 //
dīrghabāhurajājjātaś cājapālastato nṛpaḥ  /
tasmāddaśaratho jātas tasya putracatuṣṭayam  // 12.49 //
nārāyaṇātmakāḥ sarve rāmasteṣvagrajo 'bhavat  /
rāvaṇāntakarastadvad raghūṇāṃ vaṃśavardhanaḥ  // 12.50 //
vālmīkistasya caritaṃ cakre bhārgavasattamaḥ  /
tasya putrau kuśalavāv ikṣvākukulavardhanau  // 12.51 //
atithistu kuśājjajñe niṣadhastasya cātmajaḥ  /
nalastu naiṣadhastasmān nabhāstasmādajāyata  // 12.52 //
nabhasaḥ puṇḍarīko 'bhūt kṣemadhanvā tataḥ smṛtaḥ  /
tasya putro 'bhavadvīro devānīkaḥ pratāpavān  // 12.53 //
ahīnagustasya sutaḥ sahasrāśvastataḥ paraḥ  /
tataścandrāvalokastu tārāpīḍastato 'bhavat  // 12.54 //
tasyātmajaś candragirir bhānuś candrastato 'bhavat  /
śrutāyur abhavattasmād bhārate yo nipātitaḥ  // 12.55 //
nalau dvāv eva vikhyātau vaṃśe kaśyapasambhave  /
vīrasenasutastadvan naiṣadhaśca narādhipaḥ  // 12.56 //
ete vaivasvate vaṃśe rājāno bhūridakṣiṇāḥ  /
ikṣvākuvaṃśaprabhavāḥ prādhānyena prakīrtitāḥ  // 12.57 //


13
manuruvāca
bhagavañśrotumicchāmi pitṝṇāṃ vaṃśamuttamam  /
raveśca śrāddhadevatvaṃ somasya ca viśeṣataḥ  // 13.1 //
matsya uvāca
hanta te kathayiṣyāmi pitṝṇāṃ vaṃśamuttamam  /
svarge pitṛgaṇāḥ sapta trayasteṣāmamūrtayaḥ  // 13.2 //
mūrtimanto 'tha catvāraḥ sarveṣāmamitaujasaḥ  /
amūrtayaḥ pitṛgaṇā vairājasya prajāpateḥ  // 13.3 //
jayanti yāndevagaṇā vairājā iti viśrutāḥ  /
ye caite yogavibhraṣṭāḥ prāpya lokānsanātanān  // 13.4 //
punarbrahmadinānte tu jāyante brahmavādinaḥ  /
samprāpya tāṃ smṛtiṃ bhūyo yogaṃ sāṃkhyamanuttamam  // 13.5 //
siddhiṃ prayānti yogena punarāvṛttidurlabhām  /
yogināmeva deyāni tasmācchrāddhāni dātṛbhiḥ  // 13.6 //
eteṣāṃ mānasī kanyā patnī himavato matā  /
mainākastasya dāyādaḥ krauñcastasyāgrajo 'bhavat  /
krauñcadvīpaḥ smṛto yena caturtho ghṛtasaṃvṛtaḥ  // 13.7 //
menā ca suṣuve tisraḥ kanyā yogavatīstataḥ  /
umaikaparṇāparṇā ca tīvravrataparāyaṇāḥ  // 13.8 //
rudrasyaikā sitasyaikā jaigīṣavyasya cāparā  /
dattā himavatā bālāḥ sarvā loke tapo 'dhikāḥ  // 13.9 //
ṛṣaya ūcuḥ
kasmāddākṣāyaṇī pūrvaṃ dadāhātmānamātmanā  /
himavadduhitā tadvat kathaṃ jātā mahītale  // 13.10 //
saṃharantī kimuktāsau sutā vā brahmasūnunā  /
dakṣeṇa lokajananī sūta vistarato vada  // 13.11 //
sūta uvāca
dakṣasya yajñe vitate prabhūtavaradakṣiṇe  /
samāhūteṣu deveṣu provāca pitaraṃ satī  // 13.12 //
kimarthaṃ tāta bhartā me yajñe 'sminnābhimantritaḥ  /
ayogya iti tāmāha dakṣo yajñeṣu śūlabhṛt  // 13.13 //
upasaṃhārakṛdrudras tenāmaṅgalabhāgayam  /
cukopātha satī dehaṃ tyakṣyāmīti tvadudbhavam  // 13.14 //
daśānāṃ tvaṃ ca bhavitā pitṝṇāmekaputrakaḥ  /
kṣatriyatve 'śvamedhe ca rudrāttvaṃ nāśameṣyasi  // 13.15 //
ityuktvā yogamāsthāya svadehodbhavatejasā  /
nirdahantī tadātmānaṃ sadevāsurakiṃnaraiḥ  // 13.16 //
kiṃ kimetaditi proktā gandharvagaṇaguhyakaiḥ  /
upagamyābravīddakṣaḥ praṇipatyātha duḥkhitaḥ  // 13.17 //
tvamasya jagato mātā jagatsaubhāgyadevatā  /
duhitṛtvaṃ gatā devi mamānugrahakāmyayā  // 13.18 //
na tvayā rahitaṃ kiṃcid brahmāṇḍe sacarācaram  /
prasādaṃ kuru dharmajñe na māṃ tyaktumihārhasi  // 13.19 //
prāha devī yadārabdhaṃ tatkāryaṃ me na saṃśayaḥ  /
kiṃtvavaśyaṃ tvayā martye hatayajñena śūlinā  // 13.20 //
prasāde lokasṛṣṭyarthaṃ tapaḥ kāryaṃ mamāntike  /
prajāpatistvaṃ bhavitā daśānāmaṅgajo 'pyalam  // 13.21 //
madaṃśenāṅganā ṣaṣṭir bhaviṣyantyaṅgajāstava  /
matsaṃnidhau tapaḥ kurvan prāpsyase yogamuttamam  // 13.22 //
evamukto 'bravīddakṣaḥ keṣu keṣu mayānaghe  /
tīrtheṣu ca tvaṃ draṣṭavyā stotavyā kaiśca nāmabhiḥ  // 13.23 //
devyuvāca
sarvadā sarvabhūteṣu draṣṭavyā sarvato bhuvi  /
sarvalokeṣu yatkiṃcid rahitaṃ na mayā vinā  // 13.24 //
tathāpi yeṣu sthāneṣu draṣṭavyā siddhimīpsubhiḥ  /
smartavyā bhūtikāmair vā tāni vakṣyāmi tattvataḥ  // 13.25 //
vārāṇasyāṃ viśālākṣī naimiṣe liṅgadhāriṇī  /
prayāge lalitā devī kāmākṣī gandhamādane  /
mānase kumudā nāma viśvakāyā tathāmbare  // 13.26 //
gomante gomatī nāma mandare kāmacāriṇī  /
madotkaṭā caitrarathe jayantī hastināpure  // 13.27 //
kānyakubje tathā gaurī rambhā malayaparvate  /
ekāmbhake kīrtimatī viśvāṃ viśveśvare viduḥ  // 13.28 //
puṣkare puruhūteti kedāre mārgadāyinī  /
nandā himavataḥ pṛṣṭhe gokarṇe bhadrakarṇikā  // 13.29 //
sthāneśvare bhavānī tu bilvale bilvapattrikā  /
śrīśaile mādhavī nāma bhadrā bhadreśvare tathā  // 13.30 //
jayā varāhaśaile tu kāmalā kamalālaye  /
rudrakoṭyāṃ ca rudrāṇī kālī kālañjare girau  // 13.31 //
mahāliṅge tu kapilā markoṭe mukuṭeśvarī  /
śālagrāme mahādevī śivaliṅge jalapriyā  // 13.32 //
māyāpuryāṃ kumārī tu saṃtāne lalitā tathā  /
utpalākṣī sahasrākṣe kamalākṣe mahotpalā  // 13.33 //
gaṅgāyāṃ maṅgalā nāma vimilā puruṣottame  /
vipāśāyām amoghākṣī pāṭalā puṇḍravardhane  // 13.34 //
nārāyaṇī supārśve tu vikūṭe bhadrasundarī  /
vipule vipulā nāma kalyāṇī malayācale  // 13.35 //
koṭavī koṭitīrthe tu sugandhā mādhave vane  /
godāśrame trisaṃdhyā tu gaṅgādvāre ratipriyā  // 13.36 //
śivakuṇḍe śivānandā nandinī devikātaṭe  /
rukmiṇī dvāravatyāṃ tu rādhā vṛndāvane vane  // 13.37 //
devakī mathurāyāṃ tu pātāle parameśvarī  /
citrakūṭe tathā sītā vindhye vindhyādhivāsinī  // 13.38 //
sahyādrāv ekavīrā tu hariścandre tu candrikā  /
ramaṇā rāmatīrthe tu yamunāyāṃ mṛgāvatī  // 13.39 //
karavīre mahālakṣmīr umā devī vināyake  /
arogā vaidyanāthe tu mahākāle maheśvarī  // 13.40 //
abhayetyuṣṇatīrtheṣu cāmṛtā vindhyakandare  /
māṇḍavye māṇḍavī nāma svāhā māheśvare pure  // 13.41 //
chāgalāṇḍe pracaṇḍā tu caṇḍikā makarandake  /
someśvare varārohā prabhāse puṣkarāvatī  // 13.42 //
devamātā sarasvatyāṃ pārāvārataṭe matā  /
mahālaye mahābhāgā payoṣṇyāṃ piṅgaleśvarī  // 13.43 //
siṃhikā kṛtaśauce tu kārttikeye yaśaskarī  /
utpalāvartake lolā subhadrā śoṇasaṃgame  // 13.44 //
mātā siddhapure lakṣmīr aṅganā bharatāśrame  /
jālaṃdhare viśvamukhī tārā kiṣkindhaparvate  // 13.45 //
devadāruvane puṣṭir medhā kāśmīramaṇḍale  /
bhīmā devī himādrau tu puṣṭirviśveśvare tathā  // 13.46 //
kapālamocane śuddhir mātā kāyāvarohaṇe  /
śaṅkhoddhāre dhvanirnāma dhṛtiḥ piṇḍārake tathā  // 13.47 //
kālā tu candrabhāgāyām acchode śivakāriṇī  /
veṇāyāmamṛtā nāma badaryāmurvaśī tathā  // 13.48 //
oṣadhī cottarakurau kuśadvīpe kuśodakā  /
manmathā hemakūṭe tu mukuṭe satyavādinī  // 13.49 //
aśvatthe vandanīyā tu nidhirvaiśravaṇālaye  /
gāyatrī vedavadane pārvatī śivasaṃnidhau  // 13.50 //
devaloke tathendrāṇī brahmāsyeṣu sarasvatī  /
sūryabimbe prabhā nāma mātṝṇāṃ vaiṣṇavī matā  // 13.51 //
arundhatī satīnāṃ tu rāmāsu ca tilottamā  /
citte brahmakalā nāma śaktiḥ sarvaśarīriṇām  // 13.52 //
etaduddeśataḥ proktaṃ nāmāṣṭaśatamuttamam  /
aṣṭottaraṃ ca tīrthānāṃ śatametadudāhṛtam  // 13.53 //
yaḥ smarecchṛṇuyād vāpi sarvapāpaiḥ pramucyate  /
eṣu tīrtheṣu yaḥ kṛtvā snānaṃ paśyati māṃ naraḥ  // 13.54 //
sarvapāpavinirmuktaḥ kalpaṃ śivapure vaset  /
yastu matparamaṃ kālaṃ karotyeteṣu mānavaḥ  // 13.55 //
sa bhittvā brahmasadanaṃ padamabhyeti śāṃkaram  /
nāmnāmaṣṭaśataṃ yastu śrāvayecchivasaṃnidhau  // 13.56 //
tṛtīyāyāmathāṣṭamyāṃ bahuputro bhavennaraḥ  /
godāne śrāddhadāne vā ahany ahani vā budhaḥ  // 13.57 //
devārcanavidhau vidvān paṭhan brahmādhigacchati  /
evaṃ vadantī sā tatra dadāhātmānam ātmanā  // 13.58 //
svāyambhuvo 'pi kālena dakṣaḥ prācetaso 'bhavat  /
pārvatī sābhavaddevī śivadehārdhadhāriṇī  // 13.59 //
menāgarbhasamutpannā bhuktamuktiphalapradā  /
arundhatī japantyetat prāpa yogamanuttamam  // 13.60 //
purūravāśca rājarṣi- loke vyajeyatām agāt  /
yayātiḥ putralābhaṃ ca dhanalābhaṃ ca bhārgavaḥ  // 13.61 //
tathānye devadaityāśca brāhmaṇāḥ kṣatriyāstathā  /
vaiśyāḥ śūdrāśca bahavaḥ siddhimīyuryathepsitām  // 13.62 //
yatraitallikhitaṃ tiṣṭhet pūjyate devasaṃnidhau  /
na tatra śoko daurgatyaṃ kadācidapi jāyate  // 13.63 //


14
sūta uvāca
lokāḥ somapathā nāma yatra mārīcanandanāḥ  /
vartante deva pitaro devā yānbhāvayantyalam  // 14.1 //
agniṣvāttā iti khyātā yajvāno yatra saṃsthitāḥ  /
acchodā nāma teṣāṃ tu mānasī kanyakā nadī  // 14.2 //
acchodaṃ nāma ca saraḥ pitṛbhirnirmitaṃ purā  /
acchodā tu tapaścakre divyaṃ varṣasahasrakam  // 14.3 //
ājagmuḥ pitarastuṣṭāḥ kila dātuṃ ca tāṃ varam  /
divyarūpadharāḥ sarve divyamālyānulepanāḥ  // 14.4 //
sarve yuvāno balinaḥ kusumāyudhasaṃnibhāḥ  /
tanmadhye 'māvasuṃ nāma pitaraṃ vīkṣya sāṅganā  // 14.5 //
vavre varārthinī saṅgaṃ kusumāyudhapīḍitā  /
yogādbhraṣṭā tu sā tena vyabhicāreṇa bhāminī  // 14.6 //
dharāṃ tu nāspṛśatpūrvaṃ papātātha bhuvastale  /
tithāv amāvasur yasyām icchāṃ cakre na tāṃ prati  // 14.7 //
dhairyeṇa tasya sā lokair amāvāsyeti viśrutā  /
pitṝṇāṃ vallabhā tasmāt tasyāmakṣayakārakam  // 14.8 //
acchodādhomukhī dīnā lajjitā tapasaḥ kṣayāt  /
sā pitṝn prārthayāmāsa pure cātmaprasiddhaye  // 14.9 //
vilapyamānā pitṛbhir idamuktā tapasvinī  /
bhaviṣyamarthamālokya devakāryaṃ ca te tadā  // 14.10 //
idam ūcur mahābhāgāḥ prasādaśubhayā girā  /
divi divyaśarīreṇa yat kiṃcit kriyate budhaiḥ  // 14.11 //
tenaiva tatkarmaphalaṃ bhujyate varavarṇinī  /
sadyaḥ phalanti karmāṇi devatve pretya mānuṣe  // 14.12 //
tasmāttvaṃ putri tapasaḥ prāpsyase pretya tatphalam  /
aṣṭāviṃśe bhavitrī tvaṃ dvāpare matsyayonijā  // 14.13 //
vyatikramātpitṝṇāṃ tvaṃ kaṣṭaṃ kulamavāpsyasi  /
tasmādrājño vasoḥ kanyā tvam avaśyaṃ bhaviṣyasi  // 14.14 //
kanyā bhūtvā ca lokānsvān punarāpsyasi durlabhān  /
parāśarasya vīryeṇa putramekamavāpsyasi  // 14.15 //
dvīpe tu badarīprāye bādarāyaṇamacyutam  /
sa vedamekaṃ bahudhā vibhajiṣyati te sutaḥ  // 14.16 //
pauravasyātmajau dvau tu samudrāṃśasya śaṃtanoḥ  /
vicitravīryastanayas tathā citrāṅgado nṛpaḥ  // 14.17 //
imāv utpādya tanayau kṣetrajāvasya dhīmataḥ  /
prauṣṭhapadyaṣṭakārūpā pitṛloke bhaviṣyasi  // 14.18 //
nāmnā satyavatī loke pitṛloke tathāṣṭakā  /
āyurārogyadā nityaṃ sarvakāmaphalapradā  // 14.19 //
bhaviṣyasi pare kāle nadītvaṃ ca gamiṣyasi  /
puṇyatoṣā saricchreṣṭhā loke hy acchodanāmikā  // 14.20 //
ityuktvā sa gaṇasteṣāṃ tatraivāntaradhīyata  /
sāpyavāpa ca tatsarvaṃ phalaṃ yaduditaṃ purā  // 14.21 //

15
sūta uvāca
vibhrājā nāma cānye tu divi santi suvarcasaḥ  /
lokā barhiṣado yatra pitaraḥ santi suvratāḥ  // 15.1 //
yatra barhiṇayuktāni vimānāni sahasraśaḥ  /
saṃkalpyā barhiṣo yatra tiṣṭhanti phaladāyinaḥ  // 15.2 //
yatrābhyudayaśālāsu modante śrāddhadāyinaḥ  /
yāṃś ca devāsuragaṇā gandharvāpsarasāṃ gaṇāḥ  // 15.3 //
yakṣarakṣogaṇāścaiva yajanti divi devatāḥ  /
pulastyaputrāḥ śataśas tapoyogasamanvitāḥ  // 15.4 //
mahātmāno mahābhāgā bhaktānāmabhayapradāḥ  /
eteṣāṃ pīvarī kanyā mānasī divi viśrutā  // 15.5 //
yoginī yogamātā ca tapaścakre sudāruṇam  /
prasanno bhagavāṃstasyā varaṃ vavre tu sā hareḥ  // 15.6 //
yogavantaṃ surūpaṃ ca bhartāraṃ vijitendriyam  /
dehi deva prasannastvaṃ patiṃ me vadatāṃ varam  // 15.7 //
uvāca devo bhavitā vyāsaputro yadā śukaḥ  /
bhavitā tasya bhāryā tvaṃ yogācāryasya suvrate  // 15.8 //
bhaviṣyati ca te kanyā kṛtvī nāma ca yoginī  /
pāñcālādhipaterdeyā mānuṣasya tvayā tadā  // 15.9 //
jananī brahmadattasya yogasiddhā ca gauḥ smṛtā  /
kṛṣṇo gauraḥ prabhuḥ śambhur bhaviṣyanti ca te sutāḥ  // 15.10 //
mahātmāno mahābhāgā gamiṣyanti paraṃ padam  /
tānutpādya punar yogāt savarā mokṣameṣyasi  // 15.11 //
sumūrtimantaḥ pitaro vasiṣṭhasya sutāḥ smṛtāḥ  /
nāmnā tu mānasāḥ sarve sarve te dharmamūrtayaḥ  // 15.12 //
jyotirbhāsiṣu lokeṣu ye vasanti divaḥ param  /
virājamānāḥ krīḍanti yatra te śrāddhadāyinaḥ  // 15.13 //
sarvakāmasamṛddheṣu vimāneṣvapi pādajāḥ  /
kiṃ punaḥ śrāddhadā viprā bhaktimantaḥ kriyānvitāḥ  // 15.14 //
gaur nāma kanyā yeṣāṃ tu mānasī divi rājate  /
śukrasya dayitā patnī sādhyānāṃ kīrtivardhinī  // 15.15 //
marīcigarbhā nāmnā tu lokā mārtaṇḍamaṇḍale  /
pitaro yatra tiṣṭhanti haviṣmanto 'ṅgiraḥsutāḥ  // 15.16 //
tīrthaśrāddhapradā yānti ye ca kṣatriyasattamāḥ  /
rājñāṃ tu pitaraste vai svargamokṣaphalapradāḥ  // 15.17 //
eteṣāṃ mānasī kanyā yaśodā lokaviśrutā  /
patnī hy aṃśumataḥ śreṣṭhā snuṣā pañcajanasya ca  // 15.18 //
jananyatha dilīpasya bhagīrathapitāmahī  /
lokāḥ kāmadughā nāma kāmabhogaphalapradāḥ  // 15.19 //
susvadhā nāma pitaro yatra tiṣṭhanti suvratāḥ  /
ājyapā nāma lokeṣu kardamasya prajāpateḥ  // 15.20 //
pulahāṅgajadāyādā vaiśyāstānbhāvayanti ca  /
yatra śrāddhakṛtaḥ sarve paśyanti yugapadgatāḥ  // 15.21 //
mātṛbhrātṛpitṛsvasṛ- sakhisambandhibāndhavān  /
api janmāyutairdṛṣṭān anubhūtānsahasraśaḥ  // 15.22 //
eteṣāṃ mānasī kanyā virajā nāma viśrutā  /
yā patnī nahuṣasyāsīd yayāterjananī tathā  // 15.23 //
ekāṣṭakābhavat paścād brahmaloke gatā satī  /
traya ete gaṇāḥ proktāś caturthaṃ tu vadāmyataḥ  // 15.24 //
lokāstu mānasā nāma brahmāṇḍopari saṃsthitāḥ  /
yeṣāṃ tu mānasī kanyā narmadā nāma viśrutā  // 15.25 //
somapā nāma pitaro yatra tiṣṭhanti śāśvatāḥ  /
dharmamūrtidharāḥ sarve parato brahmaṇaḥ smṛtāḥ  // 15.26 //
utpannāḥ svadhayā te tu brahmatvaṃ prāpya yoginaḥ  /
kṛtvā sṛṣṭyādikaṃ sarvaṃ mānase sāmprataṃ sthitāḥ  // 15.27 //
narmadā nāma teṣāṃ tu kanyā toyavahā sarit  /
bhūtāni yā pāvayati dakṣiṇāpathagāminī  // 15.28 //
tebhyaḥ sarve tu manavaḥ prajāḥ sargeṣu nirmitāḥ  /
jñātvā śrāddhāni kurvanti dharmābhāve 'pi sarvadā  // 15.29 //
tebhya eva punaḥ prāptuṃ prasādādyogasaṃtatim  /
pitṝṇāmādisarge tu śrāddhameva vinirmitam  // 15.30 //
sarveṣāṃ rājataṃ pātram athavā rajatānvitam  /
dattaṃ svadhā purodhāya pitṝn prīṇāti sarvadā  // 15.31 //
agnīṣomayamānāṃ tu kāryamāpyāyanaṃ budhaḥ  /
agnyabhāve 'pi viprasya prāṇāv api jale 'thavā  // 15.32 //
ajākarṇe 'śvakarṇe vā goṣṭhe vā salilāntike  /
pitṝṇām ambaraṃ sthānaṃ dakṣiṇā dikpraśasyate  // 15.33 //
prācīnāvītamudakaṃ tilāḥ savyāṅgameva ca  /
darbhā māṃsaṃ ca pāṭhīnaṃ gokṣīraṃ madhurā rasāḥ  // 15.34 //
khaḍgalohāmiṣamadhu- kuśaśyāmākaśālayaḥ  /
yavanīvāramudgekṣu- śuklapuṣpaghṛtāni ca  // 15.35 //
vallabhāni praśastāni pitṝṇāmiha sarvadā  /
dveṣyāṇi sampravakṣyāmi śrāddhe varjyāni yāni tu  // 15.36 //
masūraśaṇaniṣpāva- rājamāṣakusumbhikāḥ  /
padmabilvārkadhattūra- pāribhadrāṭarūṣakāḥ  // 15.37 //
na deyāḥ pitṛkāryeṣu payaścājāvikaṃ tathā  /
kodravodāracaṇakāḥ kapitthaṃ madhukātasī  // 15.38 //
etānyapi na deyāni pitṛbhyaḥ priyamicchatā  /
pitṝn prīṇāti yo bhaktyā te punaḥ prīṇayanti tam  // 15.39 //
yacchanti pitaraḥ puṣṭiṃ svargārogyaṃ prajāphalam  /
devakāryādapi punaḥ pitṛkāryaṃ viśiṣyate  // 15.40 //
devatānāṃ ca pitaraḥ pūrvamāpyāyanaṃ smṛtam  /
śīghraprasādāstvakrodhā niḥśastrāḥ sthirasauhṛdāḥ  // 15.41 //
śāntātmānaḥ śaucaparāḥ satataṃ priyavādinaḥ  /
bhaktānuraktāḥ sukhadāḥ pitaraḥ pūrvadevatāḥ  // 15.42 //
haviṣmatāmādhipatye śrāddhadevaḥ smṛto raviḥ  /
etadvaḥ sarvamākhyātaṃ pitṛvaṃśānukīrtanam  /
puṇyaṃ pavitram āyuṣyaṃ kīrtanīyaṃ sadā nṛbhiḥ  // 15.43 //


16
sūta uvāca
śrutvaitat sarvam akhilaṃ manuḥ papraccha keśavam  /
śrāddhakālaṃ ca vividhaṃ śrāddhabhedaṃ tathaiva ca  // 16.1 //
śrāddheṣu bhojanīyā ye ye ca varjyā dvijātayaḥ  /
kasminvāsarabhāge vā pitṛbhyaḥ śrāddhamācaret  // 16.2 //
kasmindattaṃ kathaṃ yāti śrāddhaṃ tu madhusūdana  /
vidhinā kena kartavyaṃ kathaṃ prīṇāti tatpitṝn  // 16.3 //
matsya uvāca
kuryādaharahaḥ śrāddham annādyenodakena vā  /
payomūlaphalair vāpi pitṛbhyaḥ prītimāvahan  // 16.4 //
nityaṃ naimittikaṃ kāmyaṃ trividhaṃ śrāddhamucyate  /
nityaṃ tāvatpravakṣyāmi arghyāvāhanavarjitam  // 16.5 //
adaivaṃ tadvijānīyāt pārvaṇaṃ parvasu smṛtam  /
pārvaṇaṃ trividhaṃ proktaṃ śṛṇu tāvanmahīpate  // 16.6 //
pārvaṇe ye niyojyāstu tāñśṛṇuṣva narādhipa  /
pañcāgniḥ snātakaścaiva trisuparṇaḥ ṣaḍaṅgavit  // 16.7 //
śrotriyaḥ śrotriyasuto vidhivākyaviśāradaḥ  /
sarvajño vedavinmantrī jñātavaṃśaḥ kulānvitaḥ  // 16.8 //
purāṇavettā dharmajñaḥ svādhyāyajapatatparaḥ  /
śivabhaktaḥ pitṛparaḥ sūryabhakto 'tha vaiṣṇavaḥ  // 16.9 //
brahmaṇyo yogavicchānto vijitātmā ca śīlavān  /
bhojayeccāpi dauhitraṃ yatnataḥ svasuhṛdgurūn  // 16.10 //
viṭpītaṃ mātulaṃ bandhum ṛtvigācāryasomapān  /
yaśca vyākurute vākyaṃ yaśca mīmāṃsate 'dhvaram  // 16.11 //
sāmasvaravidhijñaśca paṅktipāvanapāvanaḥ  /
sāmago brahmacārī ca vedayukto 'tha brahmavit  // 16.12 //
yatraite bhuñjate śrāddhe tadeva paramārthavat  /
ete bhojyāḥ prayatnena varjanīyānnibodha me  // 16.13 //
patito 'bhiśastaḥ klībaḥ piśunavyaṅgarogiṇaḥ  /
kunakhī śyāvadantaśca kuṇḍagolāśvapālakāḥ  // 16.14 //
parivittirniyuktātmā pramattonmattadāruṇāḥ  /
baiḍālī bakavṛttiśca dambhī devalakādayaḥ  // 16.15 //
kṛtaghnānnāstikāṃs tadvan mlecchadeśanivāsinaḥ  /
triśaṅkur barbaradrāva- vītadraviḍakoṅkaṇān  // 16.16 //
varjayelliṅginaḥ sarvāñ śrāddhakāle viśeṣataḥ  /
pūrvedyuraparedyurvā vinītātmā nimantrayet  // 16.17 //
nimantritānhi pitara upatiṣṭhanti tāndvijān  /
vāyubhūtā nu gacchanti tathāsīnānupāsate  // 16.18 //
dakṣiṇaṃ jānumālabhya tvaṃ mayā tu nimantritaḥ  /
evaṃ nimantrya niyamaṃ śrāvayetpitṛbāndhavān  // 16.19 //
akrodhanaiḥ śaucaparaiḥ satataṃ brahmacāribhiḥ  /
bhavitavyaṃ bhavadbhiśca mayā ca śrāddhakāriṇā  // 16.20 //
pitṛyajñaṃ vinirvartya tarpaṇākhyaṃ tu yo 'gnimān  /
piṇḍānvāhāryakaṃ kuryāc chrāddhamindukṣaye sadā  // 16.21 //
gomayenopalipte tu dakṣiṇapravaṇe sthale  /
śrāddhaṃ samācaredbhaktyā goṣṭhe vā jalasaṃnidhau  // 16.22 //
agnimānnirvapetpitryaṃ caruṃ ca samamuṣṭibhiḥ  /
pitṛbhyo nirvapāmīti sarvaṃ dakṣiṇato nyaset  // 16.23 //
abhighāryaṃ tataḥ kuryān nirvāpatrayamagrataḥ  /
te 'pi tasyāyatāḥ kāryāś caturaṅgulavistṛtāḥ  // 16.24 //
darvītrayaṃ tu kurvīta khādiraṃ rajatānvitam  /
ratnimātraṃ pariślakṣṇaṃ hastākārāgramuttamam  // 16.25 //
udapātraṃ ca kāṃsyaṃ ca mekṣaṇaṃ ca samitkuśān  /
tilāḥ pātrāṇi sadvāso gandhadhūpānulepanam  // 16.26 //
āharedapasavyaṃ tu sarvaṃ dakṣiṇataḥ śanaiḥ  /
evamāsādya tatsarvaṃ bhavanasyāgrato bhuvi  // 16.27 //
gomayenopaliptāyāṃ gomūtreṇa tu maṇḍalam  /
akṣatābhiḥ sapuṣpābhis tadabhyarcyāpasavyavat  // 16.28 //
viprāṇāṃ kṣālayet pādāv abhinandya punaḥ punaḥ  /
āsaneṣūpakḷpteṣu darbhavatsu vidhānavat  // 16.29 //
upaspṛṣṭodakānviprān upaveśyānumantrayet  /
dvau daive pitṛkṛtye trīn ekaikamubhayatra ca  // 16.30 //
bhojayedīśvaro 'pīha na kuryādvistaraṃ budhaḥ  /
daivapūrvaṃ niyojyātha viprānarghyādinā budhaḥ  // 16.31 //
agnau kuryādanujñāto viprair vipro yathāvidhi  /
svagṛhyoktavidhānena kāṃsye kṛtvā caruṃ tataḥ  // 16.32 //
agnīṣomayamābhyāṃ tu kuryādāpyāyanaṃ budhaḥ  /
dakṣiṇāgnau pratīte vā ya ekāgnirdvijottamaḥ  // 16.33 //
yajñopavītī nirvartya tataḥ paryukṣaṇādikam  /
prācīnāvītinā kāryam ataḥ sarvaṃ vijānatā  // 16.34 //
ṣaṭ ca tasmāddhaviḥśeṣāt piṇḍānkṛtvā tatodakam  /
dadyādudakapātrais tu satilaṃ savyapāṇinā  // 16.35 //
jānv ācya savyaṃ yatnena darbhayukto vimatsaraḥ  /
vidhāya lekhā yatnena nirvāpeṣvavanejanam  // 16.36 //
dakṣiṇābhimukhaḥ kuryāt kare darvīṃ nidhāya vai  /
nidhāya piṇḍam ekaikaṃ sarvadarbheṣvanukramāt  // 16.37 //
ninayedatha darbheṣu nāmagotrānukīrtanaiḥ  /
teṣu darbheṣu taṃ hastaṃ vimṛjyāllepabhāginām  // 16.38 //
tathaiva ca tataḥ kuryāt punaḥ pratyavanejanam  /
ṣaḍapyṛtūnnamaskṛtya gandhadhūpārhaṇādibhiḥ  // 16.39 //
evamāvāhya tatsarvaṃ vedamantrairyathoditaiḥ  /
ekāgnereka eva syān nirvāpo darvikā tathā  // 16.40 //
tataḥ kṛtvāntare dadyāt patnībhyo 'nnaṃ kuśeṣu saḥ  /
tadvatpiṇḍādike kuryād āvāhanavisarjanam  // 16.41 //
tato gṛhītvā piṇḍebhyo mātrāḥ sarvāḥ krameṇa tu  /
tāneva viprānprathamaṃ prāśayedyatnato naraḥ  // 16.42 //
yasmādannāddhṛtā mātrā bhakṣayanti dvijātayaḥ  /
anvāhāryakam ityuktaṃ tasmāttaccandrasaṃkṣaye  // 16.43 //
pūrvaṃ dattvā tu taddhaste sapavitraṃ tilodakam  /
tatpiṇḍāgraṃ prayaccheta svadhaiṣāmastviti bruvan  // 16.44 //
varṇayanbhojayedannaṃ miṣṭaṃ pūtaṃ ca sarvadā  /
varjayetkrodhaparatāṃ smarannārāyaṇaṃ harim  // 16.45 //
tṛptāñjñātvā tataḥ kuryād vikiransārvavarṇikam  /
sodakaṃ cānnamuddhṛtya salilaṃ prakṣipedbhuvi  // 16.46 //
ācānteṣu punardadyāj jalapuṣpākṣatodakam  /
svastivācanakaṃ sarvaṃ piṇḍopari samāharet  // 16.47 //
devādyantaṃ prakurvīta śrāddhanāśo 'nyathā bhavet  /
visṛjya brāhmaṇāṃstadvat teṣāṃ kṛtvā pradakṣiṇam  // 16.48 //
dakṣiṇāṃ diśamākāṅkṣan pitṝn yāceta mānavaḥ  /
dātāro no 'bhivardhantāṃ vedāḥ saṃtatireva ca  // 16.49 //
śraddhā ca no mā vyagamad bahu deyaṃ ca no 'stviti  /
annaṃ ca no bahu bhaved atithīṃśca labhemahi  // 16.50 //
yācitāraśca naḥ santu mā ca yāciṣma kaṃcana  /
etadastviti tatproktam anvāhāryaṃ tu pārvaṇam  // 16.51 //
yathendusaṃkṣaye tadvad anyatrāpi nigadyate  /
piṇḍāṃstu gojaviprebhyo dadyādagnau jale 'pi vā  // 16.52 //
viprāgrato vā vikired vayobhir abhivāśayet  /
patnī tu madhyamaṃ piṇḍaṃ prāśayed vinayānvitā  // 16.53 //
ādhatta pitaro garbham atra saṃtānavardhanam  /
tāvaduccheṣaṇaṃ tiṣṭhed yāvadviprā visarjitāḥ  // 16.54 //
vaiśvadevaṃ tataḥ kuryān nivṛtte pitṛkarmaṇi  /
iṣṭaiḥ saha tataḥ śānto bhuñjīta pitṛsevitam  // 16.55 //
punar bhojanamadhvānaṃ yānamāyāsamaithunam  /
śrāddhakṛcchrāddhabhukcaiva sarvametadvivarjayet  // 16.56 //
svādhyāyakalahaṃ caiva divāsvapnaṃ ca sarvadā  /
anena vidhinā śrāddhaṃ nirudvāsyeha nirvapet  // 16.57 //
kanyākumbhavṛṣasthe 'rke kṛṣṇapakṣeṣu sarvadā  /
yatra yatra pradātavyaṃ sapiṇḍīkaraṇāt param  /
tatrānena vidhānena deyam agnimatā sadā  // 16.58 //


17
sūta uvāca
ataḥ paraṃ pravakṣyāmi viṣṇunā yadudīritam  /
śrāddhaṃ sādhāraṇaṃ nāma bhuktimuktiphalapradam  // 17.1 //
ayane viṣuve yugme sāmānye cārkasaṃkrame  /
amāvāsyāṣṭakākṛṣṇa- pakṣe pañcadaśīṣu ca  // 17.2 //
ārdrāmaghārohiṇīṣu dravyabrāhmaṇasaṃgame  /
gajacchāyāvyatīpāte viṣṭivaidhṛtivāsare  // 17.3 //
vaiśākhasya tṛtīyāyāṃ navamī kārttikasya ca  /
pañcadaśī ca māghasya nabhasye ca trayodaśī  // 17.4 //
yugādayaḥ smṛtā hy etā dattasyākṣayyakārikāḥ  /
tathā manvantarādau ca deyaṃ śrāddhaṃ vijānatā  // 17.5 //
aśvayukchuklanavamī dvādaśī kārttike tathā  /
tṛtīyā caitramāsasya tathā bhādrapadasya ca  // 17.6 //
phālgunasya hy amāvāsyā pauṣasyaikādaśī tathā  /
āṣāḍhasyāpi daśamī māghamāsasya saptamī  // 17.7 //
śrāvaṇasyāṣṭamī kṛṣṇā tathāṣāḍhī ca pūrṇimā  /
kārttikī phālgunī caitrī jyeṣṭhapañcadaśī sitā  /
manvantarādayaś caitā dattasyākṣayyakārikāḥ  // 17.8 //
yasyāṃ manvantarasyādau rathamāste divākaraḥ  /
māghamāsasya saptamyāṃ sā tu syādrathasaptamī  // 17.9 //
pānīyamapyatra tilairvimiśraṃ dadyātpitṛbhyaḥ prayato manuṣyaḥ  /
śrāddhaṃ kṛtaṃ tena samāḥ sahasraṃ rahasyametatpitaro vadanti  // 17.10 //
vaiśākhyāmuparāgeṣu tathotsavamahālaye  /
tīrthāyatanagoṣṭheṣu dīpodyānagṛheṣu ca  // 17.11 //
vivikteṣūpalipteṣu śrāddhaṃ deyaṃ vijānatā  /
viprānpūrve pare cāhni vinītātmā nimantrayet  // 17.12 //
śīlavṛttaguṇopetān vayorūpasamanvitān  /
dvau daive trīṃstathā pitrya ekaikamubhayatra vā  // 17.13 //
bhojayetsusamṛddho 'pi na prasajjate vistare  /
viśvāndevānyavaiḥ puṣpair abhyarcyāsanapūrvakam  // 17.14 //
pūrayetpātrayugmaṃ tu sthāpya darbhapavitrakam  /
śaṃ no devīty apaḥ kuryād yavo 'sīti yavānapi  // 17.15 //
gandhapuṣpaiśca saṃpūjya vaiśvadevaṃ prati nyaset  /
viśve devāsa ityābhyām āvāhya vikiredyavān  // 17.16 //
gandhapuṣpair alaṃkṛtya yā divyetyarghyamutsṛjet  /
abhyarcya tābhyāmutsṛṣṭaṃ pitṛkāryaṃ samārabhet  // 17.17 //
darbhāsanaṃ tu dattvādau trīṇi pātrāṇi pūrayet  /
khapavitrāṇi kṛtvādau śaṃ no devīty apaḥ kṣipet  // 17.18 //
tilo 'sīti tilānkuryād gandhapuṣpādikaṃ punaḥ  /
pātraṃ vanaspatimayaṃ tathā parṇamayaṃ punaḥ  // 17.19 //
jalajaṃ vātha kurvīta tathā sāgarasambhavam  /
sauvarṇaṃ rājataṃ vāpi pitṝṇāṃ pātramucyate  // 17.20 //
rajatasya kathā vāpi darśanaṃ dānameva vā  /
rājatair bhājanaireṣām athavā rajatānvitaiḥ  // 17.21 //
vāryapi śraddhayā dattam akṣayāyopakalpate  /
tathārghyapiṇḍabhojyādau pitṝṇāṃ rājataṃ matam  // 17.22 //
śivanetrodbhavaṃ yasmāt tasmāttatpitṛvallabham  /
amaṅgalaṃ tadyatnena devakāryeṣu varjayet  // 17.23 //
evaṃ pātrāṇi saṃkalpya yathālābhaṃ vimatsaraḥ  /
yā divyeti piturnāma gotrairdarbhakaro nyaset  // 17.24 //
pitṝn āvāhayiṣyāmi kurvityuktastu taiḥ punaḥ  /
uśantastvā tathāyantu ṛgbhyām āvāhayetpitṝn  // 17.25 //
yā divyetyarghyamutsṛjya dadyādgandhādikāṃstataḥ  /
hastāttadudakaṃ pūrvaṃ dattvā saṃsravamāditaḥ  // 17.26 //
pitṛpātre nidhāyātha nyubjamuttarato nyaset  /
pitṛbhyaḥ sthānamasīti nidhāya pariṣecayet  // 17.27 //
tatrāpi pūrvavatkuryād agnikāryaṃ vimatsaraḥ  /
ubhābhyāmapi hastābhyām āhṛtya pariveṣayet  // 17.28 //
praśāntacittaḥ satataṃ darbhapāṇiraśeṣataḥ  /
guṇāḍhyaiḥ sūpaśākaistu nānābhakṣyairviśeṣataḥ  // 17.29 //
annaṃ tu sadadhikṣīraṃ goghṛtaṃ śarkarānvitam  /
māsaṃ prīṇāti vai sarvān pitṝnityāha keśavaḥ  // 17.30 //
dvau māsau matsyamāṃsena trīnmāsānhāriṇena tu  /
aurabhreṇātha caturaḥ śākunenātha pañca vai  // 17.31 //
ṣaṇmāsaṃ chāgamāṃsena tṛpyanti pitarastathā  /
sapta pārṣatamāṃsena tathāṣṭāv eṇajena tu  // 17.32 //
daśa māsāṃstu tṛpyanti varāhamahiṣāmiṣaiḥ  /
śaśakūrmajamāṃsena māsānekādaśaiva tu  // 17.33 //
saṃvatsaraṃ tu gavyena payasā pāyasena ca  /
rauraveṇa ca tṛpyanti māsānpañcadaśaiva tu  // 17.34 //
vārdhrīṇasasya māṃsena tṛptir dvādaśavārṣikī  /
kālaśākena cānantā khaḍgamāṃsena caiva hi  // 17.35 //
yatkiṃcinmadhusammiśraṃ gokṣīraṃ ghṛtapāyasam  /
dattamakṣayamityāhuḥ pitaraḥ pūrvadevatāḥ  // 17.36 //
svādhyāyaṃ śrāvayetpitryaṃ purāṇānyakhilāni ca  /
brahmaviṣṇvarkarudrāṇāṃ sūktāni vividhāni ca  // 17.37 //
indrāgnisomasūktāni pāvanāni svaśaktitaḥ  /
bṛhadrathaṃtaraṃ tadvaj jyeṣṭhasāma sarauhiṇam  // 17.38 //
tathaiva śāntikādhyāyaṃ madhubrāhmaṇameva ca  /
maṇḍalaṃ brāhmaṇaṃ tadvat prītikāri tu yatpunaḥ  // 17.39 //
viprāṇāmātmanaścaiva tatsarvaṃ samudīrayet  /
bhuktavatsu tatasteṣu bhojanopāntike nṛpa  // 17.40 //
sārvavarṇikamannādyaṃ saṃnīyāplāvya vāriṇā  /
samutsṛjedbhuktavatām agrato vikiredbhuvi  // 17.41 //
agnidagdhāstu ye jīvā ye 'pyadagdhāḥ kule mama  /
bhūmau dattena tṛpyantu prayāntu paramāṃ gatim  // 17.42 //
yeṣāṃ na mātā na pitā na bandhur na gotraśuddhirna tathānnam asti  /
tattṛptaye 'nnaṃ bhuvi dattametat prayāntu lokeṣu sukhāya tadvat  // 17.43 //
asaṃskṛtapramītānāṃ tyaktānāṃ kulayoṣitām  /
ucchiṣṭabhāgadheyaḥ syād darbhe vikirayośca yaḥ  // 17.44 //
tṛptāñjñātvodakaṃ dadyāt sakṛdviprakare tathā  /
upalipte mahīpṛṣṭhe gośakṛnmūtravāriṇā  // 17.45 //
nidhāya darbhānvidhivad dakṣiṇāgrān prayatnataḥ  /
sarvavarṇena cānnena piṇḍāṃstu pitṛyajñavat  // 17.46 //
avanejanapūrvaṃ tu nāmagotreṇa mānavaḥ  /
gandhadhūpādikaṃ dadyāt kṛtvā pratyavanejanam  // 17.47 //
jānvācya savyaṃ savyena pāṇinātha pradakṣiṇam  /
pitryamānīya tatkāryaṃ vidhivaddarbhapāṇinā  // 17.48 //
dīpaprajvālanaṃ tadvat kuryātpuṣpārcanaṃ budhaḥ  /
athācānteṣu cācamya vāri dadyātsakṛtsakṛt  // 17.49 //
atha puṣpākṣatān paścād akṣayyodakam eva ca  /
satilaṃ nāmagotreṇa dadyācchaktyā ca dakṣiṇām  // 17.50 //
gobhūhiraṇyavāsāṃsi bhavyāni śayanāni ca  /
dadyādyadiṣṭaṃ viprāṇām ātmanaḥ pitureva ca  // 17.51 //
vittaśāṭhyena rahitaḥ pitṛbhyaḥ prītimāvahan  /
tataḥ svadhāvācanakaṃ viśvadeveṣu codakam  // 17.52 //
dattvāśīḥ pratigṛhṇīyād dvijebhyaḥ prāṅmukho budhaḥ  /
aghorāḥ pitaraḥ santu santvityuktaḥ punar dvijaiḥ  // 17.53 //
gotraṃ tathā vardhatāṃ nas tathetyuktaśca taiḥ punaḥ  /
dātāro no 'bhivardhantām iti caivamudīrayet  // 17.54 //
etāḥ satyāśiṣaḥ santu santvityuktaśca taiḥ punaḥ  /
svastivācanakaṃ kuryāt piṇḍānuddhṛtya bhaktitaḥ  // 17.55 //
uccheṣaṇaṃ tu tattiṣṭhed yāvadviprā visarjitāḥ  /
tato grahabaliṃ kuryād iti dharmavyavasthitiḥ  // 17.56 //
uccheṣaṇaṃ bhūmigatam ajihmasyāstikasya ca  /
dāsavargasya tatpitryaṃ bhāgadheyaṃ pracakṣate  // 17.57 //
pitṛbhir nirmitaṃ pūrvam etadāpyāyanaṃ sadā  /
aputrāṇāṃ saputrāṇāṃ strīṇāmapi narādhipa  // 17.58 //
tatastānagrataḥ sthitvā parigṛhyodapātrakam  /
vāje vāja iti japan kuśāgreṇa visarjayet  // 17.59 //
bahiḥ pradakṣiṇāṃ kuryāt padāny aṣṭāv anuvrajan  /
bandhuvargeṇa sahitaḥ putrabhāryāsamanvitaḥ  // 17.60 //
nivṛtya praṇipatyātha paryukṣyāgniṃ samantravat  /
vaiśvadevaṃ prakurvīta naityakaṃ balimeva ca  // 17.61 //
tatastu vaiśvadevānte sabhṛtyasutabāndhavaḥ  /
bhuñjītātithisaṃyuktaḥ sarvaṃ pitṛniṣevitam  // 17.62 //
etaccānupanīto 'pi kuryātsarveṣu parvasu  /
śrāddhaṃ sādhāraṇaṃ nāma sarvakāmaphalapradam  // 17.63 //
bhāryāvirahito 'pyetat pravāsastho 'pi bhaktimān  /
śūdro 'pyamantravatkuryād anena vidhinā budhaḥ  // 17.64 //
tṛtīyamābhyudayikaṃ vṛddhiśrāddhaṃ taducyate  /
utsavānandasambhāre yajñodvāhādimaṅgale  // 17.65 //
mātaraḥ prathamaṃ pūjyāḥ pitarastadanantaram  /
tato mātāmahā rājan viśve devāstathaiva ca  // 17.66 //
pradakṣiṇopacāreṇa dadhyakṣataphalodakaiḥ  /
prāṅmukho nirvapetpiṇḍān dūrvayā ca kuśairyutān  // 17.67 //
sampannam ityabhyudaye dadyādarghyaṃ dvayor dvayoḥ  /
yugmā dvijātayaḥ pūjyā vastrakārtasvarādibhiḥ  // 17.68 //
tilārthastu yavaiḥ kāryo nāndīśabdānupūrvakaḥ  /
māṅgalyāni ca sarvāṇi vācayeddvijapuṃgavaiḥ  // 17.69 //
evaṃ śūdro 'pi sāmānya- vṛddhiśrāddhe 'pi sarvadā  /
namaskāreṇa mantreṇa kuryādāmānnataḥ sadā  // 17.70 //
dānapradhānaḥ śūdraḥ syād ityāha bhagavānprabhuḥ  /
dānena sarvakāmāptir asya saṃjāyate yataḥ  // 17.71 //


18
sūta uvāca
ekoddiṣṭamato vakṣye yaduktaṃ cakrapāṇinā  /
mṛte putrairyathā kāryam āśaucaṃ ca pitaryapi  // 18.1 //
daśāhaṃ śāvamāśaucaṃ brāhmaṇeṣu vidhīyate  /
kṣatriyeṣu daśa dve ca pakṣaṃ vaiśyeṣu caiva hi  // 18.2 //
śūdreṣu māsamāśaucaṃ sapiṇḍeṣu vidhīyate  /
naiśaṃ vākṛtacūḍasya trirātraṃ parataḥ smṛtam  // 18.3 //
janane 'pyevameva syāt sarvavarṇeṣu sarvadā  /
tathāsthisaṃcayād ūrdhvam aṅgasparśo vidhīyate  // 18.4 //
pretāya piṇḍadānaṃ tu dvādaśāhaṃ samācaret  /
pātheyaṃ tasya tatproktaṃ yataḥ prītikaraṃ mahat  // 18.5 //
tasmātpretapuraṃ preto dvādaśāhaṃ na nīyate  /
gṛhaṃ putraṃ kalatraṃ ca dvādaśāhaṃ prapaśyati  // 18.6 //
tasmānnidheyamākāśe daśarātraṃ payastathā  /
sarvadāhopaśāntyartham adhvaśramavināśanam  // 18.7 //
tata ekādaśāhe tu dvijānekādaśaiva tu  /
kṣatrādiḥ sūtakānte tu bhojayedayujo dvijān  // 18.8 //
dvitīye 'hni punastadvad ekoddiṣṭaṃ samācaret  /
āvāhanāgnaukaraṇaṃ daivahīnaṃ vidhānataḥ  // 18.9 //
ekaṃ pavitrameko 'rgha ekaḥ piṇḍo vidhīyate  /
upatiṣṭhatāmityetad deyaṃ paścāttilodakam  // 18.10 //
svaditaṃ vikiredbrūyād visarge cābhiramyatām  /
śeṣaṃ pūrvavadatrāpi kāryaṃ vedavidā pituḥ  // 18.11 //
anena vidhinā sarvam anumāsaṃ samācaret  /
sūtakāntāddvitīye 'hni śayyāṃ dadyādvilakṣaṇām  // 18.12 //
kāñcanaṃ puruṣaṃ tadvat phalavastrasamanvitām  /
saṃpūjya dvijadāmpatyaṃ nānābharaṇabhūṣaṇaiḥ  // 18.13 //
vṛṣotsargaṃ prakurvīta deyā ca kapilā śubhā  /
udakumbhaśca dātavyo bhakṣyabhojyasamanvitaḥ  // 18.14 //
yāvadabdaṃ naraśreṣṭha satilodakapūrvakam  /
tataḥ saṃvatsare pūrṇe sapiṇḍīkaraṇaṃ bhavet  // 18.15 //
sapiṇḍīkaraṇādūrdhvaṃ pretaḥ pārvaṇabhāgbhavet  /
vṛddhipūrveṣu yogyaśca gṛhasthaśca bhavettataḥ  // 18.16 //
sapiṇḍīkaraṇe śrāddhe devapūrvaṃ niyojayet  /
pitṝn nivāsayettatra pṛthakpretaṃ vinirdiśet  // 18.17 //
gandhodakatilairyuktaṃ kuryātpātracatuṣṭayam  /
arghārthaṃ pitṛpātreṣu pretapātraṃ prasecayet  // 18.18 //
tadvatsaṃkalpya caturaḥ piṇḍānpiṇḍapradastathā  /
ye samānā iti dvābhyām antyaṃ tu vibhajettridhā  // 18.19 //
caturthasya punaḥ kāryaṃ na kadācidato bhavet  /
tataḥ pitṛtvamāpannaḥ sarvatastuṣṭimāgataḥ  // 18.20 //
agniṣvāttādimadhyatvaṃ prāpnotyamṛtamuttamam  /
sapiṇḍīkaraṇādūrdhvaṃ tasmai tasmānna dīyate  // 18.21 //
pitṛṣveva tu dātavyaṃ tatpiṇḍo yeṣu saṃsthitaḥ  /
tataḥ prabhṛti saṃkrāntāv uparāgādiparvasu  // 18.22 //
tripiṇḍamācarecchrāddham ekoddiṣṭe mṛte 'hani  /
ekoddiṣṭaṃ parityajya mṛtāhe yaḥ samācaret  // 18.23 //
sadaiva pitṛhā sa syān mātṛbhrātṛvināśakaḥ  /
mṛtāhe pārvaṇaṃ kurvan nadho 'dho yāti mānavaḥ  // 18.24 //
saṃpṛkteṣvākulībhāvaḥ preteṣu tu gato bhavet  /
pratisaṃvatsaraṃ tasmād ekoddiṣṭaṃ samācaret  // 18.25 //
yāvadabdaṃ tu yo dadyād udakumbhaṃ vimatsaraḥ  /
pretāyānnasamāyuktaṃ so 'śvamedhaphalaṃ labhet  // 18.26 //
āmaśrāddhaṃ yadā kuryād vidhijñaḥ śrāddhadastadā  /
tenāgnaukaraṇaṃ kuryāt piṇḍāṃstenaiva nirvapet  // 18.27 //
tribhiḥ sapiṇḍīkaraṇe aśeṣatritaye pitā  /
yadā prāpsyati kālena tadā mucyeta bandhanāt  // 18.28 //
mukto 'pi lepabhāgitvaṃ prāpnoti kuśamārjanāt  /
lepabhājaścaturthādyāḥ pitrādyāḥ piṇḍabhāginaḥ  /
piṇḍadaḥ saptamasteṣāṃ sāpiṇḍyaṃ sāptapauruṣam  // 18.29 //


19
ṛṣaya ūcuḥ
kathaṃ kavyāni deyāni havyāni ca janairiha  /
gacchanti pitṛlokasthān prāpakaḥ ko 'tra gadyate  // 19.1 //
yadi martyo dvijo bhuṅkte hūyate yadi vānale  /
śubhāśubhātmakaiḥ pretair dattaṃ tadbhujyate katham  // 19.2 //
sūta uvāca
vasūnvadanti ca pitṝn rudrāṃścaiva pitāmahān  /
prapitāmahāṃstathādityān ityevaṃ vaidikī śrutiḥ  // 19.3 //
nāma gotraṃ pitṝṇāṃ tu prāpakaṃ havyakavyayoḥ  /
śrāddhasya mantrāḥ śraddhā ca upayojyātibhaktitaḥ  // 19.4 //
agniṣvāttādayasteṣām ādhipatye vyavasthitāḥ  /
nāmagotrakāladeśā bhavāntaragatānapi  // 19.5 //
prāṇinaḥ prīṇayantyete tadāhāratvamāgatān  /
devo yadi pitā jātaḥ śubhakarmānuyogataḥ  // 19.6 //
tasyānnamamṛtaṃ bhūtvā divyatve 'pyanugacchati  /
daityatve bhogarūpeṇa paśutve ca tṛṇaṃ bhavet  // 19.7 //
śrāddhānnaṃ vāyurūpeṇa sarpatve 'pyupatiṣṭhati  /
pānaṃ bhavati yakṣatve rākṣasatve tathāmiṣam  // 19.8 //
danujatve tathā māyā pretatve rudhirodakam  /
manuṣyatve 'nnapānāni nānābhogarasaṃ bhavet  // 19.9 //
ratiśaktiḥ striyaḥ kāntā bhojyaṃ bhojanaśaktitā  /
dānaśaktiḥ savibhavā rūpamārogyameva ca  // 19.10 //
śraddhā puṣpamidaṃ proktaṃ phalaṃ brahmasamāgamaḥ  /
āyuḥ putrān dhanaṃ vidyāṃ svargaṃ mokṣaṃ sukhāni ca  // 19.11 //
rājyaṃ caiva prayacchanti prītāḥ pitṛgaṇā nṛṇām  /
śrūyate ca purā mokṣaṃ prāptāḥ kauśikasūnavaḥ  /
pañcabhirjanmasambandhair gatā viṣṇoḥ paraṃ padam  // 19.12 //


20
ṛṣaya ūcuḥ
kathaṃ kauśikadāyādāḥ prāptāste yogamuttamam  /
pañcabhirjanmasambandhaiḥ kathaṃ karmakṣayo bhavet  // 20.1 //
sūta uvāca
kauśiko nāma dharmātmā kurukṣetre mahānṛṣiḥ  /
nāmataḥ karmatastasya sutānsapta nibodhata  // 20.2 //
śvasṛpaḥ krodhano hiṃsraḥ piśunaḥ kavireva ca  /
vāgduṣṭaḥ pitṛvartī ca gargaśiṣyāstadābhavan  // 20.3 //
pitaryuparate teṣām abhūddurbhikṣamulbaṇam  /
anāvṛṣṭiśca mahatī sarvalokabhayaṃkarī  // 20.4 //
gargādeśādvane dogdhrīṃ rakṣantaste tapodhanāḥ  /
khādāmaḥ kapilāmetāṃ vayaṃ kṣutpīḍitā bhṛśam  // 20.5 //
iti cintayatāṃ pāpaṃ laghuḥ prāha tadānujaḥ  /
yadyavaśyamiyaṃ vadhyā śrāddharūpeṇa yojyatām  // 20.6 //
śrāddhe niyojyamāneyaṃ pāpāttrāsyati no dhruvam  /
eva kurvityanujñātaḥ pitṛvartī tadānujaiḥ  // 20.7 //
cakre samāhitaḥ śrāddham upayujya ca tāṃ punaḥ  /
dvau daive bhrātarau kṛtvā pitre trīnapyanukramāt  // 20.8 //
tathaikamatithiṃ kṛtvā śrāddhadaḥ svayameva tu  /
cakāra mantravacchrāddhaṃ smaranpitṛparāyaṇaḥ  // 20.9 //
vinā gavā vatsako 'pi gurave viniveditaḥ  /
vyāghreṇa nihatā dhenur vatso 'yaṃ pratigṛhyatām  // 20.10 //
evaṃ sā bhakṣitā dhenuḥ saptabhistaistapodhanaiḥ  /
vaidikaṃ balamāśritya krūre karmaṇi nirbhayāḥ  // 20.11 //
tataḥ kālāvakṛṣṭāste vyādhā dāśapure 'bhavan  /
jātismaratvaṃ prāptāste pitṛbhāvena bhāvitāḥ  // 20.12 //
yatkṛtaṃ krūrakarmāpi śrāddharūpeṇa taistadā  /
tena te bhavane jātā vyādhānāṃ krūrakarmiṇām  // 20.13 //
pitṝṇāṃ caiva māhātmyāj jātā jātismarāstu te  /
te tu vairāgyayogena āsthāyānaśanaṃ punaḥ  // 20.14 //
jātismarāḥ sapta jātā mṛgāḥ kālañjare girau  /
nīlakaṇṭhasya purataḥ pitṛbhāvānubhāvitāḥ  // 20.15 //
tatrāpi jñānavairāgyāt prāṇānutsṛjya dharmataḥ  /
lokair avekṣyamāṇāste tīrthānte 'naśanena tu  // 20.16 //
mānase cakravākāste saṃjātāḥ sapta yoginaḥ  /
nāmataḥ karmataḥ sarvāñ chṛṇudhvaṃ dvijasattamāḥ  // 20.17 //
sumanāḥ kumudaḥ śuddhaś chidradarśī sunetrakaḥ  /
sunetraścāṃśumāṃścaiva saptaite yogapāragāḥ  // 20.18 //
yogabhraṣṭāstrayasteṣāṃ babhramuś cālpacetanāḥ  /
dṛṣṭvā vibhrājamānaṃ tam udyāne strībhiranvitam  // 20.19 //
krīḍantaṃ vividhairbhāvair mahābalaparākramam  /
pañcālānvayasambhūtaṃ prabhūtabalavāhanam  // 20.20 //
rājyakāmo 'bhavaccaikas teṣāṃ madhye jalaukasām  /
pitṛvartī ca yo vipraḥ śrāddhakṛt pitṛvatsalaḥ  // 20.21 //
aparau mantriṇau dṛṣṭvā prabhūtabalavāhanau  /
mantritve cakratuścecchām asminmartye dvijottamāḥ  // 20.22 //
tanmadhye ye tu niṣkāmās te babhūvur dvijottamāḥ  /
vibhrājaputrastveko 'bhūd brahmadatta iti smṛtaḥ  // 20.23 //
mantriputrau tathā cobhau kaṇḍarīkasubālakau  /
brahmadatto 'bhiṣiktaḥ san purohitavipaścitā  // 20.24 //
pañcālarājo vikrāntaḥ sarvaśāstraviśāradaḥ  /
yogavitsarvajantūnāṃ rutavettābhavattadā  // 20.25 //
tasya rājño 'bhavadbhāryā devalasyātmajā śubhā  /
saṃnatirnāma vikhyātā kapilā yābhavatpurā  // 20.26 //
pitṛkārye niyuktatvād abhavadbrahmavādinī  /
tayā cakāra sahitaḥ sa rājyaṃ rājanandanaḥ  // 20.27 //
kadācidudyānagatas tayā saha sa pārthivaḥ  /
dadarśa kīṭamithunam anaṅgakalahākulam  // 20.28 //
pipīlikāmanunayan paritaḥ kīṭakāmukaḥ  /
pañcabāṇābhitaptāṅgaḥ sagadgadamuvāca ha  // 20.29 //
na tvayā sadṛśī loke kāminī vidyate kvacit  /
madhyakṣāmātijaghanā bṛhadvakṣo 'bhigāminī  // 20.30 //
suvarṇavarṇā suśroṇī mañjūktā cāruhāsinī  /
sulakṣyanetrarasanā guḍaśarkaravatsalā  // 20.31 //
bhokṣyase mayi bhukte tvaṃ snāsi snāte tathā mayi  /
proṣite sati dīnā tvaṃ kruddhe 'pi bhayacañcalā  // 20.32 //
kimarthaṃ vada kalyāṇi saroṣavadanā sthitā  /
sā tamāha sakopā tu kim ālapasi māṃ śaṭha  // 20.33 //
tvayā modakacūrṇaṃ tu māṃ vihāya vineṣyatā  /
pradattaṃ samatikrānte dine 'nyasyāḥ samanmatha  // 20.34 //
pipīlika uvāca
tvatsādṛśyānmayā dattam anyasyai varavarṇini  /
tadekamaparādhaṃ me kṣantumarhasi bhāmini  // 20.35 //
naitadevaṃ kariṣyāmi punaḥ kvāpīha suvrate  /
spṛśāmi pādau satyena prasīda praṇatasya me  // 20.36 //
sūta uvāca
iti tadvacanaṃ śrutvā sā prasannābhavattataḥ  /
ātmānam arpayāmāsa mohanāya pipīlikā  // 20.37 //
brahmadatto 'pyaśeṣaṃ taṃ jñātvā vismayam āgamat  /
sarvasattvarutajñatvāt prasādāc cakrapāṇinaḥ  // 20.38 //


21
ṛṣaya ūcuḥ
kathaṃ sattvarutajño 'bhūd brahmadatto dharātale  /
taccābhavatkasya kule cakravākacatuṣṭayam  // 21.1 //
sūta uvāca
tasminneva pure jātās te ca cakrāhvayāstadā  /
vṛddhadvijasya dāyādā viprā jātismarāḥ purā  // 21.2 //
dhṛtimāṃstattvadarśī ca vidyācaṇḍas tapotsukaḥ  /
nāmataḥ karmataścaite sudaridrasya te sutāḥ  // 21.3 //
tapase buddhirabhavat tadā teṣāṃ dvijanmanām  /
yāsyāmaḥ paramāṃ siddhim ity ūcus te dvijottamāḥ  // 21.4 //
tatastadvacanaṃ śrutvā sudaridro mahātapāḥ  /
uvāca dīnayā vācā kimetaditi putrakāḥ  // 21.5 //
adharma eṣa iti vaḥ pitā tān abhyavārayat  /
vṛddhaṃ pitaramutsṛjya daridraṃ vanavāsinaḥ  // 21.6 //
ko nu dharmo 'tra bhavitā mattyāgādgatireva vā  /
ūcuste kalpitā vṛttis tava tāta vadasva tat  // 21.7 //
vittametatpuro rājñaḥ sa te dāsyati puṣkalam  /
dhanaṃ grāmasahasrāṇi prabhāte paṭhatastava  // 21.8 //
ye vipramukhyāḥ kurujāṅgaleṣu dāśāstathā dāśapure mṛgāśca  /
kālañjare sapta ca cakravākā ye mānase te vayamatra siddhāḥ  // 21.9 //
ityuktvā pitaraṃ jagmus te vanaṃ tapase punaḥ  /
vṛddho 'pi rājabhavanaṃ jagāmātmārthasiddhaye  // 21.10 //
anagho nāma vaibhrājaḥ pāñcālādhipatiḥ purā  /
putrārthī devadeveśaṃ hariṃ nārāyaṇaṃ prabhum  // 21.11 //
ārādhayāmāsa vibhuṃ tīvravrataparāyaṇaḥ  /
tataḥ kālena mahatā tuṣṭastasya janārdanaḥ  // 21.12 //
varaṃ vṛṇīṣva bhadraṃ te hṛdayenepsitaṃ nṛpa  /
evamuktastu devena vavre sa varamuttamam  // 21.13 //
putraṃ me dehi deveśa mahābalaparākramam  /
pāragaṃ sarvaśāstrāṇāṃ dhārmikaṃ yogināṃ param  // 21.14 //
sarvasattvarutajñaṃ me dehi yoginamātmajam  /
evamastviti viśvātmā tamāha parameśvaraḥ  // 21.15 //
paśyatāṃ sarvadevānāṃ tatraivāntaradhīyata  /
tataḥ sa tasya putro 'bhūd brahmadattaḥ pratāpavān  // 21.16 //
sarvasattvānukampī ca sarvasattvabalādhikaḥ  /
sarvasattvarutajñaśca sarvasattveśvareśvaraḥ  // 21.17 //
ahasattena yogātmā sa pipīlikarāgataḥ  /
yatra tatkīṭamithunaṃ ramamāṇamavasthitam  // 21.18 //
tataḥ sā saṃnatirdṛṣṭvā taṃ hasantaṃ suvismitā  /
kimapyāśaṅkya manasā tamapṛcchannareśvaram  // 21.19 //
saṃnatiruvāca
akasmādatihāsaste kimarthamabhavannṛpa  /
hāsyahetuṃ na jānāmi yadakāle kṛtaṃ tvayā  // 21.20 //
sūta uvāca
avadadrājaputro 'pi sa pipīlikabhāṣitam  /
rāgavāgbhiḥ samutpannam etaddhāsyaṃ varānane  // 21.21 //
na cānyatkāraṇaṃ kiṃcid dhāsyahetau śucismite  /
na sāmanyat tadā devī prāhālīkamidaṃ vacaḥ  // 21.22 //
ahamevādya hasitā na jīviṣye tvayādhunā  /
kathaṃ pipīlikālāpaṃ martyo vetti vinā surān  // 21.23 //
tasmāttvayāham eveha hasitā kimataḥ param  /
tato niruttaro rājā jijñāsustatpuro hareḥ  // 21.24 //
āsthāya niyamaṃ tasthau saptarātramakalmaṣaḥ  /
svapne prāha hṛṣīkeśaḥ prabhāte paryaṭanpuram  // 21.25 //
vṛddhadvijo yastadvākyāt sarvaṃ jñāsyasyaśeṣataḥ  /
ityuktvāntardadhe viṣṇuḥ prabhāte 'tha nṛpaḥ purāt  // 21.26 //
nirgacchanmantrisahitaḥ sabhāryo vṛddhamagrataḥ  /
gadantaṃ vipram āyāntaṃ taṃ vṛddhaṃ saṃdadarśa ha  // 21.27 //
brāhmaṇa uvāca
ye vipramukhyāḥ kurujāṅgaleṣu dāśāstathā dāśapure mṛgāśca  /
kālañjare sapta ca cakravākā ye mānase te vayamatra siddhāḥ  // 21.28 //
sūta uvāca
ityākarṇya vacastābhyāṃ sa papāta śucā tataḥ  /
jātismaratvamagamat tau ca mantrivarāv ubhau  // 21.29 //
kāmaśāstrapraṇetā ca bābhravyastu subālakaḥ  /
pāñcāla iti lokeṣu viśrutaḥ sarvaśāstravit  // 21.30 //
kaṇḍarīko 'pi dharmātmā vedaśāstrapravartakaḥ  /
bhūtvā jātismarau śokāt patitāv agratastadā  // 21.31 //
hā vayaṃ yogavibhraṣṭāḥ kāmataḥ karmabandhanāḥ  /
evaṃ vilapya bahuśas trayaste yogapāragāḥ  // 21.32 //
vismayācchrāddhamāhātmyam abhinandya punaḥ punaḥ  /
tatastasmai dhanaṃ dattvā prabhūtagrāmasaṃyutam  // 21.33 //
visṛjya brāhmaṇaṃ taṃ ca vṛddhaṃ dhanamudānvitam  /
ātmīyaṃ nṛpatiḥ putraṃ nṛpalakṣaṇasaṃyutam  // 21.34 //
viṣvaksenābhidhānaṃ tu rājā rājye 'bhyaṣecayat  /
mānase militāḥ sarve tataste yogināṃ varāḥ  // 21.35 //
brahmadattādayastasmin pitṛsaktā vimatsarāḥ  /
saṃnatiścābhavadbhraṣṭā mayaitatkila kāritam  // 21.36 //
rājyatyāgaphalaṃ sarvaṃ yadetad abhilaṣyate  /
tatheti prāha rājā tu punastāmabhinandayan  // 21.37 //
tvatprasādādidaṃ sarvaṃ mayaitatprāpyate phalam  /
tataste yogamāsthāya sarva eva vanaukasaḥ  // 21.38 //
brahmarandhreṇa paramaṃ padamāpustapobalāt  /
evamāyurdhanaṃ vidyāṃ svargaṃ mokṣaṃ sukhāni ca  // 21.39 //
prayacchanti sutānrājyaṃ nṛṇāṃ prītāḥ pitāmahāḥ  /
ya idaṃ pitṛmāhātmyaṃ brahmadattasya ca dvijāḥ  // 21.40 //
dvijebhyaḥ śrāvayedyo vā śṛṇotyatha paṭhettu vā  /
kalpakoṭiśataṃ sāgraṃ brahmaloke mahīyate  // 21.41 //


22
ṛṣaya ūcuḥ
kasminkāle ca tacchrāddham anantaphaladaṃ bhavet  /
kasminvāsarabhāge tu śrāddhakṛcchrāddhamācaret  /
tīrtheṣu keṣu ca kṛtaṃ śrāddhaṃ bahuphalaṃ bhavet  // 22.1 //
sūta uvāca
aparāhṇe tu samprāpte abhijidrauhiṇodaye  /
yatkiṃciddīyate tatra tadakṣayamudāhṛtam  // 22.2 //
tīrthāni yāni śastāni pitṝṇāṃ vallabhāni ca  /
nāmatastāni vakṣyāmi saṃkṣepeṇa dvijottamāḥ  // 22.3 //
pitṛtīrthaṃ gayā nāma sarvatīrthavaraṃ śubham  /
yatrāste devadeveśaḥ svayameva pitāmahaḥ  // 22.4 //
tatraiṣā pitṛbhirgītā gāthā bhāgamabhīpsubhiḥ  // 22.5 //
eṣṭavyā bahavaḥ putrā yadyeko 'pi gayāṃ vrajet  /
yajeta vāśvamedhena nīlaṃ vā vṛṣamutsṛjet  // 22.6 //
tathā vārāṇasī puṇyā pitṝṇāṃ vallabhā sadā  /
yatrāvimuktasāṃnidhyaṃ bhuktimuktiphalapradam  // 22.7 //
pitṝṇāṃ vallabhaṃ tadvat puṇyaṃ ca vimaleśvaram  /
pitṛtīrthaṃ prayāgaṃ tu sarvakāmaphalapradam  // 22.8 //
vaṭeśvarastu bhagavān mādhavena samanvitaḥ  /
yoganidrāśayastadvat sadā vasati keśavaḥ  // 22.9 //
daśāśvamedhikaṃ puṇyaṃ gaṅgādvāraṃ tathaiva ca  /
nandātha lalitā tadvat tīrthaṃ māyāpurī śubhā  // 22.10 //
tathā mitrapadaṃ nāma tataḥ kedāramuttamam  /
gaṅgāsāgaramityāhuḥ sarvatīrthamayaṃ śubham  // 22.11 //
tīrthaṃ brahmasarastadvac chatadrusalile hrade  /
tīrthaṃ tu naimiṣaṃ nāma sarvatīrthaphalapradam  // 22.12 //
gaṅgodbhedastu gomatyāṃ yatrodbhūtaḥ sanātanaḥ  /
tathā yajñavarāhastu devadevaśca śūlabhṛt  // 22.13 //
yatra tatkāñcanaṃ dvāram aṣṭādaśabhujo haraḥ  /
nemistu haricakrasya śīrṇā yatrābhavatpurā  // 22.14 //
tadetannaimiṣāraṇyaṃ sarvatīrthaniṣevitam  /
devadevasya tatrāpi vārāhasya tu darśanam  // 22.15 //
yaḥ prayāti sa pūtātmā nārāyaṇapadaṃ vrajet  /
kṛtaśaucaṃ mahāpuṇyaṃ sarvapāpaniṣūdanam  // 22.16 //
yatrāste nārasiṃhastu svayameva janārdanaḥ  /
tīrthamikṣumatī nāma pitṝṇāṃ vallabhaṃ sadā  // 22.17 //
saṃgame yatra tiṣṭhanti gaṅgāyāḥ pitaraḥ sadā  /
kurukṣetraṃ mahāpuṇyaṃ sarvatīrthasamanvitam  // 22.18 //
tathā ca sarayūḥ puṇyā sarvadevanamaskṛtā  /
irāvatī nadī tadvat pitṛtīrthādhivāsinī  // 22.19 //
yamunā devikā kālī candrabhāgā dṛṣadvatī  /
nadī veṇumatī puṇyā parā vetravatī tathā  // 22.20 //
pitṝṇāṃ vallabhā hy etāḥ śrāddhe koṭiguṇā matāḥ  /
jambūmārgaṃ mahāpuṇyaṃ yatra mārgo hi lakṣyate  // 22.21 //
adyāpi pitṛtīrthaṃ tat sarvakāmaphalapradam  /
nīlakuṇḍamiti khyātaṃ pitṛtīrthaṃ dvijottamāḥ  // 22.22 //
tathā rudrasaraḥ puṇyaṃ saro mānasameva ca  /
mandākinī tathācchodā vipāśātha sarasvatī  // 22.23:1 //
pūrvamitrapadaṃ tadvad vaidyanāthaṃ mahāphalam  /
kṣiprā nadī mahākālas tathā kālañjaraṃ śubham  // 22.23:2 //
vaṃśodbhedaṃ harodbhedaṃ gaṅgodbhedaṃ mahāphalam  /
bhadreśvaraṃ viṣṇupadaṃ narmadādvārameva ca  // 22.24 //
gayāpiṇḍapradānena samānyāhurmaharṣayaḥ  /
etāni pitṛtīrthāni sarvapāpaharāṇi ca  // 22.25 //
smaraṇādapi lokānāṃ kim u śrāddhakṛtāṃ nṛṇām  /
oṃkāraṃ pitṛtīrthaṃ ca kāverī kapilodakam  // 22.26 //
sambhedaś caṇḍavegāyās tathaivāmarakaṇṭakam  /
kurukṣetrācchataguṇaṃ tasminsnānādikaṃ bhavet  // 22.27 //
śukratīrthaṃ ca vikhyātaṃ tīrthaṃ someśvaraṃ param  /
sarvavyādhiharaṃ puṇyaṃ śatakoṭiphalādhikam  // 22.28 //
śrāddhe dāne tathā home svādhyāye jalasaṃnidhau  /
kāyāvarohaṇaṃ nāma tathā carmaṇvatī nadī  // 22.29 //
gomatī varaṇā tadvat tīrthamauśanasaṃ param  /
bhairavaṃ bhṛgutuṅgaṃ ca gaurītīrthamanuttamam  // 22.30 //
tīrthaṃ vaināyakaṃ nāma bhadreśvaramataḥ param  /
tathā pāpaharaṃ nāma puṇyātha tapatī nadī  // 22.31 //
mūlatāpī payoṣṇī ca payoṣṇīsaṃgamastathā  /
mahābodhiḥ pāṭalā ca nāgatīrthamavantikā  // 22.32 //
tathā veṇā nadī puṇyā mahāśālaṃ tathaiva ca  /
mahārudraṃ mahāliṅgaṃ daśārṇā ca nadī śubhā  // 22.33 //
śatarudrā śatāhvā ca tathā viśvapadaṃ param  /
aṅgāravāhikā tadvan nadau tau śoṇaghargharau  // 22.34 //
kālikā ca nadī puṇyā vitastā ca nadī tathā  /
etāni pitṛtīrthāni śasyante snānadānayoḥ  // 22.35 //
śrāddhameteṣu yaddattaṃ tadanantaphalaṃ smṛtam  /
droṇī vāṭanadī dhārā saritkṣīranadī tathā  // 22.36 //
gokarṇaṃ gajakarṇaṃ ca tathā ca puruṣottamaḥ  /
dvārakā kṛṣṇatīrthaṃ ca tathārbudasarasvatī  // 22.37 //
nadī maṇimatī nāma tathā ca girikarṇikā  /
dhūtapāpaṃ tathā tīrthaṃ samudro dakṣiṇastathā  // 22.38 //
eteṣu pitṛtīrtheṣu śrāddhamānantyamaśnute  /
tīrthaṃ meghakaraṃ nāma svayameva janārdanaḥ  // 22.39 //
yatra śārṅgadharo viṣṇur mekhalāyāmavasthitaḥ  /
tathā mandodarītīrthaṃ tīrthaṃ campā nadī śubhā  // 22.40 //
tathā sāmalanāthaśca mahāśālanadī tathā  /
cakravākaṃ carmakoṭaṃ tathā janmeśvaraṃ mahat  // 22.41 //
arjunaṃ tripuraṃ caiva siddheśvaramataḥ param  /
śrīśailaṃ śāṃkaraṃ tīrthaṃ nārasiṃhamataḥ param  // 22.42 //
mahendraṃ ca tathā puṇyam atha śrīraṅgasaṃjñitam  /
eteṣvapi sadā śrāddham anantaphaladaṃ smṛtam  // 22.43 //
darśanādapi caitāni sadyaḥ pāpaharāṇi vai  /
tuṅgabhadrā nadī puṇyā tathā bhīmarathī sarit  // 22.44 //
bhīmeśvaraṃ kṛṣṇaveṇā kāverī kuḍmalā nadī  /
nadī godāvarī nāma trisaṃdhyā tīrthamuttamam  // 22.45 //
tīrthaṃ traiyambakaṃ nāma sarvatīrthanamaskṛtam  /
yatrāste bhagavānīśaḥ svayameva trilocanaḥ  // 22.46 //
śrāddhameteṣu sarveṣu koṭikoṭiguṇaṃ bhavet  /
smaraṇādapi pāpāni naśyanti śatadhā dvijāḥ  // 22.47 //
śrīparṇī tāmraparṇī ca jayā tīrthamanuttamam  /
tathā matsyanadī puṇyā śivadhāraṃ tathaiva ca  // 22.48 //
bhadratīrthaṃ ca vikhyātaṃ pampātīrthaṃ ca śāśvatam  /
puṇyaṃ rāmeśvaraṃ tadvad elāpuramalaṃ puram  // 22.49 //
aṅgabhūtaṃ ca vikhyātam āmardakam alambhuṣam  /
āmrātakeśvaraṃ tadvad ekāmbhakamataḥ param  // 22.50 //
govardhanaṃ hariścandraṃ kṛpucandraṃ pṛthūdakam  /
sahasrākṣaṃ hiraṇyākṣaṃ tathā ca kadalī nadī  // 22.51 //
rāmādhivāsastatrāpi tathā saumitrisaṃgamaḥ  /
indrakīlaṃ mahānādaṃ tathā ca priyamelakam  // 22.52 //
etānyapi sadā śrāddhe praśastānyadhikāni tu  /
eteṣu sarvadevānāṃ sāṃnidhyaṃ dṛśyate yataḥ  // 22.53 //
dānameteṣu sarveṣu dattaṃ koṭiśatādhikam  /
bāhudā ca nadīpuṇyā tathā siddhavanaṃ śubham  // 22.54 //
tīrthaṃ pāśupataṃ nāma nadī pārvatikā śubhā  /
śrāddhameteṣu sarveṣu dattaṃ koṭiśatottaram  // 22.55 //
tathaiva pitṛtīrthaṃ tu yatra godāvarī nadī  /
yutā liṅgasahasreṇa sarvāntarajalāvahā  // 22.56 //
jāmadagnyasya tattīrthaṃ kramādāyātamuttamam  /
pratīkasya bhayādbhinnaṃ yatra godāvarī nadī  // 22.57 //
tattīrthaṃ havyakavyānām apsaroyugasaṃjñitam  /
śrāddhāgnikāryadāneṣu tathā koṭiśatādhikam  // 22.58 //
tathā sahasraliṅgaṃ ca rāghaveśvaramuttamam  /
sendraphenā nadī puṇyā yatrendraḥ patitaḥ purā  // 22.59 //
nihatya namuciṃ śakras tapasā svargamāptavān  /
tatra dattaṃ naraiḥ śrāddham anantaphaladaṃ bhavet  // 22.60 //
tīrthaṃ tu puṣkaraṃ nāma śālagrāmaṃ tathaiva ca  /
somapānaṃ ca vikhyātaṃ yatra vaiśvānarālayam  // 22.61 //
tīrthaṃ sārasvataṃ nāma svāmitīrthaṃ tathaiva ca  /
malaṃdarā nadī puṇyā kauśikī candrikā tathā  // 22.62 //
vaidarbhā vātha vairā ca payoṣṇī prāṅmukhā parā  /
kāverī cottarā puṇyā tathā jālaṃdharo giriḥ  // 22.63 //
eteṣu śrāddhatīrtheṣu śrāddhamānantyamaśnute  /
lohadaṇḍaṃ tathā tīrthaṃ citrakūṭastathaiva ca  // 22.64 //
vindhyayogaśca gaṅgāyās tathā nadītaṭaṃ śubham  /
kubjābhraṃ tu tathā tīrtham urvaśīpulinaṃ tathā  // 22.65 //
saṃsāramocanaṃ tīrthaṃ tathaiva ṛṇamocanam  /
eteṣu pitṛtīrtheṣu śrāddham ānantyamaśnute  // 22.66 //
aṭṭahāsaṃ tathā tīrthaṃ gautameśvarameva ca  /
tathā vasiṣṭhatīrthaṃ nu hārītaṃ tu tataḥ param  // 22.67 //
brahmāvartaṃ kuśāvartaṃ hayatīrthaṃ tathaiva ca  /
piṇḍārakaṃ ca vikhyātaṃ śaṅkhoddhāraṃ tathaiva ca  // 22.68 //
ghaṇṭeśvaraṃ bilvakaṃ ca nīlaparvatameva ca  /
tathā ca dharaṇītīrthaṃ rāmatīrthaṃ tathaiva ca  // 22.69 //
aśvatīrthaṃ ca vikhyātam anantaṃ śrāddhadānayoḥ  /
tīrthaṃ vedaśiro nāma tathaivaughavatī nadī  // 22.70 //
tīrthaṃ vasupradaṃ nāma chāgalāṇḍaṃ tathaiva ca  /
eteṣu śrāddhadātāraḥ prayānti paramaṃ padam  // 22.71 //
tathāca badarītīrthaṃ gaṇatīrthaṃ tathaiva ca  /
jayantaṃ vijayaṃ caiva śakratīrthaṃ tathaiva ca  // 22.72 //
śrīpateśca tathā tīrthaṃ tīrthaṃ raivatakaṃ tathā  /
tathaiva śāradātīrthaṃ bhadrakāleśvaraṃ tathā  // 22.73 //
vaikuṇṭhatīrthaṃ ca paraṃ bhīmeśvaramathāpi vā  /
eteṣu śrāddhadātāraḥ prayānti paramāṃ gatim  // 22.74 //
tīrthaṃ mātṛgṛhaṃ nāma karavīrapuraṃ tathā  /
kuśeśayaṃ ca vikhyātaṃ gaurīśikharameva ca  // 22.75 //
nakuleśasya tīrthaṃ ca kardamālaṃ tathaiva ca  /
diṇḍipuṇyakaraṃ tadvat puṇḍarīkapuraṃ tathā  // 22.76 //
saptagodāvarītīrthaṃ sarvatīrtheśvareśvaram  /
tatra śrāddhaṃ pradātavyam anantaphalamīpsubhiḥ  // 22.77 //
eṣa tūddeśataḥ proktas tīrthānāṃ saṃgraho mayā  /
vāgīśo 'pi na śaknoti vistarāt kim u mānuṣaḥ  // 22.78 //
satyaṃ tīrthaṃ dayā tīrthaṃ tīrthamindriyanigrahaḥ  /
varṇāśramāṇāṃ gehe 'pi tīrthaṃ tu samudāhṛtam  // 22.79 //
etattīrtheṣu yacchrāddhaṃ tatkoṭiguṇamiṣyate  /
yasmāttasmātprayatnena tīrthe śrāddhaṃ samācaret  // 22.80 //
prātaḥkālo muhūrtāṃs trīn saṃgavas tāvadeva tu  /
madhyāhnastrimuhūrtaḥ syād aparāhṇastataḥ param  // 22.81 //
sāyāhnastrimuhūrtaḥ syāc chrāddhaṃ tatra na kārayet  /
rākṣasī nāma sā velā garhitā sarvakarmasu  // 22.82 //
ahno muhūrtā vikhyātā daśa pañca ca sarvadā  /
tatrāṣṭamo muhūrto yaḥ sa kālaḥ kutapaḥ smṛtaḥ  // 22.83 //
madhyāhne sarvadā yasmān mandībhavati bhāskaraḥ  /
tasmādanantaphaladas tadārambho viśiṣyate  // 22.84 //
madhyāhnaḥ khaḍgapātraṃ ca tathā nepālakambalaḥ  /
rūpyaṃ darbhāstilā gāvo dauhitraścāṣṭamaḥ smṛtaḥ  // 22.85 //
pāpaṃ kutsitamityāhus tasya saṃtāpakāriṇaḥ  /
aṣṭāv ete yatastasmāt kutapā iti viśrutāḥ  // 22.86 //
ūrdhvaṃ muhūrtātkutapād yanmuhūrtacatuṣṭayam  /
muhūrtapañcakaṃ caitat svadhābhavanamiṣyate  // 22.87 //
viṣṇor dehasamudbhūtāḥ kuśāḥ kṛṣṇāstilāstathā  /
śrāddhasya rakṣaṇāyālam etatprāhurdivaukasaḥ  // 22.88 //
tilodakāñjalirdeyo jalasthaistīrthavāsibhiḥ  /
sadarbhahastenaikena śrāddhamevaṃ viśiṣyate  // 22.89 //
śrāddhasādhanakāle tu pāṇinaikena dīyate  /
tarpaṇaṃ tūbhayenaiva vidhireṣa sadā smṛtaḥ  // 22.90 //
sūta uvāca
puṇyaṃ pavitramāyuṣyaṃ sarvapāpavināśanam  /
purā malaye na kathitaṃ tīrthaśrāddhānukīrtanam  // 22.91 //
śṛṇoti yaḥ paṭhedvāpi śrīmānsaṃjāyate naraḥ  // 22.92 //
śrāddhakāle ca vaktavyaṃ tathā tīrthanivāsibhiḥ  /
sarvapāpopaśāntyartham alakṣmīnāśanaṃ param  // 22.93 //
idaṃ pavitraṃ yaśaso nidhānam idaṃ mahāpāpaharaṃ ca puṃsām  /
brahmārkarudrairapi pūjitaṃ ca śrāddhasya māhātmyamuśanti tajjñāḥ  // 22.94 //

ṛṣaya ūcuḥ
somaḥ pitṝṇāmadhipaḥ kathaṃ śāstraviśārada  /
tadvaṃśyā ye ca rājāno babhūvuḥ kīrtivardhanāḥ  // 23.1 //
sūta uvāca
ādiṣṭo brahmaṇā pūrvam atriḥ sargavidhau purā  /
anuttamaṃ nāma tapaḥ sṛṣṭyarthaṃ taptavānprabhuḥ  // 23.2 //
yadānandakaraṃ brahma jagatkleśavināśanam  /
brahmaviṣṇvarkarudrāṇām abhyantaramatīndriyam  // 23.3 //
śāntikṛc chāntamanasas tadantarnayane sthitam  /
māhātmyāttapasā viprāḥ paramānandakārakam  // 23.4 //
yasmādumāpatiḥ sārdham umayā tamadhiṣṭhitaḥ  /
taṃ dṛṣṭvā cāṣṭamāṃśena tasmātsomo 'bhavacchiśuḥ  // 23.5 //
adhaḥ susrāva netrābhyāṃ dhāma taccāmbusambhavam  /
dīpayanviśvamakhilaṃ jyotsnayā sacarācaram  // 23.6 //
taddiśo jagṛhurdhāma strīrūpeṇa sutecchayā  /
garbho bhūtvodare tāsām āsthito 'bdaśatatrayam  // 23.7 //
āśāstaṃ mumucurgarbham aśaktā dhāraṇe tataḥ  /
samādāyātha taṃ garbham ekīkṛtya caturmukhaḥ  // 23.8 //
yuvānamakarodbrahmā sarvāyudhadharaṃ naram  /
syandane 'tha sahasrāśve vedaśaktimaye prabhuḥ  // 23.9 //
āropya lokamanayad ātmīyaṃ sa pitāmaha  /
tatra brahmarṣibhiḥ proktam asmatsvāmī bhavatvayam  // 23.10 //
ṛṣibhirdevagandharvair oṣadhībhistathaiva ca  /
tuṣṭuvuḥ somadevatyair brahmādyā mantrasaṃgrahaiḥ  // 23.11 //
stūyamānasya tasyābhūd adhiko dhāmasambhavaḥ  /
tejovitānādabhavad bhuvi divyauṣadhīgaṇaḥ  // 23.12 //
taddīptiradhikā tasmād rātrau bhavati sarvadā  /
tenauṣadhīśaḥ somo 'bhūd dvijeśaścāpi gadyate  // 23.13 //
vedadhāmarasaṃ cāpi yadidaṃ candramaṇḍalam  /
kṣīyate vardhate caiva śukle kṛṣṇe ca sarvadā  // 23.14 //
viṃśatiṃ ca tathā sapta dakṣaḥ prācetaso dadau  /
rūpalāvaṇyasaṃyuktās tasmai kanyāḥ suvarcasaḥ  // 23.15 //
tataḥ padmasahasrāṇāṃ sahasrāṇi daśaiva tu  /
tapaścacāra śītāṃśur viṣṇudhyānaikatatparaḥ  // 23.16 //
tatastuṣṭastu bhagavāṃs tasmai nārāyaṇo hariḥ  /
varaṃ vṛṇīṣva provāca paramātmā janārdanaḥ  // 23.17 //
tato vavre varānsomaḥ śakralokaṃ jayāmyaham  /
pratyakṣameva bhoktāro bhavantu mama mandire  // 23.18 //
rājasūye suragaṇā brahmādyāḥ santu me dvijāḥ  /
rakṣaḥpālaḥ śivo 'smākam āstāṃ śūladharo haraḥ  // 23.19 //
tathetyuktaḥ sa ājahre rājasūyaṃ tu viṣṇunā  /
hotātrir bhṛguradhvaryur udgātābhūccaturmukhaḥ  // 23.20 //
brahmatvamagamattasya upadraṣṭā hariḥ svayam  /
sadasyāḥ sanakādyāstu rājasūyavidhau smṛtāḥ  // 23.21 //
camasādhvaryavastatra viśve devā daśaiva tu  /
trailokyaṃ dakṣiṇā tena ṛtvigbhyaḥ pratipāditam  // 23.22 //
tataḥ samāpte 'vabhṛthe tadrūpālokanecchavaḥ  /
kāmabāṇābhitaptāṅgyo nava devyaḥ siṣevire  // 23.23 //
lakṣmīrnārāyaṇaṃ tyaktvā sinīvālī ca kardamam  /
dyutirvibhāvasuṃ tadvat tuṣṭirdhātāramavyayam  // 23.24 //
prabhā prabhākaraṃ tyaktvā haviṣmantaṃ kuhūḥ svayam  /
kīrtirjayantaṃ bhartāraṃ vasurmārīcakaśyapam  // 23.25 //
dhṛtis tyaktvā pītaṃ nandiṃ somamevābhajaṃstadā  /
svakīyā iva somo 'pi kāmayāmāsa tāstadā  // 23.26 //
evaṃ kṛtāpacārasya tāsāṃ bhartṛgaṇastadā  /
na śaśākāpacārāya śāpaiḥ śastrādibhiḥ punaḥ  // 23.27 //
tathāpyarājata vidhur daśadhā bhāvayandiśaḥ  /
somaḥ prāpyātha duṣprāpyam aiśvaryamṛṣisaṃskṛtam  /
saptalokaikanāthatvam avāpa tapasā tadā  // 23.28 //
kadācidudyānagatāmapaśyad anekapuṣpābharaṇaiśca śobhitām  /
bṛhannitambastanabhārakhedāt puṣpasya bhaṅge 'pyatidurbalāṅgīm  // 23.29 //
bhāryāṃ ca tāṃ devaguror anaṅga bāṇābhirāmāyatacārunetrām  /
tārāṃ sa tārādhipatiḥ smarārtaḥ keśeṣu jagrāha viviktabhūmau  // 23.30 //
sāpi smarārtā saha tena reme tadrūpakāntyā hṛtamānasena  /
ciraṃ vihṛtyātha jagāma tārāṃ vidhurgṛhītvā svagṛhaṃ tato 'pi  // 23.31 //
na tṛptirāsīcca gṛhe 'pi tasya tārānuraktasya sukhāgameṣu  /
bṛhaspatis tadvirahāgnidagdhas taddhyānaniṣṭhaikamanā babhūva  // 23.32 //
śaśāka śāpaṃ na ca dātum asmai na mantraśastrāgniviṣairaśeṣaiḥ  /
tasyāpakartuṃ vividhairupāyair naivābhicārairapi vāgadhīśaḥ  // 23.33 //
sa yācayāmāsa tatastu dainyāt somaṃ svabhāryārthamanaṅgataptaḥ  /
sa yācyamāno 'pi dadau na tārāṃ bṛhaspatestatsukhapāśabaddhaḥ  // 23.34 //
maheśvareṇātha caturmukheṇa sādhyair marudbhiḥ saha lokapālaiḥ  /
dadau yadā tāṃ na kathaṃcid indus tadā śivaḥ krodhaparo babhūva  // 23.35 //
yo vāmadevaḥ prathitaḥ pṛthivyām anekarudrārcitapādapadmaḥ  /
tataḥ saśiṣyo giriśaḥ pinākī bṛhaspatisnehavaśānubaddhaḥ  // 23.36 //
dhanurgṛhītvājagavaṃ purārir jagāma bhūteśvarasiddhajuṣṭaḥ  /
yuddhāya somena viśeṣadīpta- tṛtīyanetrānalabhīmavaktraḥ  // 23.37 //
sahaiva jagmuśca gaṇeśakādyā viṃśaccatuḥṣaṣṭigaṇāstrayuktāḥ  /
yakṣeśvaraḥ koṭiśatair anekair yuto 'nvagāt syandanasaṃsthitānām  // 23.38 //
vetālayakṣoragakiṃnarāṇāṃ padmena caikena tathārbudena  /
lakṣais tribhir dvādaśabhī rathānāṃ somo 'pyagāttatra vivṛddhamanyuḥ  // 23.39 //
nakṣatradaityāsurasainyayuktaḥ śanaiścarāṅgārakavṛddhatejāḥ  /
jagmurbhayaṃ sapta tathaiva lokāś cacāla bhūr dvīpasamudragarbhā  // 23.40 //
sa somamevābhyagamatpinākī gṛhītadīptāstraviśālavahniḥ  /
athābhavad bhīṣaṇabhīmasena- sainyadvayasyāpi mahāhavo 'sau  // 23.41 //
aśeṣasattvakṣayakṛtpravṛddhas tīkṣṇāyudhāstrajvalanaikarūpaḥ  /
śastrairathānyonyamaśeṣasainyaṃ dvayorjagāma kṣayamugratīkṣṇaiḥ  // 23.42 //
patanti śastrāṇi tathojjvalāni svarbhūmipātālamatho dahanti  /
rudraḥ kopādbrahmaśīrṣaṃ mumoca somo 'pi somāstramamoghavīryam  // 23.43 //
tayornipātena samudrabhūmyor athāntarikṣasya ca bhītirāsīt  /

asaṃśayaṃ māmasurā dviṣanti ye me śiṣyānāgatānsūdayanti  // 25.47 //
abrāhmaṇaṃ kartumicchanti raudrā ebhir vyarthaṃ prastuto dānavairhi  /
tatkarmaṇāpyasya bhavedihāntaḥ kaṃ brahmahatyā na dahedapīndram  // 25.48 //
sa tenāpṛṣṭo vidyayā copahūtaḥ śanairvācaṃ jaṭhare vyājahāra  /
tamabravītkena cehopanīto mamodare tiṣṭhasi brūhi vatsa  // 25.49 //
kaca uvāca
bhavatprasādānna jahāti māṃ smṛtiḥ sarvaṃ smareyaṃ yacca yathā ca vṛttam  /
na tv evaṃ syāttapasaḥ kṣayo me tata kleśaṃ ghorataraṃ smarāmi  // 25.50 //
asuraiḥ surāyāṃ bhavato 'smi datto hatvā dagdhvā cūrṇayitvā ca kāvya  /
brāhmīṃ māyāṃ tv āsurī tv atra māyā tvayi sthite katham evābhibādhate  // 25.51 //
śukra uvāca
kiṃ te priyaṃ karavāṇyadya vatse vinaiva me jīvitaṃ syātkacasya  /
nānyatra kukṣermama bhedanācca dṛśyet kaco madgato devayāni  // 25.52 //
devayānyuvāca
dvau māṃ śokāv agnikalpau dahetāṃ kacasya nāśastava caivopaghātaḥ  /
kacasya nāśe mama nāsti śarma tavopaghāte jīvituṃ nāsmi śaktā  // 25.53 //
śukra uvāca
saṃsiddharūpo 'si bṛhaspateḥ suta yattvāṃ bhaktaṃ bhajate devayānī  /
vidyāmimāṃ prāpnuhi jīvanīṃ tvaṃ na cedindraḥ kacarūpī tvamadya  // 25.54 //
na nivartetpunarjīvan kaścidanyo mamodarāt  /
brāhmaṇaṃ varjayitvaikaṃ tasmādvidyām avāpnuhi  // 25.55 //
putro bhūtvā niṣkramasvodarānme bhittvā kukṣiṃ jīvaya māṃ ca tāta  /
avekṣethā dharmavatīmavekṣāṃ guroḥ sakāśātprāpya vidyāṃ savidyaḥ  // 25.56 //
śaunaka uvāca
guroḥ sakāśātsamavāpya vidyāṃ bhittvā kukṣiṃ nirvicakrāma vipraḥ  /
prāleyādreḥ śuklamudbhidya śṛṅgaṃ rātryāgame paurṇamāsyāmivenduḥ  // 25.57 //
dṛṣṭvā ca taṃ patitaṃ vedarāśim utthāpayāmāsa tataḥ kaco 'pi  /
vidyāṃ siddhāṃ tāmavāpyābhivādya tataḥ kacastaṃ gurumityuvāca  // 25.58 //
nidhiṃ nidhīnāṃ varadaṃ varāṇāṃ ye nādriyante gurumarcanīyam  /
prāleyādriprojjvaladbhālasaṃsthaṃ pāpāṃllokāṃste vrajantyapratiṣṭhāḥ  // 25.59 //
śaunaka uvāca
surāpānād vañcanāt prāpayitvā saṃjñānāśaṃ cetasaścāpi ghoram  /
dṛṣṭvā kacaṃ cāpi tathābhirūpaṃ pītaṃ tathā surayā mohitena  // 25.60 //
samanyurutthāya mahānubhāvas tadośanā viprahitaṃ cikīrṣuḥ  /
kāvyaḥ svayaṃ vākyamidaṃ jagāda surāpānaṃ pratyasau jātaśaṅkaḥ  // 25.61 //
śukra uvāca
yo brāhmaṇo 'dyaprabhṛtīha kaścin mohātsurāṃ pāsyati mandabuddhiḥ  /
apetadharmā brahmahā caiva sa syādasmiṃlloke garhitaḥ syātpare ca  // 25.62 //
mayā cemāṃ vipradharmoktasīmāṃ maryādāṃ vai sthāpitāṃ sarvaloke  /
santo viprāḥ śuśruvāṃso gurūṇāṃ devā daityāścopaśṛṇvantu sarve  // 25.63 //
śaunaka uvāca
itīdamuktvā sa mahāprabhāvas taponidhīnāṃ nidhir aprameyaḥ  /
tāndānavāṃścaiva nigūḍhabuddhīn idaṃ samāhūya vaco 'bhyuvāca  // 25.64 //
śukra uvāca
ācakṣe vo dānavā bāliśāḥ stha śiṣyaḥ kaco vatsyati matsamīpe  /
saṃjīvanīṃ prāpya vidyāṃ mayāyaṃ tulyaprabhāvo brāhmaṇo brahmabhūtaḥ  // 25.65 //
śaunaka uvāca
guroruṣya sakāśe ca daśa varṣaśatāni saḥ  /
anujñātaḥ kaco gantum iyeṣa tridaśālayam  // 25.66 //


26
śaunaka uvāca
samāpitavrataṃ taṃ tu visṛṣṭaṃ guruṇā tadā  /
prasthitaṃ tridaśāvāsaṃ devayānīdamabravīt  // 26.1 //
devayānyuvāca
ṛṣer aṅgirasaḥ pautra vṛttenābhijanena ca  /
bhrājase vidyayā caiva tapasā ca damena ca  // 26.2 //
ṛṣir yathāṅgirā mānyaḥ piturmama mahāyaśāḥ  /
tathā mānyaśca pūjyaśca mama bhūyo bṛhaspatiḥ  // 26.3 //
evaṃ jñātvā vijānīhi yadbravīmi tapodhana  /
vratasthe niyamopete yathā vartāmyahaṃ tvayi  // 26.4 //
sa samāpitavidyo māṃ bhaktāṃ na tyaktumarhasi  /
gṛhāṇa pāṇiṃ vidhivan mama mantrapuraskṛtam  // 26.5 //
kaca uvāca
pūjyo mānyaśca bhagavān yathā mama pitā tava  /
tathā tvamanavadyāṅgi pūjanīyatamā matā  // 26.6 //
ātmaprāṇaiḥ priyatamā bhārgavasya mahātmanaḥ  /
tvaṃ bhadre dharmataḥ pūjyā guruputrī sadā mama  // 26.7 //
yathā mama gururnityaṃ mānyaḥ śukraḥ pitā tava  /
devayāni tathaiva tvaṃ naivaṃ māṃ vaktumarhasi  // 26.8 //
devayānyuvāca
guruputrasya putro me na tu tvamasi me pituḥ  /
tasmānmānyaśca pūjyaśca mamāpi tvaṃ dvijottama  // 26.9 //
asurairhanyamāne tu kace tvayi punaḥ punaḥ  /
tadāprabhṛti yā prītis tāṃ tvameva smarasva me  // 26.10 //
sauhārde cānurāge ca vettha me bhaktimuttamām  /
na māmarhasi dharmajña tyaktuṃ bhaktāmanāgasam  // 26.11 //
kaca uvāca
aniyojye niyoge māṃ niyunakṣi śubhavrate  /
prasīda subhrūrmahyaṃ tvaṃ guror gurutarā śubhe  // 26.12 //
yatroṣitaṃ viśālākṣi tvayā candranibhānane  /
tatrāhamuṣito bhadre kukṣau kāvyasya bhāmini  // 26.13 //
bhaginī dharmato me tvaṃ maivaṃ vocaḥ śubhānane  /
sukhenādhyuṣito bhadre na manyurvidyate mama  // 26.14 //
āpṛcche tvāṃ gamiṣyāmi śivamastvatha me pathi  /
avirodhena dharmasya smartavyo 'smi kathāntare  /
apramattodyatā nityam ārādhaya guruṃ mama  // 26.15 //
devayānyuvāca
daityairhatastvaṃ yadbhartṛ- buddhyā tvaṃ rakṣito mayā  /
yadi māṃ dharmakāmārthaṃ pratyākhyāsyasi dharmataḥ  // 26.16 //
tataḥ kaca na te vidyā siddhimeṣā gamiṣyati  // 26.17 //
kaca uvāca
guruputrīti kṛtvāhaṃ pratyākhyāsye na doṣataḥ  /
guruṇā cābhyanujñātaḥ kāmameva śapasva mām  // 26.18 //
ārṣaṃ dharmaṃ bruvāṇo 'haṃ devayāni yathā tvayā  /
śaptuṃ nārho 'smi kalyāṇi kāmato 'dya ca dharmataḥ  // 26.19 //
tasmādbhavatyā yaḥ kāmo na tathā sambhaviṣyati  /
ṛṣiputro na te kaścij jātu pāṇiṃ grahīṣyati  // 26.20 //
phaliṣyati na me vidyā tvadvacaśceti tattayā  /
adhyāpayiṣyāmi ca yaṃ tasya vidyā phaliṣyati  // 26.21 //
śaunaka uvāca
evamuktvā nṛpaśreṣṭha devayānīṃ kacastadā  /
tridaśeśālayaṃ śīghraṃ jagāma dvijasattamaḥ  // 26.22 //
tamāgatamabhiprekṣya devāḥ sendrapurogamāḥ  /
bṛhaspatiṃ sabhājyedaṃ kacamāhurmudānvitāḥ  // 26.23 //
devā ūcuḥ
tvaṃ kacāsmaddhitaṃ karma kṛtavānmahadadbhutam  /
na te yaśaḥ praṇaśitā bhāgabhākca bhaviṣyasi  // 26.24 //


27
śaunaka uvāca
kṛtavidye kace prāpte hṛṣṭarūpā divaukasaḥ  /
kacādavetya tāṃ vidyāṃ kṛtārthā bharatarṣabha  // 27.1 //
sarva eva samāgamya śatakratumathābruvan  /
kālastvadvikramasyādya jahi śatrūnpuraṃdara  // 27.2 //
evamuktastu saha tais tridaśair maghavāṃstadā  /
tathetyuktvopacakrāma so 'paśyadvipine striyaḥ  // 27.3 //
krīḍantīnāṃ tu kanyānāṃ vane caitrarathopame  /
vāyurbhūtaḥ sa vastrāṇi sarvāṇyeva vyamiśrayat  // 27.4 //
tato jalāt samuttīrya tāḥ kanyāḥ sahitāstadā  /
vastrāṇi jagṛhustāni yathāsaṃsthānyanekaśaḥ  // 27.5 //
tatra vāso devayānyāḥ śarmiṣṭhā jagṛhe tadā  /
vyatikramamajānantī duhitā vṛṣaparvaṇaḥ  // 27.6 //
tatastayor mithastatra virodhaḥ samajāyata  /
devayānyāśca rājendra śarmiṣṭhāyāśca tatkṛte  // 27.7 //
devayānyuvāca
kasmādgṛhṇāsi me vastraṃ śiṣyā bhūtvā mamāsuri  /
samudācārahīnāyā na te śreyo bhaviṣyati  // 27.8 //
śarmiṣṭhovāca
āsīnaṃ ca śayānaṃ ca pitā te pitaraṃ mama  /
stauti pṛcchati cābhīkṣṇaṃ nīcasthaḥ suvinītavat  // 27.9 //
yācatastvaṃ ca duhitā stuvataḥ pratigṛhṇataḥ  /
sutāhaṃ stūyamānasya dadato na tu gṛhṇataḥ  // 27.10 //
anāyudhā sāyudhāyāḥ kiṃ tvaṃ kupyasi bhikṣuki  /
lapsyase pratiyoddhāraṃ na ca tvāṃ gaṇayāmyaham  // 27.11 //
śaunaka uvāca
sā vismayaṃ devayānīṃ gatāṃ saktāṃ ca vāsasi  /
śarmiṣṭhā prākṣipatkūpe tataḥ svapuramāviśat  // 27.12 //
hateyamiti vijñāya śarmiṣṭhā pāpaniścayā  /
anavekṣya yayau tasmāt krodhavegaparāyaṇā  // 27.13 //
atha taṃ deśamabhyāgād yayātirnahuṣātmajaḥ  /
śrāntayugyaḥ śrāntarūpo mṛgalipsuḥ pipāsitaḥ  // 27.14 //
nāhuṣiḥ prekṣamāṇo hi sa nipāne gatodake  /
dadarśa kanyāṃ tāṃ tatra dīptāmagniśikhām iva  // 27.15 //
tāmapṛcchatsa dṛṣṭvaiva kanyāmamaravarṇinīm  /
sāntvayitvā nṛpaśreṣṭhaḥ sāmnā paramavalgunā  // 27.16 //
kā tvaṃ cārumukhī śyāmā sumṛṣṭamaṇikuṇḍalā  /
dīrghaṃ dhyāyasi cātyarthaṃ kasmācchvasiṣi cāturā  // 27.17 //
kathaṃ ca patitā hy asmin kūpe vīruttṛṇāvṛte  /
duhitā caiva kasya tvaṃ vada sarvaṃ sumadhyame  // 27.18 //
devayānyuvāca
yo 'sau devairhatān daityān utthāpayati vidyayā  /
tasya śukrasya kanyāhaṃ tvaṃ māṃ nūnaṃ na budhyase  // 27.19 //
eṣa me dakṣiṇo rājan pāṇis tāmranakhāṅguliḥ  /
samuddhara gṛhītvā māṃ kulīnastvaṃ hi me mataḥ  // 27.20 //
jānāmi tvāṃ ca saṃśāntaṃ vīryavantaṃ yaśasvinam  /
tasmānmāṃ patitāṃ kūpād asmād uddhartumarhasi  // 27.21 //
śaunaka uvāca
tāmatha brāhmaṇīṃ strīṃ ca vijñāya nahuṣātmajaḥ  /
gṛhītvā dakṣiṇe pāṇāv ujjahāra tato 'vaṭāt  // 27.22 //
uddhṛtya caināṃ tarasā tasmātkūpānnarādhipaḥ  /
āmantrayitvā suśroṇīṃ yayātiḥ svapuraṃ yayau  // 27.23 //
gate tu nāhuṣe tasmin devayānyapi ninditā  /
uvāca śokasaṃtaptā ghūrṇikāmāgatāṃ punaḥ  // 27.24 //
devayānyuvāca
tvaritaṃ ghūrṇike gaccha sarvamācakṣva me pituḥ  /
nedānīṃ tu pravekṣyāmi nagaraṃ vṛṣaparvaṇaḥ  // 27.25 //
śaunaka uvāca
sā tu vai tvaritaṃ gatvā ghūrṇikāsuramandiram  /
dṛṣṭvā kāvyamuvācedaṃ kampamānā vicetanā  // 27.26 //
ācakhyau ca mahābhāgā devayānī vane hatā  /
śarmiṣṭhayā mahāprājña duhitrā vṛṣaparvaṇaḥ  // 27.27 //
śrutvā duhitaraṃ kāvyas tadā śarmiṣṭhayā hatām  /
tvarayā niryayau duḥkhān mārgamāṇaḥ sutāṃ vane  // 27.28 //
dṛṣṭvā duhitaraṃ kāvyo devayānīṃ tapovane  /
bāhubhyāṃ sampariṣvajya duḥkhito vākyamabravīt  // 27.29 //
ātmadoṣair niyacchanti sarve duḥkhasukhe janāḥ  /
manye duścaritaṃ te 'sti tasyeyaṃ niṣkṛtiḥ kṛtā  // 27.30 //
devayānyuvāca
niṣkṛtir vāstu vā māstu śṛṇuṣvāvahito mama  /
śarmiṣṭhayā yaduktāsmi duhitrā vṛṣaparvaṇaḥ  // 27.31 //
satyaṃ kilaitatsā prāha daityānāmasmi gāyanā  /
evaṃ hi me kathayati śarmiṣṭhā vārṣaparvaṇī  // 27.32 //
vacanaṃ tīkṣṇaparuṣaṃ krodharaktekṣaṇā bhṛśam  /
stuvato duhitāsi tvaṃ yācataḥ pratigṛhṇataḥ  // 27.33 //
sutāhaṃ stūyamānasya dadato 'pratigṛhṇataḥ  /
iti māmāha śarmiṣṭhā duhitā vṛṣaparvaṇaḥ  /
krodhasaṃraktanayanā darpapūrṇānanā tataḥ  // 27.34 //
yadyahaṃ stuvatastāta duhitā pratigṛhṇataḥ  /
prasādayiṣye śarmiṣṭhām ityuktā hi sakhī mayā  // 27.35 //
śukra uvāca
stuvato duhitā na tvaṃ bhadre na pratigṛhṇataḥ  /
atastvaṃ stūyamānasya duhitā devayānyasi  // 27.35 //
vṛṣaparvaiva tadveda śakro rājā ca nāhuṣaḥ  /
acintyaṃ brahma nirdvaṃdvam aiśvaraṃ hi balaṃ mama  // 27.37 //


28
śukra uvāca
yaḥ pareṣāṃ naro nityam ativādāṃstitikṣati  /
devayāni vijānīhi tena sarvamidaṃ jitam  // 28.1 //
yaḥ samutpatitaṃ krodhaṃ nigṛhṇāti hayaṃ yathā  /
sa yantetyucyate sadbhir na yo raśmiṣu lambate  // 28.2 //
yaḥ samutpatitaṃ krodham akrodhane niyacchati  /
devayāni vijānīhi tena sarvamidaṃ jitam  // 28.3 //
yaḥ samutpatitaṃ kopaṃ kṣamayaiva nirasyati  /
yathoragastvacaṃ jīrṇāṃ sa vai puruṣa ucyate  // 28.4 //
yastu bhāvayate dharmaṃ yo 'timātraṃ titikṣati  /
yaśca tapto na tapati bhṛśaṃ so 'rthasya bhājanam  // 28.5 //
yo yajedaśvamedhena māsi māsi śataṃ samāḥ  /
yastu kupyenna sarvasya tayorakrodhano varaḥ  // 28.6 //
ye kumārāḥ kumāryaśca vairaṃ kuryuracetasaḥ  /
naitatprājñastu kurvīta viduste na balābalam  // 28.7 //
devayānyuvāca
vedāhaṃ tāta bālāpi kāryāṇāṃ tu gatāgatam  /
krodhe caivātivāde vā kāryasyāpi balābale  // 28.8 //
śiṣyasyāśiṣyavṛttaṃ hi na kṣantavyaṃ bubhūṣuṇā  /
asatsaṃkīrṇavṛtteṣu vāso mama na rocate  // 28.9 //
puṃso ye nābhinandanti vṛttenābhijanena ca  /
na teṣu nivasetprājñaḥ śreyorthī pāpabuddhiṣu  // 28.10 //
ye nainamabhijānanti vṛttenābhijanena ca  /
teṣu sādhuṣu vastavyaṃ sa vāsaḥ śreṣṭha ucyate  // 28.11 //
tanme mathnāti hṛdayam agnikalpamivāraṇam  /
vāgduruktaṃ mahāghoraṃ duhiturvṛṣaparvaṇaḥ  // 28.12 //
na hy ato duṣkaraṃ manye tāta lokeṣvapi triṣu  /
yaḥ sapatnaśriyaṃ dīptāṃ hīnaśrīḥ paryupāsate  // 28.13 //


29
śaunaka uvāca
tataḥ kāvyo bhṛguśreṣṭhaḥ samanyurupagamya ha  /
vṛṣaparvāṇam āsīnam ityuvācāvicārayan  // 29.1 //
nādharmaścarito rājan sadyaḥ phalati gaur iva  /
śanairāvartyamānastu mūlānyapi nikṛntati  // 29.2 //
yadi nātmani putreṣu na cetpaśyati naptṛṣu  /
pāpamācaritaṃ karma trivargamativartate  // 29.3 //
phalatyevaṃ dhruvaṃ pāpaṃ gurubhuktamivodare  /
yadā ghātayase vipraṃ kacamāṅgirasaṃ tadā  // 29.4 //
apāpaśīlaṃ dharmajñaṃ śuśrūṣuṃ madgṛhe ratam  /
vadhādanarhatastasya vadhācca duhiturmama  // 29.5 //
vṛṣaparvannibodha tvaṃ tyakṣyāmi tvāṃ sabāndhavam  /
sthātuṃ tvadviṣaye rājan na śaknomi tvayā saha  // 29.6 //
adyaivamabhijānāmi daityaṃ mithyāpralāpinam  /
yatastvamātmanodīrṇāṃ duhitāṃ kimupekṣase  // 29.7 //
vṛṣaparvovāca
nāvadyaṃ na mṛṣāvādaṃ tvayi jānāmi bhārgava  /
tvayi satyaṃ ca dharmaśca tatprasīdatu māṃ bhavān  // 29.8 //
adyāsmānapahāya tvam ito yāsyasi bhārgava  /
samudraṃ sampravekṣyāmi nānyadasti parāyaṇam  // 29.9 //
śukra uvāca
samudraṃ praviśadhvaṃ vo diśo vā vrajatāsurāḥ  /
duhiturnāpriyaṃ soḍhuṃ śakto 'haṃ dayitā hi me  // 29.10 //
prasādyatāṃ devayānī jīvitaṃ yatra me sthitam  /
yogakṣemakaraste 'ham indrasyeva bṛhaspatiḥ  // 29.11 //
vṛṣaparvovāca
yatkiṃcid asurendrāṇāṃ vidyate vasu bhārgava  /
bhuvi hastirathāśvaṃ vā tasya tvaṃ mama ceśvaraḥ  // 29.12 //
śukra uvāca
yatkiṃcid asti draviṇaṃ daityendrāṇāṃ mahāsura  /
tasyeśvaro 'smi yadyetad devayānī prasādyatām  // 29.13 //
śaunaka uvāca
tatastu tvaritaḥ śukras tena rājñā samaṃ yayau  /
uvāca caināṃ subhage pratipannaṃ vacastava  // 29.14 //
devayānyuvāca
yadi tvamīśvarastāta rājño vittasya bhārgava  /
nābhijānāmi tatte 'haṃ rājā vadatu māṃ svayam  // 29.15 //
vṛṣaparvovāca
yaṃ kāmamabhijānāsi devayāni śucismite  /
tatte 'haṃ sampradāsyāmi yadyapi syātsudurlabham  // 29.16 //
devayānyuvāca
dāsīṃ kanyāsahasreṇa śarmiṣṭhāmabhikāmaye  /
anuyāsyati māṃ tatra yatra dāsyati me pitā  // 29.17 //
vṛṣaparvovāca
uttiṣṭha dhātri gaccha tvaṃ śarmiṣṭhāṃ śīghramānaya  /
yaṃ ca kāmayate kāmaṃ devayānī karotu tam  // 29.18 //
śaunaka uvāca
tato dhātrī tatra gatvā śarmiṣṭhām idamabravīt  /
uttiṣṭha bhadre śarmiṣṭhe jñātīnāṃ sukhamāvaha  // 29.19 //
tyajati brāhmaṇaḥ śiṣyān devayānyā pracoditaḥ  /
yaṃ sā kāmayate kāmaṃ sa kāryo 'tra tvayānaghe  /
dāsītvam abhijātāsi devayānyāḥ suśobhane  // 29.20 //
śarmiṣṭhovāca
yaṃ ca kāmayate kāmaṃ karavāṇyahamadya tam  /
mā gānmanyuvaśaṃ śukro devayānī ca matkṛte  // 29.21 //
śaunaka uvāca
tataḥ kanyāsahasreṇa vṛtā śibikayā tadā  /
piturnideśāttvaritā niścakrāma purottamāt  // 29.22 //
śarmiṣṭhovāca
ahaṃ kanyāsahasreṇa dāśī te paricārikā  /
dhruvaṃ tvāṃ tatra yāsyāmi yatra dāsyati te pitā  // 29.23 //
devayānyuvāca
stuvato duhitā cāhaṃ yācataḥ pratigṛhṇataḥ  /
stūyamānasya duhitā kathaṃ dāsī bhaviṣyasi  // 29.24 //
śarmiṣṭhovāca
yena kenacidārtānāṃ jñātīnāṃ sukhamāvahet  /
anuyāsyāmyahaṃ tatra yatra dāsyati te pitā  // 29.25 //
śaunaka uvāca
pratiśrute dāsabhāve duhitrā vṛṣaparvaṇaḥ  /
devayānī nṛpaśreṣṭha pitaraṃ vākyamabravīt  // 29.26 //
devayānyuvāca
praviśāmi puraṃ tāta tuṣṭāsmi dvijasattama  /
amoghaṃ tava vijñānam asti vidyābalaṃ ca te  // 29.27 //
śaunaka uvāca
evamukto dvijaśreṣṭho duhitrā sumahāyaśāḥ  /
praviveśa puraṃ hṛṣṭaḥ pūjitaḥ sarvadānavaiḥ  // 29.28 //


30
śaunaka uvāca
atha dīrgheṇa kālena devayānī nṛpottama  /
vanaṃ tadaiva niryātā krīḍārthaṃ varavarṇinī  // 30.1 //
tena dāsīsahasreṇa sārdhaṃ śarmiṣṭhayā tadā  /
tameva deśaṃ samprāptā yathākāmaṃ cacāra sā  // 30.2 //
tābhiḥ sakhībhiḥ sahitā sarvābhirmuditā bhṛśam  /
krīḍantyo 'bhiratāḥ sarvāḥ pibantyo madhu mādhavam  // 30.3 //
khādantyo vividhānbhakṣyān phalāni vividhāni ca  /
punaśca nāhuṣo rājā mṛgalipsuryadṛcchayā  // 30.4 //
tameva deśaṃ samprāpto jalalipsuḥ pratarṣitaḥ  /
dadarśa devayānīṃ ca śarmiṣṭhāṃ tāśca yoṣitaḥ  // 30.5 //
pibantyo lalanāstāśca divyābharaṇabhūṣitāḥ  /
upaviṣṭāṃ ca dadṛśe devayānīṃ śucismitām  // 30.6 //
rūpeṇāpratimāṃ tāsāṃ strīṇāṃ madhye varāṅganām  /
śarmiṣṭhayā sevyamānāṃ pādasaṃvāhanādibhiḥ  // 30.7 //
yayātiruvāca
dvābhyāṃ kanyāsahasrābhyāṃ dve kanye parivārite  /
gotre ca nāmanī caiva dvayoḥ pṛcchāmyato hy aham  // 30.8 //
devayānyuvāca
ākhyāsyāmyaham ādatsva vacanaṃ me narādhipa  /
śuko nāmāsuraguruḥ sutāṃ jānīhi tasya mām  // 30.9 //
iyaṃ ca me sakhī dāsī yatrāhaṃ tatra gāminī  /
duhitā dānavendrasya śarmiṣṭhā vṛṣaparvaṇaḥ  // 30.10 //
yayātiruvāca
kathaṃ tu te sakhī dāsī kanyeyaṃ varavarṇinī  /
asurendrasutā subhūḥ paraṃ kautūhalaṃ hi me  // 30.11 //
devayānyuvāca
sarvameva naravyāghra vidhānamanuvartate  /
vidhinā vihitaṃ jñātvā mā vicitraṃ manaḥ kṛthāḥ  // 30.12 //
rājavadrūpaveṣau te brāhmīṃ vācaṃ bibharṣi ca  /
kiṃnāmā tvaṃ kutaścāsi kasya putraśca śaṃsa me  // 30.13 //
yayātiruvāca
brahmacaryeṇa vedo me kṛtsnaḥ śrutipathaṃ gataḥ  /
rājāhaṃ rājaputraśca yayātiriti viśrutaḥ  // 30.14 //
devayānyuvāca
kena cārthena nṛpate hy enaṃ deśaṃ samāgataḥ  /
jighṛkṣurvāri yatkiṃcid athavā mṛgalipsayā  // 30.15 //
yayātiruvāca
mṛgalipsurahaṃ bhadre pānīyārtham ihāgataḥ  /
bahudhāpyanuyukto 'smi tvam anujñātumarhasi  // 30.16 //
devayānyuvāca
dvābhyāṃ kanyāsahasrābhyāṃ dāsyā śarmiṣṭhayā saha  /
tvadadhīnāsmi bhadraṃ te sakhe bhartā ca me bhava  // 30.17 //
yayātiruvāca
viddhyauśanasi bhadraṃ te na tvadarho 'smi bhāmini  /
avivāhyāḥ sma rājāno devayāni pitustava  // 30.18 //
devayānyuvāca
saṃsṛṣṭaṃ brahmaṇā kṣatraṃ kṣatraṃ brahmaṇi saṃśritam  /
ṛṣiśca ṛṣiputraśca nāhuṣādya bhajasva mām  // 30.19 //
yayātiruvāca
ekadehodbhavā varṇāś catvāro 'pi varānane  /
pṛthagdharmāḥ pṛthakchaucās teṣāṃ vai brāhmaṇo varaḥ  // 30.20 //
devayānyuvāca
pāṇigraho nāhuṣāyaṃ na puṃbhiḥ sevitaḥ purā  /
tvaṃ pāṇimagrahīdagre vṛṇomi tvāmahaṃ tataḥ  // 30.21 //
kathaṃ tu me manasvinyāḥ pāṇimanyaḥ pumānspṛśet  /
gṛhītamṛṣiputreṇa svayaṃ vāpyṛṣiṇā tvayā  // 30.22 //
yayātiruvāca
kruddhādāśīviṣāt sarpāj jvalanātsarvatomukhāt  /
durādharṣataro vipraḥ puruṣeṇa vijānatā  // 30.23 //
devayānyuvāca
kathamāśīviṣāt sarpāj jvalanāt sarvatomukhāt  /
durādharṣataro vipra ityāttha puruṣarṣabha  // 30.24 //
yayātiruvāca
daśedāśīviṣastvekaṃ śastreṇaikaśca vadhyate  /
hanti vipraḥ sarāṣṭrāṇi purāṇyapi hi kopitaḥ  // 30.25 //
durādharṣataro vipras tasmādbhīru mato mama  /
ato 'dattāṃ ca pitrā tvāṃ bhadre na vivahāmyaham  // 30.26 //
devayānyuvāca
dattāṃ vahasva pitrā māṃ tvaṃ hi rājanvṛto mayā  /
ayācato bhayaṃ nāsti dattāṃ ca pratigṛhṇataḥ  // 30.27 //
śaunaka uvāca
tvaritaṃ devayānyātha preṣitā piturātmanaḥ  /
sarvaṃ nivedayāmāsa dhātrī tasmai yathātatham  // 30.28 //
śrutvaiva ca sa rājānaṃ darśayāmāsa bhārgavaḥ  /
dṛṣṭvaivam āgataṃ vipraṃ yayātiḥ pṛthivīpatiḥ  // 30.29 //
vavande brāhmaṇaṃ kāvyaṃ prāñjaliḥ praṇataḥ sthitaḥ  /
taṃ cāpyabhyavadatkāvyaḥ sāmnā paramavalgunā  // 30.30 //
devayānyuvāca
rājāyaṃ nāhuṣastāta durgame pāṇimagrahīt  /
namaste dehi māmasmai loke nānyaṃ patiṃ vṛṇe  // 30.31 //
śukra uvāca
vṛto 'nayā patirvīra sutayā tvaṃ mameṣṭayā  /
gṛhāṇemāṃ mayā dattāṃ mahiṣīṃ nahuṣātmaja  // 30.32 //
yayātiruvāca
adharmo māṃ spṛśedevaṃ pāpam asyāśca bhārgava  /
varṇasaṃkarato brahmann iti tvāṃ pravṛṇomyaham  // 30.33 //
śukra uvāca
adharmāttvāṃ vimuñcāmi varaṃ varaya cepsitam  /
asminvivāhe tvaṃ ślāghyo rahaḥ pāpaṃ nudāmi te  // 30.34 //
vahasva bhāryāṃ dharmeṇa devayānīṃ śucismitām  /
anayā saha samprītim atulāṃ samavāpnuhi  // 30.35 //
iyaṃ cāpi kumārī te śarmiṣṭhā vārṣaparvaṇī  /
saṃpūjyā satataṃ rājan na caināṃ śayane hvaya  // 30.36 //
śaunaka uvāca
evamukto yayātistu śukraṃ kṛtvā pradakṣiṇam  /
jagāma svapuraṃ hṛṣṭaḥ so 'nujñāto mahātmanā  // 30.37 //


31
śaunaka uvāca
yayātiḥ svapuraṃ prāpya mahendrapurasaṃnibham  /
praviśyāntaḥpuraṃ tatra devayānīṃ nyaveśayat  // 31.1 //
devayānyāścānumate sutāṃ tāṃ vṛṣaparvaṇaḥ  /
aśokavanikābhyāśe gṛhaṃ kṛtvā nyaveśayat  // 31.2 //
vṛtāṃ dāsīsahasreṇa śarmiṣṭhām āsurāyaṇīm  /
vāsobhirannapānaiśca saṃvibhajya susaṃvṛtām  // 31.3 //
devayānyā tu sahitaḥ sa nṛpo nahuṣātmajaḥ  /
vijahāra bahūnabdān devavanmudito bhṛśam  // 31.4 //
ṛtukāle tu samprāpte devayānī varāṅganā  /
lebhe garbhaṃ prathamataḥ kumāraśca vyajāyata  // 31.5 //
gate varṣasahasre tu śarmiṣṭhā vārṣaparvaṇī  /
dadarśa yauvanaṃ prāptā ṛtuṃ sā kamalekṣaṇā  // 31.6 //
cintayāmāsa dharmajñā ṛtuprāptau ca bhāminī  /
ṛtukālaśca samprāpto na kaścinme patirvṛtaḥ  // 31.7 //
kiṃ prāptaṃ kiṃca kartavyaṃ kathaṃ kṛtvā sukhaṃ bhavet  /
devayānī prasūtāsau vṛthāhaṃ prāptayauvanā  // 31.8 //
yathā tayā vṛto bhartā tathaivāhaṃ vṛṇomi tam  /
rājñā putraphalaṃ deyam iti me niścitā matiḥ  /
apīdānīṃ sa dharmātmā raho me darśanaṃ vrajet  // 31.9 //
śaunaka uvāca
atha niṣkramya rājāsau tasminkāle yadṛcchayā  /
aśokavanikābhyāśe śarmiṣṭhāṃ prāpya vismitaḥ  // 31.10 //
tamekaṃ rahasi dṛṣṭvā śarmiṣṭhā cāruhāsinī  /
pratyudgamyāñjaliṃ kṛtvā rājānaṃ vākyamabravīt  // 31.11 //
śarmiṣṭhovāca
somaścendraśca vāyuśca yamaśca varuṇaśca vā  /
tava vā nāhuṣa gṛhe kaḥ striyaṃ draṣṭumarhati  // 31.12 //
rūpābhijanaśīlairhi tvaṃ rājanvettha māṃ sadā  /
sā tvāṃ yāce prasādyeha rantumehi narādhipa  // 31.13 //
yayātiruvāca
vedmi tvāṃ śīlasampannāṃ daityakanyāmaninditām  /
rūpaṃ tu te na paśyāmi sūcyagramapi ninditam  // 31.14 //
māmabravīttadā śukro devayānīṃ yadāvaham  /
neyam āhvayitavyā te śayane vārṣaparvaṇī  // 31.15 //
śarmiṣṭhovāca
na narmayuktaṃ vacanaṃ hinasti na strīṣu rājan na vivāhakāle  /
prāṇātyaye sarvadhanāpahāre pañcānṛtānyāhur apātakāni  // 31.16 //
pṛṣṭāstu sākṣye pravadanti cānyathā bhavanti mithyāvacanā narendra te  /
ekārthatāyāṃ tu samāhitāyāṃ mithyā vadantaṃ hy anṛtaṃ hinasti  // 31.17 //
yayātiruvāca
rājā pramāṇaṃ bhūtānāṃ sa vinaśyenmṛṣā vadan  /
arthakṛcchramapi prāpya na mithyā kartumutsahe  // 31.18 //
śarmiṣṭhovāca
samāv etau matau rājan patiḥ sakhyāśca yaḥ patiḥ  /
samaṃ vivāha ityāhuḥ sakhyā me 'si patiryataḥ  // 31.19 //
yayātiruvāca
dātavyaṃ yācamānasya hīti me vratamāhitam  /
tvaṃ ca yācasi kāmaṃ māṃ brūhi kiṃ karavāṇi tat  // 31.20 //
śarmiṣṭhovāca
adharmāttrāhi māṃ rājan dharmaṃ ca pratipādaya  /
tvatto 'patyavatī loke careyaṃ dharmamuttamam  // 31.21 //
traya evādhanā rājan bhāryā dāsastathā sutaḥ  /
yatte samadhigacchanti yasya te tasya taddhanam  // 31.22 //
devayānyā bhujiṣyāsmi vaśyā ca tava bhārgavī  /
sā cāhaṃ ca tvayā rājan bharaṇīyāṃ bhajasva mām  // 31.23 //
śaunaka uvāca
evamuktastayā rājā tathyam ityabhijajñivān  /
pūjayāmāsa śarmiṣṭhāṃ dharmaṃ ca pratipādayan  // 31.24 //
sa samāgamya śarmiṣṭhāṃ yathākāmamavāpya ca  /
anyonyaṃ cābhisaṃpūjya jagmatustau yathāgatam  // 31.25 //
tasminsamāgame subhrūḥ śarmiṣṭhā vārṣaparvaṇī  /
lebhe garbhaṃ prathamatas tasmānnṛpatisattamāt  // 31.26 //
prajajñe ca tataḥ kāle rājñī rājīvalocanā  /
kumāraṃ devagarbhābham ādityasamatejasam  // 31.27 //


32
śaunaka uvāca
śrutvā kumāraṃ jātaṃ sā devayānī śucismitā  /
cintayāviṣṭaduḥkhārtā śarmiṣṭhāṃ prati bhārata  // 32.1 //
tato 'bhigamya śarmiṣṭhāṃ devayānyabravīdidam  /
kimarthaṃ vṛjinaṃ subhrūḥ kṛtaṃ te kāmalubdhayā  // 32.2 //
śarmiṣṭhovāca
ṛṣir abhyāgataḥ kaścid dharmātmā vedapāragaḥ  /
sa mayā tu varaḥ kāmaṃ yācito dharmasaṃhatam  // 32.3 //
nāhamanyāyataḥ kāmam ācarāmi śucismite  /
tasmād ṛṣer mamāpatyam iti satyaṃ bravīmi te  // 32.4 //
devayānyuvāca
padyetadevaṃ śarmiṣṭhe na manyurvidyate mama  /
apatyaṃ yadi te labdhaṃ jyeṣṭhācchreṣṭhācca vai dvijāt  // 32.5 //
śobhanaṃ bhīru satyaṃ cet kathaṃ sa jñāyate dvijaḥ  /
gotranāmābhijanataḥ śrotumicchāmi taṃ dvijam  // 32.6 //
śarmiṣṭhovāca
ojasā tejasā caiva dīpyamānaṃ raviṃ yathā  /
taṃ dṛṣṭvā mama saṃpraṣṭuṃ śaktir nāsīcchucismite  // 32.7 //
śaunaka uvāca
anyonyamevam uktvā ca samprahasya ca te mithaḥ  /
jagāma bhārgavī veśma tathyamityabhijānatī  // 32.8 //
yayātirdevayānyāṃ tu putrāv ajanayannṛpaḥ  /
yaduṃ ca turvasuṃ caiva śakraviṣṇū ivāparau  // 32.9 //
tasmādeva tu rājarṣeḥ śarmiṣṭhā vārṣaparvaṇī  /
druhyaṃ cānuṃ ca pūruṃ ca trīnkumārānajījanat  // 32.10 //
tataḥ kāle ca kasmiṃścid devayānī śucismitā  /
yayātisahitā rājañ jagāma haritaṃ vanam  // 32.11 //
dadarśa ca tadā tatra kumārāndevarūpiṇaḥ  /
krīḍamānān tu visrabdhān vismitā cedamabravīt  // 32.12 //
devayānyuvāca
kasyaite dārakā rājan devaputropamāḥ śubhāḥ  /
varcasā rūpataścaiva dṛśyante sadṛśāstava  // 32.13 //
evaṃ pṛṣṭvā tu rājānaṃ kumārānparyapṛcchata  /
kiṃ nāmadheyagotre vaḥ putrakā brāhmaṇaḥ pitā  // 32.14 //
vibrūta me yathātathyaṃ śrotukāmāsmyato hy aham  /
te 'darśayanpradeśinyā tameva nṛpasattamam  // 32.15 //
śarmiṣṭhāṃ mātaraṃ caiva tasyā ūcuḥ kumārakāḥ  /
śaunaka uvāca
ityuktvā sahitāstena rājānam upacakramuḥ  // 32.16 //
nābhyanandata tānrājā devayānyās tadāntike  /
rudantaste 'tha śarmiṣṭhām abhyayur bālakāstadā  // 32.17 //
dṛṣṭvā teṣāṃ tu bālānāṃ praṇayaṃ pārthivaṃ prati  /
buddhvā ca tattvato devī śarmiṣṭhāmidamabravīt  // 32.18 //
devayānyuvāca
madadhīnā satī kasmād akārṣīrvipriyaṃ mama  /
tamevāsuradharmaṃ tvam āsthitā na bibheṣi kim  // 32.19 //
śarmiṣṭhovāca
yaduktamṛṣirityeva tatsatyaṃ cāruhāsini  /
nyāyato dharmataścaiva carantī na bibhemi te  // 32.20 //
yadā tvayā vṛto rājā vṛta eva tadā mayā  /
sakhībhartā hi dharmeṇa bhartā bhavati śobhane  // 32.21 //
pūjyāsi mama mānyā ca śreṣṭhā jyeṣṭhā ca brāhmaṇī  /
tvatto hi me pūjyataro rājarṣiḥ kiṃ na vetsi tat  // 32.22 //
śaunaka uvāca
śrutvā tasyāstato vākyaṃ devayānyabravīd idam  /
rājannādyeha vatsyāmi vipriyaṃ me tvayā kṛtam  // 32.23 //
sahasotpatitāṃ śyāmāṃ dṛṣṭvā tāṃ sāśrulocanām  /
tūrṇaṃ sakāśaṃ kāvyasya prasthitāṃ vyathitastadā  // 32.24 //
anuvavrāja saṃbhrāntaḥ pṛṣṭhataḥ sāntvayannṛpaḥ  /
nyavartata na sā caiva krodhasaṃraktalocanā  // 32.25 //
avibruvantī kiṃcicca rājānaṃ sāśrulocanā  /
acirādeva samprāptā kāvyasyośanaso 'ntikam  // 32.26 //
sā tu dṛṣṭvaiva pitaram abhivādyāgrataḥ sthitā  /
anantaraṃ yayātistu pūjayāmāsa bhārgavam  // 32.27 //
devayānyuvāca
adharmeṇa jito dharmaḥ pravṛttamadharottaram  /
śarmiṣṭhā yātivṛttāsti duhitā vṛṣaparvaṇaḥ  // 32.28 //
trayo 'syāṃ janitāḥ putrā rājñānena yayātinā  /
durbhagāyā mama dvau tu putrau tāta bravīmi te  // 32.29 //
dharmajña iti vikhyāta eṣa rājā bhṛgūdvaha  /
atikrāntaśca maryādāṃ kāvyaitatkathayāmi te  // 32.30 //
śukra uvāca
dharmajñastvaṃ mahārāja yo 'dharmam akṛthāḥ priyam  /
tasmājjarā tvām acirād dharṣayiṣyati durjayā  // 32.31 //
yayātiruvāca
ṛtuṃ yo yācyamānāyā na dadāti pumānvṛtaḥ  /
bhrūṇahetyucyate brahman sa ceha brahmavādibhiḥ  // 32.32 //
ṛtukāmāṃ striyaṃ yastu gamyāṃ rahasi yācitaḥ  /
nopaiti yo hi dharmeṇa brahmahetyucyate budhaiḥ  // 32.33 //
ityetāni samīkṣyāhaṃ kāraṇāni bhṛgūdvaha  /
adharmabhayasaṃvignaḥ śarmiṣṭhām upajagmivān  // 32.34 //
śukra uvāca
na tv ahaṃ pratyavekṣyaste madadhīno 'si pārthiva  /
mithyācaraṇadharmeṣu cauryaṃ bhavati nāhuṣa  // 32.35 //
śaunaka uvāca
krodhenośanasā śapto yayātirnāhuṣastadā  /
pūrvaṃ vayaḥ parityajya jarāṃ sadyo 'nvapadyata  // 32.36 //
yayātiruvāca
atṛpto yauvanasyāhaṃ devayānyāṃ bhṛgūdvaha  /
prasādaṃ kuru me brahmañ jareyaṃ mā viśeta mām  // 32.37 //
śukra uvāca
nāhaṃ mṛṣā vadāmyetaj jarāṃ prāpto 'si bhūmipa  /
jarāṃ tv etāṃ tvamanyasmin saṃkrāmaya yadīcchasi  // 32.38 //
yayātiruvāca
rājyabhāk sa bhavedbrahman puṇyabhākkīrtibhāktathā  /
yo dadyānme vayaḥ putras tadbhavānanumanyatām  // 32.39 //
śukra uvāca
saṃkrāmayiṣyasi jarāṃ yatheṣṭaṃ nahuṣātmaja  /
māmanudhyāya tattvena na ca pāpamavāpsyasi  // 32.40 //
vayo dāsyati te putro yaḥ sa rājā bhaviṣyati  /
āyuṣmānkīrtimāṃścaiva bahvapatyas tathaiva ca  // 32.41 //
