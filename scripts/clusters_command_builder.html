Clusters Command Builder

Runs a comprehensive clustering analysis using stylo. Loads the corpus once, then produces visualizations across multiple analysis types (CA, BCT, MDS, PCV) and distance measures. All output plots and data tables are saved to a timestamped results directory.

Path to the clusters.R script, relative to your working directory.

Directory containing your .txt corpus files. Defaults to corpus/main if left blank.

The unit of analysis. Words are better for longer texts with stable vocabulary; characters are more robust for short texts or highly inflected languages.

Length of the feature sequence. Unigrams (1) count individual words or characters; bigrams and trigrams capture short sequences. Character trigrams are a common choice for morphologically rich languages.

Minimum number of most frequent features to include. BCT iterates from this value up to max; CA and MDS use only the max. Recommended: ~50 for words, ~2000 for character n-grams.

Maximum number of most frequent features. Character n-grams need much larger values than words because the feature space is bigger. Recommended: ~80–500 for words, ~2000–5000 for character n-grams.

Step size between MFW min and max. Used by BCT to build consensus across multiple feature-set sizes. Recommended: ~10 for words, ~1000 for character n-grams.

Minimum culling threshold. Features appearing in fewer than this percentage of texts are dropped. 0 disables culling.

Maximum culling threshold. Set equal to min for a fixed culling level rather than a range.

Step size for iterating between culling min and max. Set to 0 when min and max are equal.

Used by BCT (Bootstrap Consensus Tree). An edge is kept in the consensus only if it appears in at least this fraction of bootstrap trees. Range 0–1; 0.5 is standard.

Height of output plot files in inches.

Width of output plot files in inches.

Font size for text labels on plots. Increase if labels are hard to read on large plots.

When checked, text is analyzed with original capitalization. By default, all text is lowercased before analysis.
